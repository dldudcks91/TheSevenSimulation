# 영웅 시스템 설계

> 상태: 구조 확정, 수치 미확정
> 작성일: 2026-03-24

---

## 1. 개요

영웅은 **인간**이다. 용병, 모험가, 떠돌이 — 바알에게 고용된 자들.
각자 깊은 성격적 결함을 안고 있으며, 그 결함이 곧 **죄종 태그**다.

멀쩡한 인간은 악마 밑에서 일하지 않는다.
갈 곳 없는 자, 버림받은 자, 망가진 자들이 바알에게 모인다.

### 죄종별 인간 결함

| 죄종 | 인간으로서의 결함 | 예시 |
|------|----------------|------|
| 분노 | 충동적, 폭력적, 참을성 없음 | 전쟁에서 돌아온 용병, 복수에 사로잡힌 검사 |
| 시기 | 비교 강박, 열등감, 뒷공작 | 인정받지 못한 2인자, 형제의 그림자에 산 궁수 |
| 탐욕 | 독점욕, 소유욕, 나누지 못함 | 모든 것을 잃은 상인, 보물을 찾는 도적 |
| 나태 | 무기력, 회피, 전염되는 체념 | 전쟁에 지친 기사, 재능 있지만 포기한 학자 |
| 폭식 | 과잉, 불안한 비축, 끝없는 허기 | 굶주림을 겪은 유민, 채워지지 않는 전사 |
| 색욕 | 집착, 의존, 관계 중독 | 사랑에 실패한 시인, 혼자가 되는 게 두려운 치유사 |
| 교만 | 자기확신, 독선, 복종 거부 | 왕좌에서 쫓겨난 지휘관, 자기가 옳다고 믿는 성직자 |

Phase 1에서는 7체 (죄종당 1체)만 구현. 랜덤 생성.

## 2. 영웅 구조

```
[영웅] — 랜덤 생성
├── 이름
├── 죄종 1개 (고정) — 성격 + 만족/불만 조건 + 폭주/이탈 행동 결정
├── 사기 (0~100) — 유일한 관리 수치
├── 기본 스탯 7개 (각 1~20, 랜덤) — 능력 결정
├── 세부 스탯
│   ├── 파생 세부스탯 11개 — 기본 스탯에서 산출 (행동 9 + 관계 2)
│   └── 독립 세부스탯 7개 (각 1~20, 랜덤) — 본성, 양면성
├── 히든: 성장률 + 상한선 (스탯별) — 사용 기반 성장
├── 고유 스킬 1개
├── 특성풀 3개 (고유 1 + 죄종 2) — Phase 2
└── 장비 3슬롯 — Phase 2
```

> **UI**: 영웅 고용 시 기본 스탯만 표시. 영입 후 세부스탯 버튼으로 상세 확인 가능.

## 2-1. 스탯 시스템

### 설계 원칙
- **죄종 = 성격**, **스탯 = 능력** — 완전 분리
- 죄종이 스탯을 결정하지 않음. 랜덤으로 굴림
- 한 스탯이 여러 행동에 영향 → 배치 딜레마 발생
- UI에서 행동별 적합도를 별점으로 자동 표시

### 기본 스탯 7개 (각 1~20)

| 스탯 | 의미 | 영향 |
|------|------|------|
| **힘** | 물리적 파워 | 사냥, 건설, 대장간, 전투 |
| **민첩** | 빠르기, 손재주 | 사냥, 채집, 연금술, 전투 |
| **지능** | 머리, 판단력 | 외교, 연구(단독), 교역, 전투 |
| **체력** | 지구력, 내구 | 전투 HP풀 + 부상 회복 속도 |
| **감각** | 눈썰미, 직감 | 채집, 대장간, 연금술 |
| **통솔** | 지휘, 리더십 | 건설, 고용, 통솔력(관계), 전투 |
| **매력** | 영향력, 교섭 | 외교, 교역, 고용, 매력도(관계) |

### 체력(HP) 시스템

- **전투 전용 HP** — 낮 행동(건설, 연구 등)으로는 소모되지 않음
- HP 공식: `hp_base(200) + 체력×hp_vitality_mult(15) + 힘×hp_strength_mult(5)` (balance.csv)
- 전투 후 부상(HP 0) 상태로 귀환 → 매 턴 **자동 회복** (체력 스탯이 높을수록 회복량↑)
- **병원**에서 치료 시 회복 속도 2배, **치료약** 사용 시 즉시 회복
- 체력 스탯의 의미 = **"전투에서 얼마나 오래 버티는가 + 부상 회복 속도"**

| 체력 스탯 | 전투 HP풀 | 턴당 자동 회복 | 병원 치료 |
|----------|----------|-------------|---------|
| 낮음 (1~7) | 낮음 | 적게 회복 | 2배 속도 |
| 보통 (8~14) | 보통 | 보통 회복 | 2배 속도 |
| 높음 (15~20) | 높음 | 많이 회복 | 2배 속도 |

### 행동별 스탯 매핑

모든 행동은 **동등 가중치(50:50)**. 행동 효율 = (스탯A + 스탯B) / 2. 연구만 지능 단독.

|  | 야외 |||| 실내 ||||||
|---|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|
|  | 전투 | 사냥 | 채집 | 외교 | 건설 | 대장간 | 연금술 | 연구 | 교역 | 고용 |
| **힘** | ? | ● | | | ● | ● | | | | |
| **민첩** | ? | ● | ● | | | | ● | | | |
| **지능** | ? | | | ● | | | | ● | ● | |
| **감각** | ? | | ● | | | ● | ● | | | |
| **통솔** | ? | | | | ● | | | | | ● |
| **매력** | ? | | | ● | | | | | ● | ● |

전투: 힘+민첩(원정), 힘+통솔(방어). 체력은 HP풀+회복 속도.

### 파생 세부스탯 — 관계 계열

| 세부스탯 | 산출 | 관계에서의 역할 |
|---------|------|---------------|
| **통솔력** | 통솔 | 다른 영웅이 이 영웅을 **따름** (존경) |
| **매력도** | 매력 | 다른 영웅이 이 영웅을 **좋아함** (호감) |
| **민감도** | 감수성 (독립) | 이 영웅이 다른 영웅에게 **민감하게 반응** |

> 민감도는 기본 스탯 감각이 아닌, 독립 세부스탯 감수성에서 산출.

## 2-2. 독립 세부스탯

### 설계 원칙
- 기본 스탯과 무관, 랜덤 생성 (각 1~20)
- **높다고 무조건 좋지 않음** — 양면성을 가짐
- 죄종별 1개씩 영감을 받았으나, 영웅의 죄종과 무관하게 랜덤 배정
- 사용 기반 성장 가능

### 7개 독립 세부스탯

| 죄종 영감 | 세부스탯 | 높으면 | 낮으면 |
|---------|---------|-------|-------|
| 분노 | **공격성** | 전투 공격력↑, 동료 충돌 잦음 | 전투 소극적, 거점 내 평화 |
| 탐욕 | **욕심** | 보상 획득량↑, 자원 갈등 | 수확 적지만 나눔 |
| 교만 | **자존심** | 리더 시 능력↑, 부하 역할 시 사기↓ | 어디든 불만 없이 배치 |
| 시기 | **호기심** | 탐색/연구 보너스, 위험 이벤트 조우↑ | 안전하지만 발견 기회 적음 |
| 폭식 | **집요함** | 장기 작업 보너스, 중단/철수 판단 느림 | 판단 빠르지만 오래 못 버팀 |
| 색욕 | **감수성** | 관계 형성 빠름, 동료 사기에 같이 흔들림 | 냉정, 연쇄 전염 저항 |
| 나태 | **독립심** | 혼자 행동 효율↑, 팀 시너지↓ | 팀에서 잘 맞지만 혼자선 약함 |

## 2-3. 성장 시스템

### 사용 기반 성장
- 기본 스탯과 독립 세부스탯 모두 사용을 통해 성장 가능
- 모든 스탯의 기본 범위는 **1~20**, 성장으로 **20을 초과**할 수 있음

### 히든 파라미터 (플레이어에게 비공개)

| 파라미터 | 의미 | 비고 |
|---------|------|------|
| **성장률** (스탯별) | 해당 스탯 상승 속도 | 같은 행동을 해도 영웅마다 성장 속도 다름 |
| **상한선** (스탯별) | 해당 스탯 최대 도달치 | 재능의 한계 — 상한이 20 이상이면 초월 성장 가능 |

> 플레이어는 수치를 볼 수 없고, 행동 결과를 관찰하며 추론한다.
> "이 영웅 사냥 계속 보냈더니 힘이 쑥쑥 오르네" → 재능 있는 영웅
> "이 영웅은 건설 시켜도 힘이 안 오르네..." → 재능 없는 영웅

### 딜레마 예시

```
힘 16, 감각 15인 영웅
  → 대장간 ★★★★★, 사냥도 강함
  → 장비 만들게 할까? 사냥 보낼까?

민첩 17, 감각 14인 영웅
  → 채집 ★★★★★, 연금술 ★★★★★
  → 채집 보내면 연금술 못 함, 연금술 시키면 채집 공백

매력 16, 통솔 15인 영웅
  → 고용 ★★★★★, 외교도 유용, 교역에서도 활약
  → 오늘 상인 왔는데 외교도 보내야 함... 누굴 어디에?

힘 18, 체력 5인 영웅
  → 사냥/건설/대장간 효율 최고, but 행동 하나만 해도 HP 바닥
  → 사냥 보내면 밤 전투 못 버팀
```

## 3. Phase 1 영웅 (7체)

영웅은 **랜덤 생성**. 이름, 죄종, 스탯이 랜덤으로 결정된다. 직업 없음 — 스탯과 장비가 역할을 정의.
- 이름: 랜덤 생성 (이름 풀에서 선택)
- 죄종: 7종 중 랜덤
- 기본 스탯: 7개 각 1~20 랜덤
- 독립 세부스탯: 7개 각 1~20 랜덤

## 4. 죄종별 만족/불만 조건

죄종이 곧 욕망. 별도의 갈망 없이, 죄종 자체가 사기 변동의 핵심 조건을 결정한다.

| 죄종 | 만족 조건 (사기 상승) | 불만 트리거 (사기 하락) |
|------|-------------------|---------------------|
| 교만 | 보스 원정 성공 | 3연속 원정 실패 |
| 탐욕 | 자원 투자받음 | 자원 배분 안 됨 |
| 분노 | 원정 파견 | 3일 이상 미파견 |
| 시기 | 본인 레벨업 | 타 영웅 더 성장 |
| 색욕 | 선호 동료 함께 배치 | 선호 동료 장기 분리 |
| 나태 | 시설 미배치 (휴식) | 연속 배치 |
| 폭식 | 장비 풀장착 | 빈 슬롯 |

## 5. 로스터 관리

| 항목 | 규칙 |
|------|------|
| 시작 인원 | **3명** (바알의 초기 부하) |
| 로스터 최대 | **7명** |
| 영입 방법 | **주점**에서 골드로 고용 (후보 중 선택) |
| 파티 편성 | 3명 (원정 시) |
| 해고 | 언제든 가능, 영구 제거 |
| 사망 | 영구 상실 |

### 시설 역할 변경
- **주점**: 영웅 고용 + 사기 상승 + 영웅 교류 (기존 역마차 영입 기능 흡수)
- **역마차**: 교역 상인 도착, 외부 정보 등

## 2-4. 이름 수식어 시스템

### 개요
영웅 이름은 **"수식어" 이름** 형태 (예: "뼈칼" 밀드, "고독한" 리아).
수식어는 영웅의 **독립 세부스탯 상위 2개 조합**(7C2 = 21가지)에 의해 결정된다.

### 수식어 테이블 (21 조합 × 3 후보 = 63개)

| # | 조합 | 후보 1 | 후보 2 | 후보 3 |
|---|------|--------|--------|--------|
| 1 | 공격성 + 욕심 | 약탈자 | 피묻은 손 | 탐욕의 칼날 |
| 2 | 공격성 + 자존심 | 폭군 | 부서진 왕좌 | 불굴의 |
| 3 | 공격성 + 호기심 | 광전사 | 피에 미친 | 전장의 바람 |
| 4 | 공격성 + 집요함 | 피바람 | 뼈칼 | 돌주먹 |
| 5 | 공격성 + 감수성 | 상처입은 | 불꽃눈물 | 핏빛 심장 |
| 6 | 공격성 + 독립심 | 외로운 칼 | 야수 | 붉은 늑대 |
| 7 | 욕심 + 자존심 | 황금왕 | 탐욕의 왕관 | 빛나는 |
| 8 | 욕심 + 호기심 | 보물사냥꾼 | 끝없는 탐색 | 황금눈 |
| 9 | 욕심 + 집요함 | 끝없는 | 채워지지않는 | 황금손 |
| 10 | 욕심 + 감수성 | 집착하는 | 놓지못하는 | 그림자 상인 |
| 11 | 욕심 + 독립심 | 도적 | 검은별 | 혼자 먹는 |
| 12 | 자존심 + 호기심 | 꺾이지않는 | 높은 곳의 | 별을 보는 |
| 13 | 자존심 + 집요함 | 왕관없는 | 무릎꿇지않는 | 강철의지 |
| 14 | 자존심 + 감수성 | 고고한 | 유리왕관 | 차가운 눈 |
| 15 | 자존심 + 독립심 | 쫓겨난 | 홀로선 | 왕좌를 버린 |
| 16 | 호기심 + 집요함 | 떠도는 | 끝을 모르는 | 길 위의 |
| 17 | 호기심 + 감수성 | 길잃은 | 바람따라 | 눈물의 여행자 |
| 18 | 호기심 + 독립심 | 외눈의 | 경계 너머 | 지도없는 |
| 19 | 집요함 + 감수성 | 깨지지않는 | 깨진방패 | 마지막까지 |
| 20 | 집요함 + 독립심 | 회색늑대 | 잿빛 | 끝까지 서는 |
| 21 | 감수성 + 독립심 | 고독한 | 버림받은 | 반쪽의 |

### 규칙
- 영웅 생성 시 7개 독립 세부스탯을 정렬 → **상위 2개** 추출
- 2개 조합에 해당하는 수식어 후보 3개 중 랜덤 1개 선택
- 이름 형식: `"수식어" 이름` (예: "피바람" 타론)
- 데이터: `src/data/hero_epithets.csv`

### 게임플레이 의미
- 수식어 = **세부스탯 상위 2개의 교차점**
- "약탈자" = 공격성 + 욕심 높음 → 전투 강하고 보상 많지만, 충돌 + 자원 갈등
- "고독한" = 감수성 + 독립심 높음 → 혼자 강하지만 관계에 흔들림
- 같은 죄종이라도 세부스탯에 따라 **완전히 다른 수식어** → 개성 부여
