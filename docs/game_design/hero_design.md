# 영웅 시스템 설계

> 상태: 구조 재설계 (죄종 수치화 + 특성 시스템 + primarySin 라벨 제거)
> 작성일: 2026-03-24

---

## 1. 개요

영웅은 **인간**이다. 용병, 모험가, 떠돌이 — 바알에게 고용된 자들.
각자 깊은 성격적 결함을 안고 있으며, 그 결함이 **죄종 수치**로 표현된다.

멀쩡한 인간은 악마 밑에서 일하지 않는다.
갈 곳 없는 자, 버림받은 자, 망가진 자들이 바알에게 모인다.

### 죄종 수치 = 성격 차원

기존의 "죄종 태그 1개" 방식을 폐기하고, **7죄종을 수치 스탯**으로 전환한다.
모든 영웅은 7가지 죄종 수치를 가지며, **주 성향 라벨(primarySin) 없음**. 7수치가 직접 확률/비례로 행동에 영향한다.

| 죄종 수치 | 높을 때 (성격) | 낮을 때 (성격) |
|---------|--------------|--------------|
| **분노** (공격성) | 충동적, 폭력적, 참을성 없음 | 온순, 전투 소극적 |
| **시기** (비교심) | 비교 강박, 열등감, 뒷공작 | 타인에 무관심, 자족 |
| **탐욕** (소유욕) | 독점욕, 소유욕, 나누지 못함 | 무욕, 나눔 |
| **나태** (무기력) | 무기력, 회피, 전염되는 체념 | 근면, 자발적 |
| **폭식** (탐닉) | 과잉, 불안한 비축, 끝없는 허기 | 절제, 소식 |
| **색욕** (집착) | 집착, 의존, 관계 중독 | 냉정, 독립적 |
| **교만** (자존) | 자기확신, 독선, 복종 거부 | 겸손, 순종적 |

Phase 1에서는 7체 구현. 랜덤 생성.

## 2. 영웅 구조

```
[영웅] — 랜덤 생성
├── 이름 = "수식어" + 이름 (죄종 수치 상위 2개 조합 → 21종 수식어 풀)
├── 죄종 7수치 (각 1~20, 랜덤) — 성격 + 만족/불만 + 폭주/이탈 + 영웅 간 궁합 + 수식어
├── 사기 (0~100) — 유일한 관리 수치
├── 기본 스탯 7개 (각 1~20, 랜덤) — 전투/작업 능력 결정
├── 파생 세부스탯 — 기본 스탯에서 산출 (행동 + 관계)
├── 특성 2~3개 (선천, 풀 45개에서 랜덤) — 행동 변주, 이벤트 분기
├── 후천 특성 슬롯 (행동 반복으로 획득, 풀 10개 + 후천 폭주 7종)
├── acquiredTraits[] — 후천 특성 배열 (폭주 이력 등으로 획득)
├── rampageHistory{} — 폭주 유형별 횟수 기록 (2회 반복 → 후천 특성 고정)
├── 히든: 성장률 + 상한선 (스탯별) — 사용 기반 성장
├── 고유 스킬 1개
└── 장비 3슬롯 — Phase 2
```

> **UI**: 영웅 고용 시 기본 스탯 + 죄종 수치 상위 2~3개 + 특성 표시. 영입 후 상세 확인 가능. (primarySin 라벨 없음)

## 2-1. 스탯 시스템

### 설계 원칙
- **죄종 수치 = 성격**, **기본 스탯 = 능력** — 완전 분리
- 죄종 수치와 기본 스탯 모두 랜덤 생성, 서로 독립
- 한 스탯이 여러 행동에 영향 → 배치 딜레마 발생
- UI에서 행동별 적합도를 별점으로 자동 표시

### 기본 스탯 7개 (각 1~20)

| 스탯 | 의미 | 영향 |
|------|------|------|
| **힘** | 물리적 파워 | 사냥, 건설, 대장간, 전투 |
| **민첩** | 빠르기, 손재주 | 사냥, 채집, 연금술, 전투 |
| **지능** | 머리, 판단력 | 외교, 연구(단독), 교역, 전투 |
| **체력** | 지구력, 내구 | 전투 HP풀 + 부상 회복 속도 |
| **감각** | 눈썰미, 직감 | 채집, 대장간, 연금술 |
| **통솔** | 지휘, 리더십 | 건설, 고용, 통솔력(관계), 전투 |
| **매력** | 영향력, 교섭 | 외교, 교역, 고용, 매력도(관계) |

### 체력(HP) 시스템

- **전투 전용 HP** — 낮 행동(건설, 연구 등)으로는 소모되지 않음
- HP 공식: `hp_base(200) + 체력×hp_vitality_mult(15) + 힘×hp_strength_mult(5)` (balance.csv)
- 전투 후 부상(HP 0) 상태로 귀환 → 매 턴 **자동 회복** (체력 스탯이 높을수록 회복량↑)
- **병원**에서 치료 시 회복 속도 2배, **치료약** 사용 시 즉시 회복
- 체력 스탯의 의미 = **"전투에서 얼마나 오래 버티는가 + 부상 회복 속도"**

| 체력 스탯 | 전투 HP풀 | 턴당 자동 회복 | 병원 치료 |
|----------|----------|-------------|---------|
| 낮음 (1~7) | 낮음 | 적게 회복 | 2배 속도 |
| 보통 (8~14) | 보통 | 보통 회복 | 2배 속도 |
| 높음 (15~20) | 높음 | 많이 회복 | 2배 속도 |

### 행동별 스탯 매핑

모든 행동은 **동등 가중치(50:50)**. 행동 효율 = (스탯A + 스탯B) / 2. 연구만 지능 단독.

|  | 야외 |||| 실내 ||||||
|---|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|
|  | 전투 | 사냥 | 채집 | 외교 | 건설 | 대장간 | 연금술 | 연구 | 교역 | 고용 |
| **힘** | ? | ● | | | ● | ● | | | | |
| **민첩** | ? | ● | ● | | | | ● | | | |
| **지능** | ? | | | ● | | | | ● | ● | |
| **감각** | ? | | ● | | | ● | ● | | | |
| **통솔** | ? | | | | ● | | | | | ● |
| **매력** | ? | | | ● | | | | | ● | ● |

전투: 힘+민첩(원정), 힘+통솔(방어). 체력은 HP풀+회복 속도.

### 파생 세부스탯 — 관계 계열

| 세부스탯 | 산출 | 관계에서의 역할 |
|---------|------|---------------|
| **통솔력** | 통솔 | 다른 영웅이 이 영웅을 **따름** (존경) |
| **매력도** | 매력 | 다른 영웅이 이 영웅을 **좋아함** (호감) |
| **민감도** | 색욕 수치 (죄종) | 이 영웅이 다른 영웅에게 **민감하게 반응** |

> 민감도는 기본 스탯 감각이 아닌, 죄종 수치 중 색욕(lust)에서 산출.

## 2-2. 죄종 수치 (성격 시스템)

### 설계 원칙
- 기존 "독립 세부스탯 7개"를 **죄종 7수치**로 교체
- 기본 스탯과 무관, 랜덤 생성 (각 1~20)
- **높다고 무조건 좋지 않음** — 양면성을 가짐
- **주 성향 라벨(primarySin) 없음** — 7수치가 직접 확률/비례로 행동에 영향
- 폭주/이탈 행동, 만족/불만 조건 모두 수치 비례로 판정
- 죄종 수치 간 관계 매트릭스(생성/강화/대립)가 영웅 내부+영웅 간 모두 적용

### 7개 죄종 수치

| 죄종 수치 | 높으면 | 낮으면 |
|---------|-------|-------|
| **분노** (공격성) | 전투 공격력↑, 동료 충돌 잦음 | 전투 소극적, 거점 내 평화 |
| **탐욕** (소유욕) | 보상 획득량↑, 자원 갈등 | 수확 적지만 나눔 |
| **교만** (자존) | 리더 시 능력↑, 부하 역할 시 사기↓ | 어디든 불만 없이 배치 |
| **시기** (비교심) | 탐색/연구 보너스, 위험 이벤트 조우↑ | 안전하지만 발견 기회 적음 |
| **폭식** (탐닉) | 장기 작업 보너스, 중단/철수 판단 느림 | 판단 빠르지만 오래 못 버팀 |
| **색욕** (집착) | 관계 형성 빠름, 동료 사기에 같이 흔들림 | 냉정, 연쇄 전염 저항 |
| **나태** (무기력) | 혼자 행동 효율↑, 팀 시너지↓ | 팀에서 잘 맞지만 혼자선 약함 |

### 행동 결정 방식 (특성 우선 → 확률 → 이력 고정)

primarySin 라벨을 사용하지 않는다. 폭주/이탈/만족/불만 등 모든 행동은 다음 3단계로 결정:

1. **특성 우선** — 행동 강박 특성("방화광", "파괴충동", "도벽" 등)이 있으면 해당 특성의 강박 행동이 **우선 발동**
2. **sinStats 가중 확률** — 특성으로 결정되지 않으면, 7개 죄종 수치를 가중치로 한 **확률 판정**으로 행동 결정 (예: 분노 15, 탐욕 8이면 분노형 행동 확률이 약 2배)
3. **이력 누적 → 후천 특성 고정** — 같은 유형의 폭주가 2회 반복되면 후천 특성("후천 폭주: X") 획득, 이후 해당 유형이 **고정 발동**

> 수식어 결정에는 여전히 상위 2개 죄종 조합을 사용 (이름 생성용, 행동 판정과 무관)
> 예: 분노+탐욕 높음 = "약탈자" 수식어 (전투 + 자원 집착 성격)

## 2-3. 특성 시스템

### 개요
영웅은 **선천 특성 2~3개**와 **후천 특성 1개(슬롯)**를 가진다.
죄종 수치가 "어떤 성격인가"를 정하고, 특성이 "그걸 어떤 방식으로 표현하는가"를 정한다.

### 특성 카테고리

| 카테고리 | 개수 | 비율 | 설명 |
|---------|------|------|------|
| 양면성 | 28개 | 51% | 장점+단점 동시 보유 (메인) |
| 순수 장점 | 6개 | 11% | 장점만 |
| 순수 단점 | 6개 | 11% | 단점만 |
| 행동 강박 | 5개 | 9% | 평소 보너스 + 특정 상황 강박 행동 |
| 후천 획득 | 10개 | 18% | 행동 반복으로 획득 |

### 선천 특성 (생성 시 랜덤 2~3개, 풀 45개)

#### 양면성 특성

| 특성 | 장점 | 단점 |
|------|------|------|
| 호전적 | 전투 데미지 +15% | 3일 미파견 시 사기↓↓ |
| 겁쟁이 | 위험 감지(기습 방지), 정찰↑ | 방어전 배치 시 사기↓, 도주 확률↑ |
| 잔인한 | 전투 승리 시 사기↑↑ | 동료 부상/사망에 무반응 (관계↓) |
| 자비로운 | 동료 치료 효율↑, 관계↑ | 적 처치 시 사기 소폭↓ |
| 낙천적 | 사기 하락폭 -30% | 사기 상승폭도 -20% (둔감) |
| 변덕쟁이 | 사기 상승폭 +50% | 사기 하락폭도 +50% |
| 독불장군 | 단독 원정/사냥 보너스 | 팀 시너지↓, 같은 시설 배치 효과↓ |
| 사교적 | 연회 효과↑, 영웅 간 관계 보너스 | 혼자 배치 시 사기↓ |
| 완벽주의자 | 건설/연구 품질↑ | 건설/연구 속도↓ |
| 대충대충 | 건설/연구 속도↑ | 시설 고장/연구 실패 확률 소폭↑ |
| 대식가 | 체력 회복↑, 전투 HP↑ | 식량 소비 +50% |
| 금욕적 | 자원 소비↓, 불만 조건 완화 | 고양 상태 도달 어려움 |
| 야행성 | 밤 방어전 스탯↑ | 낮 작업 효율↓ |
| 조기형 | 낮 작업 효율↑ | 밤 방어전 스탯↓ |
| 수집광 | 채집 효율↑ | 장비 해제 시 사기↓↓ |
| 방랑벽 | 원정 사기↑, 원정 보상↑ | 거점 연속 3일 배치 시 사기↓ |
| 집착적 | 반복 작업 효율 누적↑ | 작업 변경 시 사기↓ |
| 시비꾼 | 대립 이벤트 촉발 → 드라마 | 같은 시설 영웅 사기↓ |
| 허영심 | 장비 장착 시 사기↑↑ | 다른 영웅보다 장비 나쁘면 사기↓ |
| 의리있는 | 동료 폭주 시 사기하락 면역 | 동료 이탈/사망 시 사기↓↓↓ |
| 질투쟁이 | 경쟁 대상보다 강해지면 사기↑↑ | 타 영웅 승진/장비 획득 시 사기↓ |
| 신중한 | 전투 피해 -15%, 함정 회피 | 선제공격 불가, 이니셔티브↓ |
| 무모한 | 선제공격↑, 이니셔티브↑ | 전투 피해 +15%, 함정 취약 |
| 카리스마 | 같은 파티 영웅 사기↑ | 본인 폭주 시 주변 영향 범위↑ |
| 둔감한 | 폭주/이탈 임계값 넓음 (안정적) | 이벤트 반응↓, 드라마 비참여 |
| 예민한 | 이벤트 보정↑, 연쇄 반응 참여 | 폭주/이탈 임계값 좁음 |
| 탐험가 | 미개척 구역 개척 속도↑ | 이미 해금된 구역 작업 효율↓ |
| 텃새 | 거점 시설 작업 효율↑ | 원정 파견 시 사기↓ |

| 특성 | 효과 |
|------|------|
| 허약한 | 전투 피해 +20% |

| 특성 | 평소 보너스 | 강박 행동 |
|------|-----------|----------|
| 방화광 | 화염 전투↑ | 폭주 시 시설 방화 |
| 도벽 | 채집/전리품↑ | 자원 획득 시 일부 착복 |
| 파괴충동 | 전투 크리티컬↑ | 사기 불만 구간에서 시설 파손 |
| 중재자 | 영웅 간 대립 완화 | 분쟁 발생 시 강제 개입 (본인 사기 소모) |
| 과시욕 | 전투 성공 시 전체 사기↑ | 실패 시 전체 사기↓ (체면 손상) |

### 후천 특성 (행동 반복으로 획득, 슬롯 1개)

| 특성 | 획득 조건 | 효과 |
|------|----------|------|
| 역전의 용사 | 원정 5회+ 승리 | 원정 전투 데미지 +10%, 전투 사기↑ |
| 수호자 | 방어전 3회+ 생존 | 방어전 스탯↑, 방어 배치 사기↑ |
| 노동의 기쁨 | 채집/벌목 10회+ | 자원 작업 효율↑, 작업 시 사기 소폭↑ |
| 연구자 | 연구 5회+ 참여 | 연구 속도↑ |
| 건축가 | 건설 5회+ 참여 | 건설 속도↑, 건설 비용↓ |
| 사냥꾼 | 사냥 5회+ 성공 | 사냥 성공률↑, 전리품↑ |
| 지옥 생존자 | 폭주 2회 경험 | 사기 변동폭↑↑, 전투력↑ |
| 개심자 | 구원 달성 | 사기 안정, 고유 능력 해금 |
| 타락자 | 타락 달성 | 능력 강화, 폭주 재발 확률↑ |
| 전우 | 같은 영웅과 원정 5회+ | 해당 영웅과 시너지↑, 사망 시 사기↓↓↓ |

#### 후천 폭주 특성 (7종, 폭주 이력 2회 반복으로 획득)

같은 유형의 폭주가 2회 반복되면 자동 획득. 이후 해당 유형의 폭주가 **고정 발동**된다.

| 특성 | 획득 조건 | 효과 |
|------|----------|------|
| 후천 폭주: 분노 | 분노형 폭주 2회 | 폭주 시 항상 건물 파괴, 전투 공격력 보정↑ |
| 후천 폭주: 탐욕 | 탐욕형 폭주 2회 | 폭주 시 항상 자원 도난, 보상 획득↑ |
| 후천 폭주: 교만 | 교만형 폭주 2회 | 폭주 시 항상 명령 거부, 단독 행동 효율↑ |
| 후천 폭주: 시기 | 시기형 폭주 2회 | 폭주 시 항상 장비 파괴, 탐색 보정↑ |
| 후천 폭주: 폭식 | 폭식형 폭주 2회 | 폭주 시 항상 식량 소진, 체력 회복↑ |
| 후천 폭주: 나태 | 나태형 폭주 2회 | 폭주 시 항상 행동 거부, 사기 전염 범위↑ |
| 후천 폭주: 색욕 | 색욕형 폭주 2회 | 폭주 시 항상 분쟁 유발, 관계 형성 속도↑ |

### 특성 ↔ 죄종 수치 관계
- 특성은 죄종 수치와 **독립적**으로 랜덤 부여 (연동 없음)
- 단, 조합에 따라 시너지/충돌 발생:
  - "호전적" + 분노 높음 = 전투 보너스 크지만 폭주 위험도 극대
  - "금욕적" + 폭식 높음 = 내적 갈등, 특수 이벤트 가능
  - "겁쟁이" + 분노 높음 = 겉으로는 싸우고 싶지만 두려운 캐릭터

### 레퍼런스
- RimWorld: 특성 1~3개, 양면성 50% + 순장/순단 혼합
- Darkest Dungeon: 긍정/부정 벽 분리, 후천 획득, 치료 가능
- CK3: 거의 전부 양면성, 대립쌍, 상황별 가치 변동
- Battle Brothers: 전투/사기 연동 특성

## 2-4. 성장 시스템

### 사용 기반 성장
- 기본 스탯과 죄종 수치 모두 사용을 통해 성장 가능
- 모든 스탯의 기본 범위는 **1~20**, 성장으로 **20을 초과**할 수 있음

### 히든 파라미터 (플레이어에게 비공개)

| 파라미터 | 의미 | 비고 |
|---------|------|------|
| **성장률** (스탯별) | 해당 스탯 상승 속도 | 같은 행동을 해도 영웅마다 성장 속도 다름 |
| **상한선** (스탯별) | 해당 스탯 최대 도달치 | 재능의 한계 — 상한이 20 이상이면 초월 성장 가능 |

> 플레이어는 수치를 볼 수 없고, 행동 결과를 관찰하며 추론한다.
> "이 영웅 사냥 계속 보냈더니 힘이 쑥쑥 오르네" → 재능 있는 영웅
> "이 영웅은 건설 시켜도 힘이 안 오르네..." → 재능 없는 영웅

### 딜레마 예시

```
힘 16, 감각 15인 영웅
  → 대장간 ★★★★★, 사냥도 강함
  → 장비 만들게 할까? 사냥 보낼까?

민첩 17, 감각 14인 영웅
  → 채집 ★★★★★, 연금술 ★★★★★
  → 채집 보내면 연금술 못 함, 연금술 시키면 채집 공백

매력 16, 통솔 15인 영웅
  → 고용 ★★★★★, 외교도 유용, 교역에서도 활약
  → 오늘 상인 왔는데 외교도 보내야 함... 누굴 어디에?

힘 18, 체력 5인 영웅
  → 사냥/건설/대장간 효율 최고, but 행동 하나만 해도 HP 바닥
  → 사냥 보내면 밤 전투 못 버팀
```

## 3. Phase 1 영웅 (7체)

영웅은 **랜덤 생성**. 이름, 죄종 수치, 스탯, 특성이 랜덤으로 결정된다. 직업 없음 — 스탯과 장비가 역할을 정의.
- 이름: 랜덤 생성 (이름 풀에서 선택) + 죄종 상위 2개 조합 수식어
- 죄종 수치: 7개 각 1~20 랜덤 (primarySin 라벨 없음, 수치 비례 확률 판정)
- 기본 스탯: 7개 각 1~20 랜덤
- 선천 특성: 풀 45개에서 2~3개 랜덤
- 후천 특성: 슬롯, 행동 반복으로 획득 (풀 10개 + 후천 폭주 7종)
- acquiredTraits[]: 후천 특성 배열
- rampageHistory{}: 폭주 유형별 횟수 기록

## 4. 죄종 수치별 만족/불만 조건

모든 죄종 수치가 **비례 강도**로 만족/불만에 영향한다. 수치가 높을수록 해당 조건의 사기 변동폭이 커진다. (고정 주 성향 라벨 없음)

| 죄종 | 만족 조건 (사기 상승) | 불만 트리거 (사기 하락) |
|---------|-------------------|---------------------|
| 교만 | 보스 원정 성공 | 3연속 원정 실패 |
| 탐욕 | 자원 투자받음 | 자원 배분 안 됨 |
| 분노 | 원정 파견 | 3일 이상 미파견 |
| 시기 | 본인 레벨업 | 타 영웅 더 성장 |
| 색욕 | 선호 동료 함께 배치 | 선호 동료 장기 분리 |
| 나태 | 시설 미배치 (휴식) | 연속 배치 |
| 폭식 | 장비 풀장착 | 빈 슬롯 |

> 예: 분노 15 + 탐욕 8인 영웅은 원정 파견 시 분노 비례 강한 만족 + 자원 배분 시 탐욕 비례 약한 만족. 모든 수치가 동시에 작용.

## 2-5. 이름 수식어 시스템

### 개요
영웅 이름은 **"수식어" 이름** 형태 (예: "뼈칼" 밀드, "고독한" 리아).
수식어는 영웅의 **죄종 수치 상위 2개 조합**(7C2 = 21가지)에 의해 결정된다.

### 규칙
- 영웅 생성 시 7개 죄종 수치를 정렬 → **상위 2개** 추출
- 2개 조합에 해당하는 수식어 후보 3개 중 랜덤 1개 선택
- 이름 형식: `"수식어" 이름` (예: "피바람" 타론)
- 데이터: `src/data/hero_epithets.csv`

### 게임플레이 의미
- 수식어 = **죄종 수치 상위 2개의 교차점**
- "약탈자" = 분노 + 탐욕 높음 → 전투 강하고 보상 많지만, 충돌 + 자원 갈등
- "고독한" = 색욕 + 나태 높음 → 혼자 강하지만 관계에 흔들림
- 같은 주 성향이라도 부 성향에 따라 **완전히 다른 수식어** → 개성 부여
