# 영웅 시스템 설계

> 상태: 구조 확정, 수치 미확정
> 작성일: 2026-03-24

---

## 1. 개요

영웅은 **인간**이다. 용병, 모험가, 떠돌이 — 바알에게 고용된 자들.
각자 깊은 성격적 결함을 안고 있으며, 그 결함이 곧 **죄종 태그**다.

멀쩡한 인간은 악마 밑에서 일하지 않는다.
갈 곳 없는 자, 버림받은 자, 망가진 자들이 바알에게 모인다.

### 죄종별 인간 결함

| 죄종 | 인간으로서의 결함 | 예시 |
|------|----------------|------|
| 분노 | 충동적, 폭력적, 참을성 없음 | 전쟁에서 돌아온 용병, 복수에 사로잡힌 검사 |
| 시기 | 비교 강박, 열등감, 뒷공작 | 인정받지 못한 2인자, 형제의 그림자에 산 궁수 |
| 탐욕 | 독점욕, 소유욕, 나누지 못함 | 모든 것을 잃은 상인, 보물을 찾는 도적 |
| 나태 | 무기력, 회피, 전염되는 체념 | 전쟁에 지친 기사, 재능 있지만 포기한 학자 |
| 폭식 | 과잉, 불안한 비축, 끝없는 허기 | 굶주림을 겪은 유민, 채워지지 않는 전사 |
| 색욕 | 집착, 의존, 관계 중독 | 사랑에 실패한 시인, 혼자가 되는 게 두려운 치유사 |
| 교만 | 자기확신, 독선, 복종 거부 | 왕좌에서 쫓겨난 지휘관, 자기가 옳다고 믿는 성직자 |

Phase 1에서는 7체 (죄종당 1체)만 구현. 랜덤 생성.

## 2. 영웅 구조

```
[영웅] — 랜덤 생성
├── 이름 / 직업 (5종)
├── 죄종 1개 (고정) — 성격 + 만족/불만 조건 + 폭주/이탈 행동 결정
├── 사기 (0~100) — 유일한 관리 수치
├── 7스탯 (각 1~20, 랜덤) — 죄종과 무관하게 개별 능력 결정
├── 고유 스킬 1개
├── 특성풀 3개 (고유 1 + 죄종 2) — Phase 2
└── 장비 3슬롯 — Phase 2
```

## 2-1. 스탯 시스템

### 설계 원칙
- **죄종 = 성격**, **스탯 = 능력** — 완전 분리
- 죄종이 스탯을 결정하지 않음. 랜덤으로 굴림
- 한 스탯이 여러 행동에 영향 → 배치 딜레마 발생
- UI에서 행동별 적합도를 별점으로 자동 표시

### 7스탯 (각 1~20)

| 스탯 | 의미 | 영향받는 행동 |
|------|------|-------------|
| **힘** | 물리적 파워 | 전투 공격, 건설 속도, 방어전 |
| **민첩** | 빠르기, 손재주 | 전투 행동순서, 생산 품질, 함정 제작 |
| **지능** | 머리, 판단력 | 연구 속도, 전투 AI 품질, 탐색(전리품) |
| **체력** | 지구력, 내구 | 전투 생존, 건설 지구력, 연속 행동 |
| **감각** | 눈썰미, 직감 | 탐색(숨겨진 루트), 생산 품질, 감시탑 효율 |
| **통솔** | 지휘, 리더십 | 방어전 팀 버프, 사기 전염 저항, 원정 파티 버프 |
| **매력** | 영향력, 교섭 | 이벤트 추가 선택지, 교역 가격, 영웅 간 관계 |

### 행동별 스탯 가중치

| 행동 | 핵심 스탯 | 보조 스탯 |
|------|----------|----------|
| 원정 전투 | 힘, 민첩 | 체력, 지능 |
| 밤 방어 | 힘, 통솔 | 체력 |
| 건설 | 힘, 체력 | - |
| 연구 | 지능, 감각 | - |
| 생산 | 민첩, 감각 | 지능 |
| 탐색 (원정 보상) | 감각, 지능 | - |
| 이벤트/교역 | 매력, 지능 | 통솔 |
| 사기 영향 | 통솔, 매력 | - |

### 딜레마 예시

```
힘 16, 체력 14인 영웅
  → 전투 ★★★★★, 건설 ★★★★★
  → 원정 보내면 전투 에이스, 근데 건설에도 최적
  → 어디에 쓸까?

지능 15, 감각 14인 영웅
  → 연구 ★★★★★, 생산 ★★★★, 탐색 ★★★★★
  → 연구 시킬까? 원정 보내서 전리품 늘릴까?

통솔 16, 매력 15인 영웅
  → 방어전 핵심 (팀 버프), 이벤트에서도 유용
  → 원정 보내면 파티 버프... 밤 방어는 약해짐
```

## 3. Phase 1 영웅 (7체)

영웅은 **랜덤 생성**. 이름, 직업, 죄종이 랜덤으로 결정된다.
- 이름: 랜덤 생성 (이름 풀에서 선택)
- 직업: 5종 중 랜덤
- 죄종: 7종 중 랜덤
- 스탯: 7스탯 각 1~20 랜덤

## 4. 죄종별 만족/불만 조건

죄종이 곧 욕망. 별도의 갈망 없이, 죄종 자체가 사기 변동의 핵심 조건을 결정한다.

| 죄종 | 만족 조건 (사기 상승) | 불만 트리거 (사기 하락) |
|------|-------------------|---------------------|
| 교만 | 보스 원정 성공 | 3연속 원정 실패 |
| 탐욕 | 자원 투자받음 | 자원 배분 안 됨 |
| 분노 | 원정 파견 | 3일 이상 미파견 |
| 시기 | 본인 레벨업 | 타 영웅 더 성장 |
| 색욕 | 선호 동료 함께 배치 | 선호 동료 장기 분리 |
| 나태 | 시설 미배치 (휴식) | 연속 배치 |
| 폭식 | 장비 풀장착 | 빈 슬롯 |

## 5. 로스터 관리

| 항목 | 규칙 |
|------|------|
| 시작 인원 | **3명** (바알의 초기 부하) |
| 로스터 최대 | **7명** |
| 영입 방법 | **주점**에서 골드로 고용 (후보 중 선택) |
| 파티 편성 | 3명 (원정 시) |
| 해고 | 언제든 가능, 영구 제거 |
| 사망 | 영구 상실 |

### 시설 역할 변경
- **주점**: 영웅 고용 + 사기 상승 + 영웅 교류 (기존 역마차 영입 기능 흡수)
- **역마차**: 교역 상인 도착, 외부 정보 등
