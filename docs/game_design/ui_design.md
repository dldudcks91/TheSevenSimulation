## 2. 화면 전환 흐름

```
[타이틀] → 새 게임 → [영웅 선택] → 여정 시작
                                      ↓
                      ┌─→ [아침: 이벤트] → 선택지 고르면
                      │       ↓
                      │   [낮: 거점 메인] → "턴 종료" 누르면
                      │       ↓
                      │   [저녁: 원정 결과] → 확인 (원정 없으면 스킵)
                      │       ↓
                      │   [밤: 방어전] → 자동 진행
                      │       ↓
                      │   [밤: 결산] → "다음 날" 누르면
                      └───────┘

[타이틀] → 이어하기 → [낮: 거점 메인] (세이브 복원)
```

## 4. 각 화면 상세

### 4-1. 타이틀 (TitleScene)

- 7죄종 심볼 원형 배치 + 펄스 애니메이션
- 제목 글로우 + 순차 페이드인 연출
- RPG 베벨 버튼: 새 게임 / 이어하기
- 배경: 붉은 파티클 + 회색 재 + 비네팅

### 4-2. 영웅 선택 (HeroSelectScene)

```
┌────────────────────────────────────────────┐
│              동 료  선 택                   │
│   바알과 함께할 세 명의 부하를 선택하십시오    │
├────────────┬────────────┬──────────────────┤
│  [카드 1]  │  [카드 2]  │  [카드 3]        │
│   이름     │   이름     │   이름           │
│  [죄종]    │  [죄종]    │  [죄종]          │
│  "결함"    │  "결함"    │  "결함"          │
│  ─능력치─  │  ─능력치─  │  ─능력치─        │
│  힘 ████16 │  힘 ██ 8  │  힘 ███12        │
│  민 ███ 12 │  민 ████15 │  ...             │
│  ...       │  ...       │                  │
├────────────┴────────────┴──────────────────┤
│      [🎲 다시 뽑기]    [▶ 여정 시작]        │
└────────────────────────────────────────────┘
```

- 영웅 카드: 이름, 죄종(색상), 결함 설명, 7스탯 바 차트
- 스탯 바 색상: 15+ 초록, 10+ 파랑, 7+ 노랑, 6- 빨강
- "다시 뽑기" → 3명 랜덤 재생성 (제한 없음)
- "여정 시작" → 확정 후 MainScene으로 페이드 전환

### 4-3. 거점 메인 (MainScene)

#### 네비게이션 바

```
THE SEVEN │ Day 12 │ [낮] │ 450G │ 방어:4명 │ [턴종료] [저장] ⚙
```

- 좌측: 게임 로고, Day, 페이즈, 골드, 방어 인원 (구분선 분리)
- 우측: 턴 종료 버튼, 저장 버튼, 설정 아이콘(⚙)

#### 거점 맵 (좌측) — 영내

- 3x3 건물 슬롯 그리드 + 중앙 광장 (바알)
- 성벽 둘러싸기 (RPG 베벨)

#### 거점 맵 (좌측) — 영외

- 3장 카드: 🌿채집 / ⚔️원정 / 🏹사냥
- 카드 클릭 → 우측 패널에 상세 행동 표시

#### 우측 탭 패널

| 탭 | 아이콘 | 콘텐츠 |
|---|---|---|
| 거점 | 🏰 | 시설 현황, 건설/연구 상태, 수입, 포고령 효과 |
| 영웅 | ⚔ | 영웅 로스터 (이름, 죄종, 스탯, 사기바, 상태) |
| 장비 | 🛡 | Phase 2 예정 |
| 원정 | 🗺 | 챕터 진행, 스테이지 목록, 감시탑 정보, 파견 가능 수 |
| 정책 | 📜 | 포고령 3종 (배급/훈련/경계) 라디오 선택 |

- 탭 활성: 하단 빨간 인디케이터 + 밝은 배경
- 탭 전환 시 콘텐츠 영역만 교체 (동적 생성/삭제)

#### 맵 조작 (영내) → 중앙 팝업

| 클릭 대상 | 결과 |
|---|---|
| 빈 슬롯 | **중앙 팝업** → 건설 목록 + 영웅 선택 |
| 지어진 건물 | **중앙 팝업** → 시설 행동 (고용/연회/안정화) |
| 광장 (바알) | **중앙 팝업** → 포고령 |

> 영내 건물 관련 행동은 모두 화면 중앙에 팝업 오버레이로 표시 (반투명 검은 배경 + 560×520 패널)

#### 맵 조작 (영외) → 우측 패널

| 클릭 대상 | 결과 |
|---|---|
| 원정 카드 | 우측 패널 → 스테이지 목록 → 병사 배정 |
| 사냥 카드 | 우측 패널 → 사냥 영웅 선택 |
| 채집 카드 | 우측 패널 → 채집 영웅 선택 |

### 4-5. 원정 결과 (ResultScene, 오버레이)

- 승패 표시, 파티 HP 상태, 전리품
- 전투 리플레이 버튼
- 감시탑 정보 (레벨별: 유무 → 규모 → 적 상세)

### 4-6. 방어전 (BattleScene)

- 좌측 아군 / 중앙 성벽 / 우측 적군 스프라이트 배치
- 공격 애니메이션 (돌진+복귀), 피격 (흔들림+데미지 팝업)
- HP 바 실시간 감소, 쓰러짐 애니메이션
- 배속 (x1/x2/x4) + 스킵 버튼
- 하단 전투 로그

### 4-7. 결산 (SettlementScene, 오버레이)

- 방어 결과 (승리/패배)
- 영웅별 사기 현황 (죄종 색상, 경고 표시)
- 연쇄 반응 발생 시 순차 표시 (폭주→타락/구원→영향)
- "다음 날" 버튼

### 4-8. 게임오버 (GameOverScene)

- 패배: GAME OVER + 일차 + 사유
- 승리: CHAPTER CLEAR + 보스 유언

## 5. 비주얼 스타일 (TheSevenRPG 통일)

### RPG 3D 베벨

```
Outset (튀어나온 패널):
  inset -2px -2px dark, inset 2px 2px highlight, 0 0 0 2px border

Inset (눌린 입력):
  inset 2px 2px dark, inset -2px -2px highlight, 0 0 0 2px border
```
