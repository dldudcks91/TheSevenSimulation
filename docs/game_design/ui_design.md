```
[타이틀] → 새 게임 → [인트로] → [영웅 선택] → 여정 시작
                                      ↓
                            ┌─→ [MapScene] (단일 맵 — 모든 페이즈 처리)

                            │   [아침] → MorningReportPopup (심리 보고) → EventScene 오버레이

                            │   [낮] → 맵 직접 조작 (건설/원정/사냥) → "턴 종료"
```

## 3. 전체 레이아웃 구조 (MapScene)

```
┌─── 네비게이션 바 (32px) ────────────────────────────────────────────┐
│ THE SEVEN │ Day 12 │ [낮] │ 🌾120 🪵50 💰500 │ 방어:4명 │ [턴종료] [저장] ⚙ │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  ◀ [영내: 건물 3x3]     [성문]       [야외: 원정/사냥] ▶            │
│     건물·영웅 스프라이트   ←→ 드래그 스크롤 →   카드·원정지도         │
│                       맵 영역 (1280 x 440)                          │
│                       월드 크기: 2560 x 440                         │
│                                                                     │
├──────────────────────────────┬──────────────────────────────────────┤
│ [🏰시설][⚔영웅][🎒아이템][🗺원정][📜정책][📖도감] │ 탭 콘텐츠 영역       │
│                              │  맵 클릭 시 해당 정보 표시            │
│    하단 UI 패널 (1280 x 248) │  행동 버튼 (건설/배정/파견 등)        │
└──────────────────────────────┴──────────────────────────────────────┘
```

### 영역별 크기 (1280x720)

| 영역 | 크기 | 설명 |
|---|---|---|
| 네비게이션 바 | 1280 x 32 | 상단 고정, 게임 정보 + 버튼 |
| 맵 영역 (뷰포트) | 1280 x 440 | 가로 스크롤, 카메라 뷰 |
| 맵 월드 | 2560 x 440 | 실제 맵 크기 (영내~성문~야외) |
| 하단 UI 패널 | 1280 x 248 | 탭 (32px) + 콘텐츠 (216px), 고정 |

### 맵 월드 구성 (2560 x 440)

| 구간 | X 범위 | 내용 |
|---|---|---|
| 영내 | 0 ~ 900 | 성벽 + 건물 3x3 그리드 + 광장(바알) |
| 성문 | 900 ~ 1100 | 성문 장식, 영내↔영외 경계 |
| 야외 | 1100 ~ 2560 | 숲/들판 + 원정/사냥/채집 카드 |

### 카메라 스크롤

- 뷰포트: 1280 x 440 (Y: 32 ~ 472)
- 월드 바운드: 0 ~ 2560 (X), 0 ~ 440 (Y)
- 드래그로 좌우 스크롤 (맵 영역 내에서만)
- 하단 UI 패널은 카메라 영향 없음 (고정 위치)

## 4. 각 화면 상세

### 4-1. 타이틀 (TitleScene)

- 7죄종 심볼 원형 배치 + 펄스 애니메이션
- 제목 글로우 + 순차 페이드인 연출
- RPG 베벨 버튼: 새 게임 / 이어하기
- 배경: 붉은 파티클 + 회색 재 + 비네팅

### 4-2. 영웅 선택 (HeroSelectScene)

```
┌────────────────────────────────────────────┐
│              동 료  선 택                   │
│   바알과 함께할 세 명의 부하를 선택하십시오    │
├────────────┬────────────┬──────────────────┤
│  [카드 1]  │  [카드 2]  │  [카드 3]        │
│   이름     │   이름     │   이름           │
│  [죄종]    │  [죄종]    │  [죄종]          │
│  "결함"    │  "결함"    │  "결함"          │
│  ─능력치─  │  ─능력치─  │  ─능력치─        │
│  힘 ████16 │  힘 ██ 8  │  힘 ███12        │
│  민 ███ 12 │  민 ████15 │  ...             │
│  ...       │  ...       │                  │
├────────────┴────────────┴──────────────────┤
│      [🎲 다시 뽑기]    [▶ 여정 시작]        │
└────────────────────────────────────────────┘
```

- 영웅 카드: 이름, 죄종(색상), 결함 설명, 7스탯 바 차트
- 스탯 바 색상: 15+ 초록, 10+ 파랑, 7+ 노랑, 6- 빨강
- "다시 뽑기" → 3명 랜덤 재생성 (제한 없음)
- "여정 시작" → 확정 후 MapScene으로 페이드 전환

### 4-3. 맵 씬 (MapScene) — 메인 게임플레이

#### 네비게이션 바 (32px)

```
THE SEVEN │ Day 12 │ [낮] │ 🌾120 🪵50 💰500 │ 방어:4명 │ [턴종료] [저장] ⚙
```

- 좌측: 게임 로고, Day, 페이즈, 자원(식량/나무/돈), 방어 인원 (구분선 분리)
- 우측: 턴 종료 버튼, 저장 버튼, 설정 아이콘(⚙)

#### 맵 영역 (1280x440 뷰포트, 2560x440 월드)

- **영내 (0~900)**: 성벽 + 5x5 건물 그리드 (초기 3x3 해금, 나머지 잠금) + 광장(바알)
- **성문 (900~1100)**: 경계 표시, 장식
- **야외 (1100~2560)**: 숲 배경 + 원정/사냥/채집 카드
- 드래그 스크롤로 영내↔야외 전환 (영내/영외 전환 탭 제거)
- 건물/카드 클릭 → 하단 패널에 행동 UI 표시

#### 하단 UI 패널 (1280x248)

| 탭 | 아이콘 | 콘텐츠 |
|---|---|---|
| 시설 | 🏰 | 시설 현황, 건설/연구 상태, 수입, 포고령 효과 |
| 영웅 | ⚔ | 영웅 로스터 (이름, 죄종, 스탯, 사기바, 상태) |
| 아이템 | 🎒 | 아이템 목록 (준비 중) |
| 원정 | 🗺 | 챕터 진행, 스테이지 목록, 감시탑 정보, 파견 가능 수 |
| 정책 | 📜 | 포고령 3종 (배급/훈련/경계) 라디오 선택 |
| 도감 | 📖 | 몬스터 도감 — 전 몬스터 스프라이트 + 이름 카드 표시 |

- 탭 바: 상단 32px, 6개 탭
- 콘텐츠 영역: 216px
- 탭 활성: 하단 빨간 인디케이터 + 밝은 배경
- 맵 클릭과 연동 (건물 클릭 → 시설 탭, 원정 카드 → 원정 탭)

#### 맵 조작 → 하단 패널 연동

| 클릭 대상 | 결과 |
|---|---|
| 빈 슬롯 (해금) | **중앙 팝업** → 건설 목록 + 영웅 선택 |
| 잠긴 슬롯 | **중앙 팝업** → 개척 확인 + 영웅 선택 (인접 해금 필요) |
| 지어진 건물 | **중앙 팝업** → 시설 행동 (고용/연회/안정화) |
| 광장 (바알) | 하단 패널 → 정책 탭 전환 |
| 원정 카드 | 하단 패널 → 원정 탭 → 스테이지 목록 → 병사 배정 |
| 사냥 카드 | 하단 패널 → 원정 탭 → 사냥 영웅 선택 → **MapHuntPopup** (중앙 팝업 전투) |
| 채집 카드 | 하단 패널 → 원정 탭 → 채집 영웅 선택 → **MapActionPopup** (중앙 팝업 결과) |
| 벌목 카드 | 하단 패널 → 원정 탭 → 벌목 영웅 선택 → **MapActionPopup** (중앙 팝업 결과) |

### 4-4. 사냥 (MapHuntPopup — MapScene 위 중앙 팝업)

사냥 영웅 선택 후, MapScene 위에 1:1 전투 팝업이 표시된다 (별도 씬 전환 없음).

```
┌─── 사냥 — 늑대 ──────────────────────────────┐
│                                               │
│    ★영웅      ← VS →      ○적               │
│  HP ████████  300/300   HP ████████  180/180  │
│                                               │
│ ──────────── 전투 로그 ────────────────────── │
│ R1 영웅 → 늑대  -24hp (잔여 156)              │
│ R1 늑대 → 영웅  -12hp (잔여 288)              │
│                                               │
│             [⏸정지] [x1] [스킵]              │
└───────────────────────────────────────────────┘
```
- 팝업 크기: 600x400, 화면 중앙 배치
- 반투명 배경 딤 처리
- DuelBattleScene과 동일한 1:1 전투 로직 (BattleEngine)
- 승리: 골드 보상 + 사기↑ / 패배: 부상 + 사기↓
- 전투 종료 → 팝업 닫힘 → MapScene 복귀

### 4-4b. 채집/벌목 (MapActionPopup — MapScene 위 중앙 팝업)

채집/벌목 영웅 선택 후, MapScene 위에 미니 행동 결과 팝업이 표시된다 (별도 씬 전환 없음).
사냥(MapHuntPopup)과 동일 위치/크기로 UX 일관성 확보.

```
┌─── 🌿 채집 ─────────────────────────────────────┐
│                                                   │
│              [영웅 스프라이트]                      │
│               영웅 이름                            │
│                                                   │
│           작업 중...                               │
│           ▓▓▓▓▓▓▓▓░░ 80%                         │
│                                                   │
│  ─────────── 결과 ────────────────────────────── │
│   획득: 🌾 식량 +12                               │
│   사기: +2                                        │
│   💬 "쓸만한 것들이 있군..." (죄종별 대사)          │
│                                                   │
│                  [ 확인 ]                         │
└───────────────────────────────────────────────────┘
```
- 팝업 크기: 600x400, 화면 중앙 배치 (MapHuntPopup과 동일)
- 반투명 배경 딤 처리
- 프로그레스 바: 1.5초 애니메이션 → 완료 시 결과 표시
- 영웅 스프라이트: idle 애니메이션
- 죄종별 대사: 7죄종 × 2~3개 랜덤 선택
- [스킵] 버튼: 프로그레스 건너뛰고 즉시 결과
- [확인] 버튼: 팝업 닫힘 → MapScene 복귀
- 채집: 초록 테마 (🌿, #30b050), 벌목: 갈색 테마 (🪓, #8a6a3a)

### 4-5. 이벤트 (EventScene, MapScene 위 오버레이)

- MapScene 위에 `scene.launch`로 병렬 실행
- 반투명 배경 + 이벤트 카드 (기존과 동일)
- 선택 후 닫히면 MapScene으로 복귀

### 4-5. 원정 결과 (ResultScene, MapScene 위 오버레이)

- 승패 표시, 파티 HP 상태, 전리품
- 전투 리플레이 버튼
- 감시탑 정보 (레벨별: 유무 → 규모 → 적 상세)

### 4-6. 방어전 (MapDefenseMode — MapScene 위 오버레이)

습격 발생 시 MapScene 위에 전투 오버레이 표시 (별도 씬 전환 없음).
습격 빈도: 3~5일 간격 (매일 밤 아님). BattleEngine MELEE 모드 사용.
**전투는 영외(우측 680~1280px) 영역에서만 진행** — 영내 건물과 분리.

```
┌─── 네비게이션 바 (32px) ─────────────────────────────────────────┐
│ THE SEVEN │ Day 12 │ [밤] │ 450G │ 방어:4명 │                     │
├────────────────────────────┬────────────────────────────────────┤
│                            │ [밤 습격 — 12일차]     민병: 8/10  │
│   영내 (어둡게 딤 처리)      │  SP ████░░ 42/100                 │
│                            │                                    │
│  ┌────┐┌────┐┌────┐       │  ★A→  ★B→  ★C→ ←→ ←○적1 ←○적2   │
│  │주점││대장││감시│       │     (X축 오토배틀, 영외에서 전투)    │
│  ├────┤├────┤├────┤       │                    ←○적3           │
│  │    ││바알││    │       │                                    │
│  └────┘└────┘└────┘       │                                    │
│   (건물 보이지만 조작 불가)  │         전투 필드 (600px)          │
├────────────────────────────┼────────────────────────────────────┤
│                            │ [🔥분노폭발][🛡보호막][💀저주][❤회복]│
│                            │  SP:30      SP:20    SP:40  SP:25 │
│                            │─────────────────────────────────── │
│                            │ 전투 로그 (최근 5줄)                │
│                            │ [증원 N명]      [⏸정지][x1][스킵]  │
└────────────────────────────┴────────────────────────────────────┘
```
- **영외에서 전투** — 영내(좌측)는 어둡게 딤 처리, 영외(우측)가 전투 필드
- 영웅(좌, 성벽 바깥) vs 적(우, 화면 끝에서 등장), X축 이동
- **SP 게이지 + 카드 핸드**: BattleSceneA와 동일 시스템
- **일기토**: 죄종 기반 자동 발동 (영외 중앙에서 연출)
- 하단 패널이 카드+로그로 전환, 전투 종료 시 원래 탭 복귀
- 병사 카운터 표시, 증원 버튼
- 습격 없는 밤: 평화롭게 결산으로 넘어감

**UI 요소**:
- **배경**: 그래디언트 밤하늘 + 풀/먼지 지면
- **HP바**: 비율별 컬러 전환 (초록→노랑→빨강), 테두리
- **데미지 텍스트**: 피해 비율 따라 폰트 크기 차등 (11~18px)
- **카드 핸드**: 호버 효과, 죄종 상단 악센트 라인, 시너지 표시
- **일기토 연출**: 오버레이 어둡게, VS 텍스트 스케일 애니메이션, 양측 이름 표시
- **결과 화면**: 풀스크린 딤 + 패널 (승패 스케일 애니메이션 + 전과 요약)
- **로그**: 이벤트별 컬러링, "전투 로그" 라벨
- **유닛 이름**: 영웅은 죄종 컬러, 적은 붉은 톤
- **라운드 카운터**: 우측 상단 `R#`

- BattleSceneB: 태그매치 프로토타입 (보관)
