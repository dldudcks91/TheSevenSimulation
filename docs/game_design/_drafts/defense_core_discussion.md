## 전제 (이미 대화/문서에 반영된 방향성)

- `GAME_DESIGN.md §1~§2`는 **"거점 디펜스 + 영웅 서사"를 코어**로, **원정은 서브 루프**로 선언한 상태 (A 단계 반영 완료)
- 그러나 `battle_design.md`, `base_design.md`, `hero_design.md` 등 하위 문서는 아직 옛 구조(3~5일 간격 습격, 단일 X축 방어전) 유지 중 → 정합성 어긋남
- 아래 축들은 **전부 미확정**. 하나씩 순차 논의 필요

## 논의 축 목록 (순서 제안)

1. **디펜스 리듬** — 습격이 얼마나 자주·어떤 강도로 오는가 ← 현재 여기
2. 전투/비전투 밤 비율 + 규모 곡선
3. 방어선 구조 (3열 그리드 / 2열 / X축 유지)
4. 폭주의 전투 영향 수위
5. 감정·관계의 전투 변수 적용 규칙
6. 원정 서브 루프의 보상/페널티 재설계
7. 관전 연출 (속도·스킵·드라마 포인트)

## 축 1. 디펜스 리듬 — 현재 논의 중 (미확정)

### 현재 문서 상태
- `battle_design.md §10`: 습격 3~5일 간격 (`raid_interval_min/max`)
- `GAME_DESIGN.md §2`: "매일 밤 디펜스" 선언 — battle_design과 충돌

### 제시된 선택지

| 안 | 요지 | 트레이드오프 |
|---|---|---|
| **A. 매일 밤 + 4단** | 매일 전투, 정찰/보통/잔혹/공성 4단 | 디펜스 코어성↑ / **관전 피로 큼, 자동처리 열면 코어 선언 무의미화** |
| **B. 매일 밤 + 2단** | 매일 전투, 소/대 2단 | 단순함 / 리듬 기복 약함, 반복 체감 |
| **C. 기존 3~5일 유지** | 현 battle_design.md 구조 | GAME_DESIGN.md "디펜스 코어" 선언과 모순 |
| **D. 주기형 + 텐션 게이지** | 2~3일 간격 + "적 진격도" 게이지 누적 → 임계 시 습격, 막지 못하면 이월 가중 | 예측/대응이 플레이, 관전 피로 낮음 / 매일 체감 약함, 시스템 복잡도↑ |
| **E. 고정 격일/3일** | 2~3일마다 전투, 규모만 4단 차등 | 준비-전투-수습 루프 깨끗 / "밤의 공포" 약함 |
| **F. 매일 밤 위협 + 주기형 교전** | 매일 밤 "뭔가" 오지만 실제 교전은 2~3일마다. 나머지 밤은 정찰·도발·스트레스 **텍스트 이벤트**(관전 없음). 공성은 챕터 말 1회 | 매일 "온다" 압박감 유지 + 관전 피로 낮음 + 비전투 밤을 영웅 서사 무대로 활용 / 전투/비전투 분기 설계 부담 |

### 사용자 피드백 이력
- A에 대해 **"매일 관전 피로 없을까?"** 문제 제기 (2026-04-24) → A 재추천에서 제외됨
- 현재 재추천: **F 우선 / D 차선**
  - F 근거: "디펜스 + 영웅 서사 코어"에 가장 부합. 매일 밤 압박감 유지 + 관전 피로↓. 비전투 밤을 감정·관계 드라마 무대로 활용
  - D 근거: 자유도 높음. 단 게이지 시스템이 또 하나의 관리 축이라 복잡도↑

### 확정 대기 중 질문
- A' / D / E / F / 기타 중 무엇?
- F로 갈 경우 "비전투 밤"에서 일어나는 텍스트 이벤트의 범위·농도는?

## 축 2~7 — 아직 논의 전 (옵션은 이전 대화에 있었으나 전부 미확정)

이전 `design-battle` 스킬 호출에서 제시된 6가지 설계 포인트 중 축 1 외 내용:

- **방어선/배치**: 3열 그리드(전열/후열/예비) 안 제시됨 — 미확정
- **폭주의 전투 영향**: 분노 폭주 = 배치 이탈 → 방어선 구멍 등 7죄종별 붕괴 메커니즘 안 제시됨 — 미확정
- **감정·관계 배치 규칙**: 같은 열 동류/강화/대립 배율 안 제시됨 — 미확정
- **원정 트레이드오프**: 파견 인원별 부재 기간·페널티 수치 안 제시됨 — 미확정
- **관전 연출**: x2 기본·자동 감속·스킵 가드 안 제시됨 — 미확정

> 위 모든 수치는 **브레인스토밍 단계**. 축 1 확정 후 하나씩 재논의.
