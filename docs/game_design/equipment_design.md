# 아이템 시스템 설계

> 상태: **초안** — 구조 확정, 수치 미확정
> 작성일: 2026-03-24

---

## 1. 개요

영웅에게 아이템을 장착하여 행동 효율, 전투력, 특수효과를 부여한다.
아이템은 전투 전용이 아니라 **모든 행동에 영향**을 줄 수 있다.

### 핵심 원칙

| # | 원칙 | 출처 |
|---|------|------|
| 1 | 장비는 비전투에서도 의미 — 행동 보정으로 모든 활동에 영향 | RimWorld, DF, CK3 |
| 2 | 트레이드오프 필수 — 죄종 연동 장단점 | DD, Battle Brothers, Kenshi |
| 3 | 장비가 드라마 생성 — 도난/파괴가 연쇄 반응의 씨앗 | DF, CK3, RimWorld |
| 4 | "누구에게 줄까" 딜레마 — 희소 아이템 배분 = 경영 판단 | XCOM 2, DD |
| 5 | 장비가 역할 강화 — 보정으로 전투원/연구원/건설자 특화 | Battle Brothers, Kenshi |
| 6 | 품질은 단순하게 — Phase 1은 등급 + 기본 보정 | DF, RimWorld |
| 7 | 제작 경제 밸런스 — 대장간이 다른 콘텐츠를 죽이지 않도록 | Bannerlord 반면교사 |

## 2. 슬롯 구조

**3슬롯, 용도 무관** — 어떤 종류의 아이템이든 자유롭게 장착.

```
[영웅]
├── 슬롯 1: (자유)
├── 슬롯 2: (자유)
└── 슬롯 3: (자유)
```

- 같은 아이템 중복 장착 불가 (같은 이름의 아이템 2개 X)
- 소모품은 슬롯을 사용하지 않음 (별도 사용)

**예시 조합**:
- 전투 특화: 검 + 중갑 + 분노의 부적
- 건설 특화: 건설 망치 + 작업 장갑 + 건설 부적
- 혼합: 검 + 학자의 안경 + 약초 주머니

## 3. 아이템 종류

### 3-1. 도구류

행동 보정 아이템. 특정 행동의 효율을 직접 올려준다.

| 아이템 | 보정 | 비고 |
|--------|------|------|
| 건설 망치 | 건설+N | |
| 작업 장갑 | 건설+N, 개척+N | |
| 벌목 도끼 | 벌목+N | |
| 약초 주머니 | 채집+N | |
| 학자의 안경 | 연구+N | |
| 장인의 집게 | 대장간+N | |
| 증류기 | 연금술+N | |
| 사냥꾼의 덫 | 사냥+N | |
| 인장 반지 | 외교+N | |
| 상인의 저울 | 교역+N | |
| 추천서 | 고용+N | |
| 개척 삽 | 개척+N | |

### 3-2. 무기류

전투 보정 아이템. ATK, 공격속도 등 전투 스탯에 영향.

| 아이템 | 보정 | 비고 |
|--------|------|------|
| 검 | ATK+N | 범용 |
| 도끼 | ATK+N (높음), 공격속도-% | 화력↑ 속도↓ |
| 단검 | ATK+N (낮음), 크리+% | 민첩형 |
| 활 | ATK+N, 사거리+% | 원거리 선공 |
| 지팡이 | ATK+N (낮음) | 마법형 |

### 3-3. 방어구류

전투 생존 아이템. HP, 피해경감 등.

| 아이템 | 보정 | 비고 |
|--------|------|------|
| 중갑 | 피해경감+% (높음) | |
| 경갑 | 피해경감+% (중간) | |
| 로브 | 피해경감+% (낮음) | |
| 방패 | 피해경감+% | |

### 3-4. 장신구류

특수효과, 죄종 연동, 사기 영향 등.

| 아이템 | 효과 | 비고 |
|--------|------|------|
| 분노의 부적 | [분노] 사기+N/턴 | 죄종 연동 |
| 탐욕의 반지 | [탐욕] 교역+N | 죄종 연동 |
| 전우의 팔찌 | 방어전 피해경감+% | |
| 미식가의 향낭 | [폭식] 사기+N/턴 | 죄종 연동 |
| ... | | 죄종별 장신구 확장 |

### 3-5. 소모품 (슬롯 미사용)

사용 시 즉시 효과 후 소멸. 연금술소에서 제작.

| 아이템 | 효과 | 비고 |
|--------|------|------|
| 치료약 | 부상 즉시 회복 | |
| 사기 물약 | 사기+N (즉시) | |
| 해독제 | 상태이상 해제 | |

## 4. 행동 보정 태그 목록

아이템이 보정할 수 있는 행동 11종:

| # | 태그 | 대응 행동 | 기존 스탯 공식 |
|---|------|----------|--------------|
| 1 | 건설 | 건설 | (힘+민첩)/2 |
| 2 | 개척 | 개척 | (힘+건강)/2 |
| 3 | 채집 | 채집 | (민첩+감각)/2 |
| 4 | 벌목 | 벌목 | (힘+민첩)/2 |
| 5 | 연구 | 연구 | 지능 |
| 6 | 대장간 | 대장간 제작 | (힘+감각)/2 |
| 7 | 연금술 | 연금술 제작 | (감각+지능)/2 |
| 8 | 외교 | 외교 | (지능+매력)/2 |
| 9 | 교역 | 교역 | (민첩+매력)/2 |
| 10 | 고용 | 고용 | (통솔+매력)/2 |
| 11 | 사냥 | 사냥 | (힘+민첩)/2 |

### 보정 적용 방식

```
최종 행동력 = 기존 스탯 공식 결과 + 장착 아이템 행동 보정 합산

예시: 힘15 민첩13 영웅 + 건설 망치(건설+3) + 작업 장갑(건설+2)
  → 건설력 = (15+13)/2 + 3 + 2 = 19
```

## 5. 품질 체계 — 3등급

| 등급 | 표시 | 보정 배율 | 획득 |
|------|------|----------|------|
| **일반** | 회색 | ×1.0 | 상점, 초기 지급, 제작(낮은 스탯) |
| **정예** | 파랑 | ×1.5 | 원정 보상, 제작(높은 스탯), 상인 |
| **유니크** | 금색 | ×2.0 + 고유효과 | 보스 트로피, 사냥 희귀몹, 특수 이벤트 |

---

## 6. 획득 방식

| 경로 | 내용 |
|------|------|
| **초기 지급** | 새 게임 시작 시 일반 등급(소모품 제외) 1개 랜덤 지급 |
| **원정 전리품** | 스테이지 클리어 보상 |
| **대장간 제작** | 골드 + 영웅 1명 투입 |
| **연금술소 제작** | 소모품 제작 (골드 + 영웅 1명 투입) |
| **교역 (상인)** | 역마차 전제, 상인별 취급 다름 |
| **사냥 드롭** | 확률적 아이템 획득 |
| **보스 트로피** | 챕터 보스 처치 시 유니크 확정 |
