# 거점(림보) 시설 설계

> 상태: 구조 확정, 수치 미확정
> 작성일: 2026-03-24
> 참고: HOMM3(건설 트리), DD(시설 운영), Frostpunk(정책), CK3(결단)

---

## 1. 개요

바알이 운영하는 거점 "림보". 매 턴 건설/연구/생산을 지시하고 영웅을 투입하여 거점을 성장시킨다.

### 자원: 골드 (단일 화폐)

| 수입원 | 설명 |
|--------|------|
| 원정 전리품 | 주 수입원. 챕터 진행 시 획득 |
| 방어전 승리 보상 | 매일 밤 습격 방어 시 소량 |
| 시장 (시설) | 건설 후 턴당 고정 수입 |
| 교역 | 상인 방문 시 장비/소모품 판매 |

| 지출처 | 설명 |
|--------|------|
| 건설 | 시설 신축/업그레이드 |
| 연구 | 기술 연구 비용 |
| 영웅 영입 | 역마차 영입 비용 |
| 장비 구매 | 상인에게서 구매 |
| 소모품 제작 | 연금술소에서 제작 |
| 포고령 유지비 | 정책에 따른 추가 비용 |

---

## 2. 건설 (HOMM3 방식)

**규칙**: 턴당 건설 1개 + 영웅 1명 투입. 죄종에 따라 건설 속도 차이.

### 건설 트리

```
[Tier 1] 전제조건 없음
  ├── 주점 ────────────────┐ (영웅 고용 + 사기)
  ├── 훈련소 ──────┐       │
  ├── 대장간 ──┐   │       │
  ├── 성벽     │   │       │
  └── 감시탑   │   │       │
               │   │       │
[Tier 2]       │   │       │
  ├── 무기고 ──┘   │       │
  ├── 성당 ────────┘       │
  ├── 역마차 ──────────────┘ (교역 상인)
  ├── 성벽 Lv.2 (←성벽)
  └── 감시탑 Lv.2 (←감시탑)
               │
[Tier 3]       │
  ├── 함정 공방 (←대장간+성벽)
  ├── 연금술소 (←대장간+성당)
  ├── 시장 (←역마차) (턴당 수입)
  ├── 성벽 Lv.3 (←성벽 Lv.2)
  └── 감시탑 Lv.3 (←감시탑 Lv.2)
```

### 시설 목록

| 시설 | Tier | 전제 | 비용 | 기능 |
|------|------|------|------|------|
| **주점** | 1 | - | 100G | 영웅 고용 + 사기 상승 + 영웅 교류 (폭주 위험) |
| **훈련소** | 1 | - | 100G | 영웅 투입 → 경험치 획득 |
| **대장간** | 1 | - | 150G | 장비 제작/수리 |
| **성벽** | 1 | - | 200G | 방어전 피해 경감 |
| **감시탑 Lv.1** | 1 | - | 100G | 습격 유무 알림 |
| **무기고** | 2 | 대장간 | 200G | 장비 보관 + 전투력 보너스 |
| **성당** | 2 | 훈련소 | 200G | 사기 안정화 (극단 → 중간) |
| **역마차** | 2 | 주점 | 150G | 교역 상인 도착, 외부 정보 |
| **성벽 Lv.2** | 2 | 성벽 | 300G | 방어 강화 |
| **감시탑 Lv.2** | 2 | 감시탑 | 200G | 습격 규모 알림 |
| **함정 공방** | 3 | 대장간+성벽 | 250G | 방어전 함정 제작/배치 |
| **연금술소** | 3 | 대장간+성당 | 300G | 소모품 제작 (치료약, 사기 물약 등) |
| **시장** | 3 | 역마차 | 200G | 턴당 골드 수입 |
| **성벽 Lv.3** | 3 | 성벽 Lv.2 | 400G | 방어 대폭 강화 |
| **감시탑 Lv.3** | 3 | 감시탑 Lv.2 | 350G | 적 구성 상세 공개 |

### 건설 시 영웅 투입 효율

핵심 스탯: **힘 + 체력**. 죄종과 무관, 스탯으로만 결정.

## 3. 연구

**규칙**: 턴당 연구 1개 진행 가능 + 영웅 1명 투입. 다턴 소요. 건설과 별도로 진행.

### 연구 목록

#### 군사 연구 (전제: 훈련소)

| 연구 | 비용 | 소요 | 효과 |
|------|------|------|------|
| 전투 훈련 I | 150G | 2턴 | 전투 시 힘 보정 +10% |
| 전투 훈련 II | 300G | 3턴 | 전투 시 힘 보정 +20% |
| 방어 훈련 I | 150G | 2턴 | 전투 시 체력 보정 +10% |
| 방어 훈련 II | 300G | 3턴 | 전투 시 체력 보정 +20% |
| 응급 처치 | 200G | 2턴 | 부상 회복 속도 2배 |

#### 경제 연구 (전제: 시장)

| 연구 | 비용 | 소요 | 효과 |
|------|------|------|------|
| 교역로 개척 | 200G | 2턴 | 시장 수입 +50% |
| 전리품 감정 | 250G | 3턴 | 원정 골드 보상 +30% |
| 효율적 건설 | 300G | 3턴 | 건설 비용 -20% |

#### 방어 연구 (전제: 성벽)

| 연구 | 비용 | 소요 | 효과 |
|------|------|------|------|
| 성벽 강화 | 200G | 2턴 | 방어전 피해 경감 +20% |
| 함정 기술 I | 250G | 2턴 | 기본 함정 해금 (말뚝, 구덩이) |
| 함정 기술 II | 400G | 3턴 | 고급 함정 해금 (화염, 독) |
| 야간 경계 | 200G | 2턴 | 방어전 선제 공격 1회 |

#### 영웅 연구 (전제: 성당)

| 연구 | 비용 | 소요 | 효과 |
|------|------|------|------|
| 사기 관리 | 250G | 3턴 | 사기 변동폭 -30% |
| 죄종 이해 | 400G | 4턴 | 폭주 임계 105로 상향 |
| 결속 의식 | 300G | 3턴 | 죄종 만족 시 사기 상승량 +50% |

### 연구 시 영웅 투입 효율

핵심 스탯: **지능 + 감각**. 죄종과 무관, 스탯으로만 결정.

## 4. 생산

건설된 시설에서 영웅을 투입하여 아이템을 만든다.

| 시설 | 생산물 | 영웅 투입 | 비용 |
|------|--------|----------|------|
| **대장간** | 무기/방어구 제작 | 1명 (민첩+감각) | 재료비 |
| **연금술소** | 치료약, 사기 물약, 해독제 | 1명 (감각+지능) | 재료비 |
| **함정 공방** | 말뚝/구덩이/화염/독 함정 | 1명 (민첩+감각) | 재료비 |

## 5. 플레이어 행동 목록 (건설/연구 외)

### 원정 파견

| 항목 | 내용 |
|------|------|
| 조건 | 영웅 1~3명 선택 |
| 목표 | 챕터/스테이지 선택 |
| 비용 | 보급품 (골드로 구매) |
| 결과 | 저녁에 결과 보고 → 계속/철수 판단 |
| 제약 | 원정 중 영웅 = 밤 방어 불가 |

### 교역 (전제: 시장)

| 항목 | 내용 |
|------|------|
| 시기 | 상인 방문 시 (3~5턴 간격) |
| 가능 | 장비/소모품 구매, 불필요 장비 판매 |
| 특수 | 상인마다 취급 물품 다름 (무기상, 약재상, 골동품상) |

### 영웅 고용 (전제: 주점)

| 항목 | 내용 |
|------|------|
| 후보 | 매 턴 2~3명 갱신 |
| 비용 | 영웅 등급에 따라 다름 |
| 정보 | 죄종, 직업 공개 / 스탯은 고용 후 확인 |
| 최대 | 7명 |

### 영웅 해고

| 항목 | 내용 |
|------|------|
| 비용 | 없음 |
| 효과 | 영구 제거. 장비 회수 가능 |
| 위험 | 해고된 영웅의 죄종에 따라 떠나며 사고 가능 |

### 장비 관리

| 항목 | 내용 |
|------|------|
| 장착/해제 | 언제든 가능 (전투 중 제외) |
| 수리 | 대장간 필요, 영웅 투입 + 골드 |
| 강화 | 대장간 Lv.2 이후 가능 (Phase 2) |

### 포고령 — 거점 정책 (Frostpunk 참고)

바알이 거점 전체에 내리는 명령. 영웅 사기에 영향.

| 정책 | 선택지 | 효과 |
|------|--------|------|
| **배급** | 풍족 | 전체 사기 +5/턴, 유지비 2배 |
| | 보통 | 변동 없음 |
| | 긴축 | 전체 사기 -3/턴, 유지비 절반 |
| **훈련 강도** | 강화 | 경험치 +50%, 사기 -3/턴 |
| | 보통 | 변동 없음 |
| | 완화 | 경험치 -50%, 사기 +3/턴 |
| **경계 수준** | 최대 | 방어력 +30%, 사기 -2/턴 |
| | 보통 | 변동 없음 |
| | 최소 | 방어력 -30%, 사기 +2/턴 |

### 방어 준비 (전제: 함정 공방)

밤 습격에 대한 **간접 개입 수단**:

| 행동 | 비용 | 효과 |
|------|------|------|
| 함정 배치 | 소모품 1개 | 방어전 시작 시 적에게 선제 피해 |
| 성벽 수리 | 골드 | 손상된 성벽 복구 |
| 바리케이드 | 골드 | 1회용 추가 방어벽 |

### 연회 개최 (전제: 주점)

| 항목 | 내용 |
|------|------|
| 비용 | 골드 (높음) |
| 효과 | 전체 사기 +15 |
| 위험 | 폭식/색욕 영웅 사기 +25 (폭주 위험) |
| 죄종 반응 | [폭식]/[색욕] 사기 +25 (폭주 위험) |
| 제한 | 5턴에 1회 |

### 치료 지시 (전제: 성당 or 연금술소)

| 방법 | 효과 | 비용 |
|------|------|------|
| 성당 치료 | 사기 안정화 (극단 → 50 방향) | 영웅 1턴 소모 |
| 치료약 사용 | 부상 즉시 회복 | 소모품 1개 |
| 사기 물약 | 사기 +20 (즉시) | 소모품 1개 |

## 6. 턴당 행동 제한

| 행동 | 제한 |
|------|------|
| 건설 | 턴당 1개 (영웅 1명 투입) |
| 연구 | 턴당 1개 진행 중 가능 (영웅 1명 투입) |
| 생산 | 시설당 1개 (영웅 1명 투입) |
| 원정 | 파티 1개 (1~3명) |
| 포고령 | 언제든 변경 가능 (즉시 적용) |
| 교역 | 상인 방문 시만 |
| 영웅 영입/해고 | 제한 없음 |
| 장비 관리 | 제한 없음 |
| 방어 준비 | 함정 1개/턴 배치 |
| 연회 | 5턴에 1회 |

### 영웅 투입 정리

매 턴 영웅 7명의 배분:

```
영웅 7명
  ├── 건설 투입 (0~1명)
  ├── 연구 투입 (0~1명)
  ├── 생산 투입 (시설당 0~1명)
  ├── 원정 파견 (0~3명) → 마을 밖, 밤 방어 X
  ├── 성당 치료 (0~1명)
  └── 나머지 → 마을 안, 밤 방어 O
```
