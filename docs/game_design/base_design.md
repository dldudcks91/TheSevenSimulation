# 거점(림보) 시설 설계

> 상태: 구조 확정, 수치 미확정
> 작성일: 2026-03-24
> 참고: HOMM3(건설 트리), DD(시설 운영), Frostpunk(정책), CK3(결단)

---

## 1. 개요

바알이 운영하는 거점 "림보". 매 턴 건설/연구/생산을 지시하고 영웅을 투입하여 거점을 성장시킨다.

### 자원: 3종 (식량 / 나무 / 골드)

| 자원 | 획득 방법 | 주요 사용처 |
|------|----------|------------|
| **식량** | 채집 (거점 주변, 밤 방어 O) | 영웅 유지비 (턴당 소비), 배급 포고령, 연회 |
| **나무** | 벌목 (외곽, 밤 방어 X) | 건설 (전 티어) |
| **골드** | 원정 전리품, 방어전 보상, 시장, 교역, 사냥 | 연구, 장비/소모품 구매, 영웅 고용 |

### 영웅 식량 비용

스탯 총합(60~80)에 비례하여 턴당 식량 소비량 결정. 강한 영웅일수록 유지비가 높다.

| 스탯 총합 | 식량 비용/턴 |
|----------|-------------|
| 60 | 5 |
| 65 | 6 |
| 70 | 8 |
| 75 | 9 |
| 80 | 10 |

공식: `비용 = 5 + round((스탯총합 - 60) / 4)`

## 2. 건설 (HOMM3 방식)

**규칙**: 턴당 건설 1개 + 영웅 1명 투입. 스탯(힘+민첩)에 따라 건설 속도 차이.

### 건설 트리

```
[Tier 1] 전제조건 없음
  ├── 주점 ────────────────┐ (영웅 고용 + 사기)
  ├── 대장간 ──┐           │
  └── 감시탑   │           │
               │           │
[Tier 2]       │           │
  ├── 병원 ────┤───────────┘ (←주점, 치료 + 사기 안정)
  ├── 역마차 ──┤───────────┘ (←주점, 교역 상인)
  └── 감시탑 Lv.2 (←감시탑)
               │
[Tier 3]       │
  ├── 연금술소 (←대장간+병원)
  ├── 시장 (←역마차) (턴당 수입)
  └── 감시탑 Lv.3 (←감시탑 Lv.2)
```

### 시설 목록

| 시설 | Tier | 전제 | 비용 (나무) | 건설 턴 | 기능 |
|------|------|------|-----------|---------|------|
| **주점** | 1 | - | 100 | 1 | 영웅 고용 + 사기 상승 + 연회 개최 |
| **대장간** | 1 | - | 150 | 1 | 장비 제작/수리 |
| **감시탑 Lv.1** | 1 | - | 100 | 1 | 습격 유무 알림 |
| **병원** | 2 | 주점 | 200 | 2 | 부상 영웅 치료 (회복 턴 절반) + 사기 안정화 |
| **역마차** | 2 | 주점 | 150 | 1 | 교역 상인 도착, 외부 정보 |
| **감시탑 Lv.2** | 2 | 감시탑 | 200 | 1 | 습격 규모 알림 |
| **연금술소** | 3 | 대장간+병원 | 300 | 2 | 소모품 제작 (치료약, 사기 물약 등) |
| **시장** | 3 | 역마차 | 200 | 2 | 턴당 골드 수입 |
| **감시탑 Lv.3** | 3 | 감시탑 Lv.2 | 350 | 2 | 적 구성 상세 공개 |

### 건설 시 영웅 투입 효율

핵심 스탯: **힘 + 민첩** (50:50). 죄종과 무관, 스탯으로만 결정.

## 3. 연구

**규칙**: 턴당 연구 1개 진행 가능 + 영웅 1명 투입. 다턴 소요. 건설과 별도로 진행.

### 연구 목록

#### 경제 연구 (전제: 시장)

| 연구 | 비용 | 소요 | 효과 |
|------|------|------|------|
| 교역로 개척 | 200G | 2턴 | 시장 수입 +50% |
| 전리품 감정 | 250G | 3턴 | 원정 골드 보상 +30% |
| 효율적 건설 | 300G | 3턴 | 건설 비용 -20% |

#### 영웅 연구 (전제: 병원)

| 연구 | 비용 | 소요 | 효과 |
|------|------|------|------|
| 사기 관리 | 250G | 3턴 | 사기 변동폭 -30% |
| 죄종 이해 | 400G | 4턴 | 폭주 임계 105로 상향 |
| 결속 의식 | 300G | 3턴 | 죄종 만족 시 사기 상승량 +50% |

### 연구 시 영웅 투입 효율

핵심 스탯: **지능** (단독). 죄종과 무관, 스탯으로만 결정.

---

## 4. 생산

건설된 시설에서 영웅을 투입하여 아이템을 만든다.

| 시설 | 생산물 | 영웅 투입 | 비용 |
|------|--------|----------|------|
| **대장간** | 무기/방어구 제작 | 1명 (힘+감각) | 재료비 |
| **연금술소** | 치료약, 사기 물약, 해독제 | 1명 (감각+지능) | 재료비 |

---

## 5. 플레이어 행동 목록 (건설/연구 외)

### 원정 파견

| 항목 | 내용 |
|------|------|
| 조건 | 영웅 1~3명 선택 |
| 목표 | 챕터/스테이지 선택 |
| 비용 | 보급품 (골드로 구매) |
| 결과 | 저녁에 결과 보고 → 계속/철수 판단 |
| 제약 | 원정 중 영웅 = 밤 방어 불가 |

### 채집 (식량 획득)

| 항목 | 내용 |
|------|------|
| 투입 | 1~2명 |
| 위치 | 거점 주변 (**밤 방어 참여 가능**) |
| 위험 | 없음 |
| 보상 | **식량** |
| 핵심 스탯 | 민첩 + 감각 (50:50) |

### 벌목 (나무 획득)

| 항목 | 내용 |
|------|------|
| 투입 | 1~2명 |
| 위치 | 거점 외곽 (**밤 방어 참여 불가**) |
| 위험 | 없음 |
| 보상 | **나무** |
| 핵심 스탯 | 힘 + 민첩 (50:50) |

### 사냥 (골드 + 아이템)

| 항목 | 내용 |
|------|------|
| 투입 | 1~2명 |
| 위치 | 거점 외곽 (**밤 방어 참여 불가**) |
| 위험 | 부상 가능, HP 소모 큼 |
| 보상 | **골드** + 확률적 아이템 획득 |
| 핵심 스탯 | 힘 + 민첩 (50:50) |

### 외교

| 항목 | 내용 |
|------|------|
| 투입 | 1명 |
| 대상 | 주변 마을 (상세 기획 예정) |
| 핵심 스탯 | 지능 + 매력 (50:50) |
| 효과 | 미정 |

### 교역 (전제: 역마차)

| 항목 | 내용 |
|------|------|
| 시기 | 상인 방문 시 (3~5턴 간격) |
| 가능 | 장비/소모품 구매, 불필요 장비 판매 |
| 특수 | 상인마다 취급 물품 다름 (무기상, 약재상, 골동품상) |
| 핵심 스탯 | 민첩 + 매력 (50:50, 가격 보정) |

### 영웅 고용 (전제: 주점)

| 항목 | 내용 |
|------|------|
| 후보 | 매 턴 2~3명 갱신 |
| 비용 | 영웅 등급에 따라 다름 |
| 정보 | 죄종 공개 / 스탯은 고용 후 확인 |
| 최대 | 7명 |
| 핵심 스탯 | 통솔 + 매력 (50:50) |

### 영웅 해고

| 항목 | 내용 |
|------|------|
| 비용 | 없음 |
| 효과 | 영구 제거. 장비 회수 가능 |
| 위험 | 해고된 영웅의 죄종에 따라 떠나며 사고 가능 |

### 장비 관리

| 항목 | 내용 |
|------|------|
| 장착/해제 | 언제든 가능 (전투 중 제외) |
| 수리 | 대장간 필요, 영웅 투입 + 골드 |
| 강화 | 대장간 Lv.2 이후 가능 (Phase 2) |

### 포고령 — 거점 정책 (Frostpunk 참고)

바알이 거점 전체에 내리는 명령. 영웅 사기에 영향.

| 정책 | 선택지 | 효과 |
|------|--------|------|
| **배급** | 풍족 | 전체 사기 +5/턴, 유지비 2배 |
| | 보통 | 변동 없음 |
| | 긴축 | 전체 사기 -3/턴, 유지비 절반 |
| **훈련 강도** | 강화 | 경험치 +50%, 사기 -3/턴 |
| | 보통 | 변동 없음 |
| | 완화 | 경험치 -50%, 사기 +3/턴 |
| **경계 수준** | 최대 | 방어력 +30%, 사기 -2/턴 |
| | 보통 | 변동 없음 |
| | 최소 | 방어력 -30%, 사기 +2/턴 |

### 연회 개최 (전제: 주점)

| 항목 | 내용 |
|------|------|
| 비용 | 골드 (높음) |
| 효과 | 전체 사기 +15 |
| 위험 | 폭식/색욕 영웅 사기 +25 (폭주 위험) |
| 죄종 반응 | [폭식]/[색욕] 사기 +25 (폭주 위험) |
| 제한 | 5턴에 1회 |

### 치료 지시 (전제: 병원 or 연금술소)

| 방법 | 효과 | 비용 |
|------|------|------|
| 병원 치료 | 부상 회복 속도 2배 + 사기 안정화 | 영웅 1턴 소모 |
| 치료약 사용 | 부상 즉시 회복 | 소모품 1개 |
| 사기 물약 | 사기 +20 (즉시) | 소모품 1개 |

## 6. 턴당 행동 제한

| 행동 | 제한 |
|------|------|
| 건설 | 턴당 1개 (영웅 1명 투입) |
| 연구 | 턴당 1개 진행 중 가능 (영웅 1명 투입) |
| 생산 | 시설당 1개 (영웅 1명 투입) |
| 원정 | 파티 1개 (1~3명) |
| 채집 | 1~2명 |
| 사냥 | 1~2명 |
| 외교 | 1명 |
| 포고령 | 언제든 변경 가능 (즉시 적용) |
| 교역 | 상인 방문 시만 |
| 영웅 영입/해고 | 제한 없음 |
| 장비 관리 | 제한 없음 |
| 연회 | 5턴에 1회 |

### UI 조작 방식

- **기본 플로우**: 거점 화면에서 시설/행동 지점 클릭 → 투입할 영웅 선택 (적합도 표시)
- **단축 플로우**: 영웅 패널에서 영웅 클릭 → 드래그로 시설/행동에 배치

### 영웅 투입 정리

매 턴 영웅 7명의 배분:

```
영웅 7명
  ├── 채집 (0~2명)         → 식량 획득, 거점 주변, 밤 방어 O
  ├── 벌목 (0~2명)         → 나무 획득, 거점 외곽, 밤 방어 X
  ├── 사냥 (0~2명)         → 골드+아이템, 거점 외곽, 밤 방어 X
  ├── 원정 파견 (0~3명)    → 골드+진행, 거점 밖, 밤 방어 X
  ├── 건설 투입 (0~1명)
  ├── 연구 투입 (0~1명)
  ├── 생산 투입 (시설당 0~1명)
  ├── 외교 (0~1명)
  ├── 병원 치료 (0~1명)
  └── 잔류 → 밤 방어 O
```
