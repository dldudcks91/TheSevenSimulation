# 챕터 시나리오 시스템 설계

> 상태: 초안
> 작성일: 2026-03-24

---

## 1. 개요

### 스토리를 시뮬레이션에 녹이는 방식

이 게임은 컷신이나 긴 텍스트로 스토리를 전달하지 않는다.
**플레이어가 챕터의 감정을 시뮬레이션으로 체감**하게 만든다.

| 전통 RPG | 이 게임 |
|---------|--------|
| 컷신으로 서사 전달 | **환경 변조**로 감정 체감 |
| 대화로 캐릭터 설명 | **이벤트 선택지**로 관계 표현 |
| 보스전에서 클라이맥스 | **보스 서사**(브리핑+유언)로 압축 전달 |

### 핵심 원칙

- **스토리는 액자, 드라마는 시스템이 만든다** — 작가의 서사는 프레임만 제공
- **플레이 자체가 서사** — 분노 챕터에서 플레이어가 실제로 분노하게 만든다
- **보스 서사는 짧고 강하게** — 대사 1~2줄로 기존 스토리의 핵심을 보존
- **공통 이벤트 + 챕터 이벤트** — 기존 30개 이벤트는 모든 챕터에서 발생, 챕터 고유 이벤트가 추가

## 2. 챕터 시나리오의 3요소

모든 챕터는 이 3가지로 구성된다:

| 요소 | 역할 | 스토리 전달 방식 |
|------|------|----------------|
| **환경 변조** | 시뮬 규칙을 바꿔 해당 죄종의 압박을 체감시킴 | 플레이 자체가 서사 |
| **챕터 이벤트** (2~3개) | 해당 죄종 테마의 고유 선택지 | 스토리 소재를 이벤트로 분해 |
| **보스 서사** (브리핑+유언) | 원정 전후 짧은 텍스트 | 기존 스토리의 핵심 대사 보존 |

### 구조 다이어그램

```
[챕터 N 시작]
  │
  ├─ 환경 변조 적용 (시뮬 규칙 변경)
  │
  ├─ 공통 이벤트 풀 (기존 30개) ─── 매 턴 발생
  │
  ├─ 챕터 이벤트 (2~3개) ────── 조건 충족 시 발생
  │
  ├─ 보스 원정 해금 조건 충족
  │     ├─ 보스 브리핑 (원정 전 텍스트)
  │     ├─ 보스전 (자동전투)
  │     └─ 보스 유언 + 기억 파편 (원정 후 텍스트)
  │
  └─ 다음 챕터로 전환 (환경 변조 해제 → 새 변조 적용)
```

## 3. 영웅의 정체성: 결함 있는 인간들

영웅은 **인간**이다. 용병, 모험가, 떠돌이 — 바알에게 고용된 자들.
이들은 각자 깊은 성격적 결함을 안고 있으며, 그 결함이 곧 **죄종 태그**다.

| 죄종 | 인간으로서의 결함 | 영웅 예시 |
|------|----------------|----------|
| 분노 | 충동적, 폭력적, 참을성 없음 | 전쟁에서 돌아온 용병, 복수에 사로잡힌 검사 |
| 시기 | 비교 강박, 열등감, 뒷공작 | 인정받지 못한 2인자, 형제의 그림자에 산 궁수 |
| 탐욕 | 독점욕, 소유욕, 나누지 못함 | 모든 것을 잃은 상인, 보물을 찾는 도적 |
| 나태 | 무기력, 회피, 전염되는 체념 | 전쟁에 지친 기사, 재능 있지만 포기한 학자 |
| 폭식 | 과잉, 불안한 비축, 끝없는 허기 | 굶주림을 겪은 유민, 채워지지 않는 전사 |
| 색욕 | 집착, 의존, 관계 중독 | 사랑에 실패한 시인, 혼자가 되는 게 두려운 치유사 |
| 교만 | 자기확신, 독선, 복종 거부 | 왕좌에서 쫓겨난 지휘관, 자기가 옳다고 믿는 성직자 |

**왜 결함 있는 자들만 오는가?**
- 멀쩡한 인간은 악마 밑에서 일하지 않는다
- 갈 곳 없는 자, 버림받은 자, 망가진 자들이 바알에게 모인다
- 이것이 DD(Darkest Dungeon)의 영웅 풀과 같은 논리

### Chapter 1 — 분노 / Wrath 「불타는 전장」

> 보스: 사탄 (Satan) — 추방된 자의 복수
> 플레이어 체감: **"안 싸우면 죽는다"**

#### 환경 변조
| 변조 | 효과 |
|------|------|
| 습격 강도 증가 | 밤 습격 규모 1.5배 (다른 챕터 대비) |
| 시설 파괴 위험 | 습격 실패 시 시설 파괴 확률 2배 |
| 공격적 보상 | 공격적/폭력적 선택지의 보상이 평소보다 강력 |

#### 챕터 이벤트

**1-1. 불타는 마을** (챕터 시작 시 고정)
장면: *"연기가 보입니다. 인근 마을이 불타고 있습니다. 사탄의 군세입니다. 살아남은 자들이 거점으로 몰려옵니다."*

| 선택지 | 결과 |
|--------|------|
| ▶ 받아들인다 | 식량 소모 증가, 노동력 획득, 전체 사기 소폭 상승 |
| ▶ 문을 닫는다 | 안전, 전체 사기 하락 — "우리도 버림받을 수 있다" |
| ▶ 마을로 역습한다 (영웅 파견) | 영웅 부상 위험, 성공 시 자원+전체 사기 대폭 상승 |

**1-2. 끊이지 않는 습격** (밤 습격 3회 연속 후)
장면: *"또입니다. 영웅들의 눈이 충혈되어 있습니다. 이틀째 잠을 못 잤습니다. '언제까지 이러는 겁니까.'"*

| 선택지 | 결과 |
|--------|------|
| ▶ "끝장내러 간다" | 보스 원정 조기 도전 가능, 준비 부족 위험 |
| ▶ "버텨라, 방법을 찾겠다" | 사기 하락, 방어 보너스 지급 |
| ▶ 교대 근무를 짠다 | 효율 하락, 부상률 감소 |

#### 보스 서사
- **브리핑**: *"불타는 전장 너머, 추방된 자가 왕을 기다린다."*
- **유언**: *사탄은 반성하지 않는다. 끝까지 분노.*
- **기억 파편**: *"낙인이 반응한다. 분노가 돌아왔다. 그리고 — 전장 너머 숲의 경계. 누군가가 보고 있다."*

### Chapter 2 — 시기 / Envy 「뒤틀린 숲」

> 보스: 레비아탄 (Leviathan) — 바알의 형제
> 플레이어 체감: **"공평이 불가능하다"**

#### 환경 변조
| 변조 | 효과 |
|------|------|
| 비교 증폭 | 원정 성공/실패 시 다른 영웅의 사기 반응 2배 |
| 격차 민감 | 레벨/장비 격차가 사기에 영향 (강한 영웅 → 약한 영웅 사기 감소) |
| 시기 이벤트 증가 | [시기] 관련 공통 이벤트 발생 확률 2배 |

#### 챕터 이벤트

**2-1. 독이 든 칭찬** (영웅 원정 2회 연속 성공 시)
장면: *"거점에 이상한 분위기가 감돕니다. [영웅X]의 성공을 축하하는 자가 없습니다. 대신 속삭입니다. '운이 좋았을 뿐이야.'"*

| 선택지 | 결과 |
|--------|------|
| ▶ 공개 시상식을 연다 | [영웅X] 사기 대폭 상승, 나머지 영웅 사기 하락 |
| ▶ 조용히 넘어간다 | 시기 분위기 지속, 전체 효율 소폭 하락 |
| ▶ 전체에게 공평한 보상 | 자원 소모 큼, 시기 분위기 해소 |

**2-2. 형제의 그림자** (보스 원정 해금 시 고정)
장면: *"숲이 속삭입니다. 바알에게만 들리는 목소리. '너는 항상 혼자서 다 가졌지.' 기억의 파편이 떠오릅니다. 형제가 있었다."*

| 선택지 | 결과 |
|--------|------|
| ▶ 기억을 붙잡는다 | 보스전 버프, 전체 사기 소폭 상승 |
| ▶ 잊으려 한다 | 스트레스 없음, 보스전 디버프 |

#### 보스 서사
- **브리핑**: *"이 숲에는 형제의 냄새가 난다."*
- **유언**: *"...나를 줄에 세울 때. 한 번이라도 미안했냐?" — (바알 — 대답하지 않는다.) — "그래. ...형제니까."*
- **기억 파편**: *"빼앗으면 채워질 줄 알았다. 안 채워졌다. — 누군가가 거래를 했다."*

### Chapter 3 — 탐욕 / Greed 「황금의 사막」

> 보스: 맘몬 (Mammon) — 바알을 팔아넘긴 자
> 플레이어 체감: **"이걸 거절할 수 있나?"**

#### 환경 변조
| 변조 | 효과 |
|------|------|
| 거래 폭증 | 상인 방문 빈도 2배, 가격은 폭리 |
| 약탈 유혹 | 약탈/착취 선택지의 보상이 매우 강력 |
| 도난 위험 | 자원 관련 사고(도난, 손실) 이벤트 빈도 증가 |

#### 챕터 이벤트

**3-1. 황금의 제안** (챕터 진입 시 고정)
장면: *"사막에서 전령이 왔습니다. 맘몬의 전령입니다. '우리 주인이 제안합니다. 영웅 하나를 넘기면 이 전쟁을 끝내주겠다고.'"*

| 선택지 | 결과 |
|--------|------|
| ▶ 거절한다 | 전체 사기 상승, 맘몬 원정 난이도 증가 |
| ▶ 가장 약한 영웅을 넘긴다 | 영웅 1명 **영구 상실**, 대량 자원+장비 획득, 전체 사기 급락 |
| ▶ 전령을 잡아 정보를 캔다 | 보스전 정보 획득, 외교 이벤트 차단 |

**3-2. 은 서른 개** (영웅 자원 불만 누적 시)
장면: *"[영웅X]가 찾아왔습니다. '다른 세력에서 제안이 왔습니다. 저를 더 대우해주겠다고. 여기 남을 이유를 주십시오.'"*

| 선택지 | 결과 |
|--------|------|
| ▶ 자원을 투자한다 | 자원 소모, [영웅X] 잔류+충성 |
| ▶ "가고 싶으면 가라" | [영웅X] 이탈 가능 |
| ▶ "배신자는 여기 있을 수 없다" | 즉시 추방, 전체 사기 분열 |

**3-3. 사막의 금고** (원정 스테이지 클리어 시)
장면: *"원정대가 금으로 가득한 방을 발견했습니다. 전부 가져갈 수 있지만, 함정일 수 있습니다."*

| 선택지 | 결과 |
|--------|------|
| ▶ 전부 가져간다 | 자원 대량 획득, 원정대 디버프 (무거움) |
| ▶ 필요한 만큼만 | 자원 소량, 안전 |
| ▶ 건드리지 않는다 | 자원 없음, 함정 회피 + 전체 사기 소폭 상승 ("욕심에 빠지지 않았다") |

#### 보스 서사
- **브리핑**: *"사막 끝에 그놈이 있다. 바알을 팔아넘기고 대가를 챙긴 놈."*
- **유언**: *(없다. 맘몬은 말 없이 죽으면서도 더 뜯어가려 한다. 끝까지 탐욕.)*
- **기억 파편**: *"기억이 돌아온다. 그 밤 — 낯선 빛. 천상의 빛. 대천사 티리엘. 배후가 있다."*

### Chapter 4 — 나태 / Sloth 「망각의 동토」

> 보스: 벨페고르 (Belphegor) — 알면서 방관한 자
> 플레이어 체감: **"왜 계속하는가?"**

#### 환경 변조
| 변조 | 효과 |
|------|------|
| 사기 자연감소 | 매 턴 전체 영웅 사기 -2 |
| 이탈 위험 증가 | 영웅 이탈 이벤트 확률 2배 |
| 죄종 만족 감소 | 죄종 만족 시 사기 상승량 절반 (의욕 상실) |
| 귀환 디버프 | 원정 귀환 시 "계속할 이유가 없다" 디버프 확률 |

#### 챕터 이벤트

**4-1. "그냥 여기서 살면 안 됩니까"** (챕터 진입 후 5턴)
장면: *"영웅 과반수가 모여 있습니다. 대표가 나섰습니다. '원정을 멈추자. 여기서 조용히 살자. 더 이상 누굴 죽일 필요가 있습니까.'"*

| 선택지 | 결과 |
|--------|------|
| ▶ "복수를 멈출 수 없다" | 반대파 사기 하락, 동조파 사기 상승 |
| ▶ 하루 생각할 시간을 준다 | 오늘 행동 제한, 내일 투표 이벤트 연결 |
| ▶ "원하는 자는 떠나라" | 영웅 1~2명 이탈 가능, 잔류자 사기 상승 |

**4-2. 얼어붙은 기억** (보스 원정 해금 시 고정)
장면: *"동토의 냉기가 거점까지 밀려옵니다. 바알이 멈춘다. 이 여정을 끝내고, 이대로 살면 안 되는가. 인간으로. 조용히."*

| 선택지 | 결과 |
|--------|------|
| ▶ "일어선다" | 전체 사기 대폭 회복, 보스전 버프 |
| ▶ "...하루만 더" | 1턴 행동 불가, 사기 변동 없음 |

#### 보스 서사
- **브리핑**: *"벨페고르. 배신의 밤이 올 것을 알고 있었다. 막지 않았다."*
- **유언**: *(없다. 벨페고르는 아무 감정도 없다. 무표정으로 쓰러진다.)*
- **기억 파편**: *"포기하고 싶다는 감각. 악마였을 때는 없던 것이다. — 맹약을 넘긴 놈이 있다."*

### Chapter 5 — 폭식 / Gluttony 「심연의 동굴」

> 보스: 바알제붑 (Beelzebub) — 한 번의 충동으로 모든 것을 무너뜨린 자
> 플레이어 체감: **"모자란다. 더 필요하다."**

#### 환경 변조
| 변조 | 효과 |
|------|------|
| 자원 소모 증가 | 식량/자원 소모 1.5배 |
| 전리품 감소 | 원정 전리품 수량 감소 |
| 폭식 이벤트 증가 | [폭식] 관련 이벤트 빈도 증가 |
| 시설 유지비 증가 | 거점 시설 유지 자원 상승 |

#### 챕터 이벤트

**5-1. 빵 한 조각** (식량 50% 이하)
장면: *"식량이 바닥나고 있습니다. 거점 밖에 인간 노인이 빵 한 조각을 들고 서 있습니다. '드시오. 배고프시죠.'"*

| 선택지 | 결과 |
|--------|------|
| ▶ 받는다 | 식량 미미하게 회복, 전체 사기 소폭 상승 |
| ▶ 거절한다 ("우린 빼앗는 쪽이다") | 사기 변동 없음 |
| ▶ 노인을 거점에 받아들인다 | 비전투 NPC 획득, 식량 소모 증가, 특수 이벤트 해금 |

**5-2. 끝없는 허기** (영웅 2명 이상 죄종 불만 시)
장면: *"동굴의 기운입니다. 영웅들이 갑자기 무언가에 굶주리기 시작합니다. 식량이 아니라 — 각자의 욕망이 폭발합니다."*

| 선택지 | 결과 |
|--------|------|
| ▶ 욕망을 채워준다 | 자원 대량 소모, 전체 사기 회복 |
| ▶ "참아라" | 전체 사기 하락, 폭주 위험 증가 |
| ▶ 원정을 서두른다 | 보스 원정 강행 가능, 준비 부족 |

#### 보스 서사
- **브리핑**: *"바알제붑. 한 번의 충동이 모든 것을 무너뜨린 놈."*
- **유언**: *"돌려받으려 했어... 안 됐어. 이미 소화됐으니까."*
- **기억 파편**: *"복수해도 잃어버린 것은 돌아오지 않는다. 처음으로 감사를 느꼈다. 인간에게. — 조종한 놈이 있다."*

### Chapter 6 — 색욕 / Lust 「타락한 궁전」

> 보스: 아스모데우스 (Asmodeus) — 처음부터 적이었던 자
> 플레이어 체감: **"누구를 믿을 것인가?"**

#### 환경 변조
| 변조 | 효과 |
|------|------|
| 내통자 시스템 | 챕터 시작 시 영웅 1명에 은밀히 내통자 태그 부여 |
| 정보 불확실 | 밤 습격 규모 예고 정확도 하락 (감시탑 효율 감소) |
| 관계 이벤트 증가 | 영웅 간 관계 이벤트 빈도 2배 |
| 배신 위험 | 내통자 존재 시 밤 습격 적 강화 (약점 유출) |

#### 챕터 이벤트

**6-1. 내통자** (챕터 시작 시 고정)
장면: *"밤 습격 패턴이 이상합니다. 적이 약점을 정확히 알고 있습니다. 누군가 정보를 흘리고 있습니다."*

| 선택지 | 결과 |
|--------|------|
| ▶ 영웅을 조사한다 (1명 지목) | 맞으면 제거, 틀리면 지목된 영웅 사기 급락 |
| ▶ 전체를 감시한다 | 전체 사기 하락, 3턴 후 내통자 힌트 추가 |
| ▶ 무시한다 | 밤 습격 난이도 증가 지속 |
| ▶ "스파이가 있다" 공개 선언 | 내통자 행동 멈춤, 전체 불신 분위기 |

**6-2. 아름다운 함정** (보스 원정 해금 시 고정)
장면: *"궁전에서 초대장이 왔습니다. 아스모데우스가 평화 회담을 제안합니다. '피를 흘릴 필요 없습니다. 이야기합시다.'"*

| 선택지 | 결과 |
|--------|------|
| ▶ 함정이다, 공격한다 | 보스전 즉시 돌입 (정면 전투) |
| ▶ 회담에 응한다 | 특수 이벤트 체인 → 배신 → 불리한 조건으로 보스전 |
| ▶ 정찰을 보낸다 | 1턴 소모, 함정 정보 획득, 보스전 유리 |

#### 보스 서사
- **브리핑**: *"아스모데우스. 충성한 적이 없다. 처음부터 적이었다."*
- **유언**: *"후회? 임무였을 뿐이다."*
- **기억 파편**: *"혼자라는 것이 처음으로 고통스럽다. 분노가 방향을 바꾼다. 악마들에서 천상으로. — 마지막 한 명. 루시퍼."*

### Chapter 7 — 교만 / Pride 「신의 폐허」

> 보스: 루시퍼 (Lucifer) — 배신이 아니었다
> 플레이어 체감: **"내가 옳았는가?"**

#### 환경 변조
| 변조 | 효과 |
|------|------|
| 과거 결산 | 이전 챕터 선택에 따른 결산 이벤트 발생 |
| 난이도 변동 | 최종 원정 난이도가 이전 플레이 방식에 따라 변동 |
| 충성 재판정 | 영웅들의 "끝까지 따를 것인가" 이벤트 발생 |
| 이벤트 톤 변화 | 이전 챕터 선택 경향에 따라 이벤트 텍스트 톤 분기 |

#### 과거 결산 규칙

플레이어의 이전 챕터 행동이 챕터 7의 상황을 결정한다:

| 이전 플레이 경향 | 챕터 7 결과 |
|-----------------|-----------|
| 폭압적 (처벌, 추방 다수) | 영웅 동요, 반란 이벤트 발생 위험 |
| 관대함 (용서, 투자 다수) | 영웅 결속, 자발적 전투 의지 |
| 방관 (무시 다수) | 무질서, 통제력 상실 위험 |
| 균형적 | 안정적, 특별 보너스 없음 |

#### 챕터 이벤트

**7-1. 마지막 결속 (또는 반란)** (챕터 시작 시 고정, 과거 결산 기반 분기)

분기 A — 영웅 이탈 많았거나 폭압적이었을 때:
장면: *"영웅들이 동요합니다. '천상까지? 우리가 왜?' 남은 자들의 눈에 확신이 없습니다."*

| 선택지 | 결과 |
|--------|------|
| ▶ "여기까지 왔다. 끝내야 한다" | 설득 성공 시 잔류, 실패 시 추가 이탈 |
| ▶ "원하지 않는 자는 가라" | 이탈자 발생, 잔류자 사기 상승 |
| ▶ "명령이다. 따르라" | 전체 사기 하락, 이탈 없음 (공포) |

분기 B — 균형적 또는 관대했을 때:
장면: *"영웅들이 스스로 무기를 들었습니다. '끝내러 갑시다.' 누가 시키지 않아도."*

| 선택지 | 결과 |
|--------|------|
| ▶ "함께 간다" | 전체 사기 대폭 상승, 보스전 팀 버프 |
| ▶ "각오는 됐나" | 전체 사기 상승 |

**7-2. 배신이 아니었다** (보스 원정 해금 시 고정)
장면: *"신의 폐허 입구. 루시퍼가 서 있다. 칼을 들고 있지만 — 공격하지 않는다."*

| 선택지 | 결과 |
|--------|------|
| ▶ "변명을 들을 시간은 없다" | 보스전 즉시 돌입 |
| ▶ "...말해봐라" | 진실 해금 → 보스전 난이도 하락, 감정적 무게 증가 |
| ▶ "알고 있었다" | 특수 대사, 보스전 조건 변화 |

#### 보스 서사
- **브리핑**: *"루시퍼. 가장 믿었던 자. 배신의 밤에 그 자리에 없었던 자."*
- **유언**: *"나는 너를 배신한 게 아니다. 네게 말하면 너는 막았을 거다. 그러면 다 죽었다. 미안하다. 이 방법밖에 없었다."*
- **기억 파편**: *"바알은 모든 힘을 되찾았다. 그리고 처음으로 — 운다. 가장 충성스러운 자를 자신의 명령으로 죽였다."*

|------|----------|

| 4 나태 | 스테이지 3개 클리어 + 이탈 없이 유지 |
| 5 폭식 | 스테이지 3개 클리어 |

## 6. 챕터 전환

보스 처치 → 다음 챕터로 전환 시:

3. 다음 챕터 환경 변조 적용
4. 챕터 시작 고정 이벤트 발생
