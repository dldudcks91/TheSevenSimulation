# 원정 시스템 설계

> 상태: 구조 확정, 수치 미확정
> 작성일: 2026-03-24

---

## 1. 개요

던전 탐색이 아닌 **파견 방식**. 파티를 보내고 결과를 받는다.

## 2. 원정 흐름

```
파티 편성 (3명) → 목표 선택 → 파견
  → [낮 동안 자동 진행]
  → 저녁에 결과 보고
    → 승리: 전리품 + 경험치 + "더 진행?" 선택
    → 패배: 부상/사망 + 부분 전리품 + 강제 귀환
```

## 3. 7챕터 구조 (TheSevenTactics 계승)

| 챕터 | 죄종 | 지역 | 보스 | 스테이지 |
|------|------|------|------|---------|
| 1 | 분노 | 불타는 전장 | 사탄 | 3 + 보스 |
| 2 | 시기 | 뒤틀린 숲 | 레비아탄 | 3 + 보스 |
| 3 | 탐욕 | 황금의 사막 | 맘몬 | 3 + 보스 |
| 4 | 나태 | 망각의 동토 | 벨페고르 | 3 + 보스 |
| 5 | 폭식 | 심연의 동굴 | 바알제붑 | 3 + 보스 |
| 6 | 색욕 | 타락한 궁전 | 아스모데우스 | 3 + 보스 |
| 7 | 교만 | 신의 폐허 | 루시퍼 | 3 + 보스 |

## 4. 원정 결과

| 결과 | 보상 | 리스크 |
|------|------|--------|
| 스테이지 클리어 | 자원 + 장비 + 경험치 | 부상 가능 |
| 보스 클리어 | 유니크 장비 + 다음 챕터 해금 + 영웅 영입 | 사망 위험 |
| 패배 | 전리품 절반 유지 | 부상/사망 |
| 철수 | 전리품 전부 유지 | 없음 |
