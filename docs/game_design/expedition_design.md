# 원정 시스템 설계

> 상태: 프로토타입 구현 완료 (노드/주사위 2모드)
> 작성일: 2026-03-24
> 마지막 업데이트: 2026-04-15

---

## 1. 개요

영웅을 파견하여 챕터 맵을 탐색하는 **액티브 원정 방식**.  
유저가 직접 노드를 선택하거나 주사위를 굴리며 진행한다.

### 핵심 제약
- 원정은 **하루 단위** — 아침 출발, 저녁 귀환 (1턴 소모)
- 원정 중인 영웅은 그날 밤 **방어전에 참여 불가**
- 방어전은 거점에 남아있는 영웅만 참여

---

## 2. 원정 흐름

```
[원정 탭] → 원정 맵 열기 버튼
  → ExpeditionScene (오버레이 씬)
    → 노드 클릭 / 주사위 굴리기
      → 전투 / 이벤트 / 야영 / 포탈 처리
        → 귀환 버튼 (언제든 가능)
          → MapScene 복귀 + 탭 갱신
```

---

## 3. 두 가지 원정 모드

인게임 설정(HUD ⚙ 또는 씬 내 버튼)으로 전환 가능. 기본값: STS 노드.

### 3-A. STS 노드 방식 (기본)

STS(Slay the Spire) 스타일 분기 노드맵.

```
          [보스 💀]
         /        \
   [전투 ⚔]     [야영 🏕]
         \        /
          [포탈 🌀]  ← 체크포인트
         /        \
   [전투 ⚔]     [조우 ?]
         \        /
          [출발 ★]
```

- **구름 가리기(Fog of war)**: 현재 노드의 연결 노드만 공개. 나머지는 '???' 로 표시
- **미래 확장**: 감시탑 건물로 추가 노드 미리보기 가능
- **선택 전략**: 현재 파티 상태(사기/HP)에 따라 전투 vs 야영 선택

#### 노드 유형

| 타입 | 아이콘 | 색상 | 효과 |
|------|--------|------|------|
| start | ★ | 초록 | 출발점 |
| combat | ⚔ | 빨강 | 자동전투 |
| event | ? | 파랑 | 조우 이벤트 (구현 예정) |
| rest | 🏕 | 초록 | 야영 — 사기 +10 |
| portal | 🌀 | 보라 | 포탈 — 체크포인트 설정 |
| boss | 💀 | 금색 | 보스 전투 |

### 3-B. 주사위 방식

직선 8칸 경로, 1~3 주사위로 이동.

```
[출발★] → [전투⚔] → [조우?] → [야영🏕] → [포탈🌀] → [전투⚔] → [조우?] → [보스💀]
```

- 주사위를 굴리면 1~3칸 전진, 도착 칸 이벤트 자동 처리
- 현재 칸 처리 전까지 다음 굴리기 불가
- **에이전시 없음** — 이동 결과는 완전 랜덤

---

## 4. 포탈 체크포인트 규칙

| 상황 | 다음 원정 시작점 |
|------|-----------------|
| 포탈 도달 후 귀환 | 포탈 이후 노드부터 출발 |
| 포탈 미도달 귀환 | 처음부터 (포탈 이전 클리어 노드는 영구 유지) |

- **포탈 이전 노드**: 영구 클리어 — 다시 밟아도 이벤트 없음
- **포탈 이후 노드**: 귀환 시 리셋 — 다음 원정에서 다시 도전

## 5. 7챕터 구조 (TheSevenTactics 계승)

| 챕터 | 죄종 | 지역 | 보스 | 스테이지 |
|------|------|------|------|---------|
| 1 | 분노 | 불타는 전장 | 사탄 | 3 + 보스 |
| 2 | 시기 | 뒤틀린 숲 | 레비아탄 | 3 + 보스 |
| 3 | 탐욕 | 황금의 사막 | 맘몬 | 3 + 보스 |
| 4 | 나태 | 망각의 동토 | 벨페고르 | 3 + 보스 |
| 5 | 폭식 | 심연의 동굴 | 바알제붑 | 3 + 보스 |
| 6 | 색욕 | 타락한 궁전 | 아스모데우스 | 3 + 보스 |
| 7 | 교만 | 신의 폐허 | 루시퍼 | 3 + 보스 |

## 6. 원정 결과

| 결과 | 보상 | 리스크 |
|------|------|--------|
| 스테이지 클리어 | 자원 + 장비 + 경험치 | 부상 가능 |
| 보스 클리어 | 유니크 장비 + 다음 챕터 해금 + 영웅 영입 | 사망 위험 |
| 패배 | 전리품 절반 유지 | 부상/사망 |
| 귀환 (자발) | 전리품 전부 유지 | 없음 |
