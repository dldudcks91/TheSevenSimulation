# 전투 시스템 설계

> 상태: TheSevenTactics 계승, 시뮬레이션 맥락으로 조정
> 작성일: 2026-03-24

---

## 1. 개요

**완전 자동전투** — 배치/조작 없음. 플레이어 판단은 전투 전(편성, 장비)과 전투 후(계속/철수)에 집중.

### 전투 유형

| | 원정 전투 | 방어 전투 (밤 습격) |
|---|---|---|
| 진행 | 파견 후 결과만 받음 | **실시간으로 지켜봄** |
| 리플레이 | 결과 화면에서 다시 볼 수 있음 | - |
| 조작 | 없음 (완전 자동) | 없음 (완전 자동) |

핵심 딜레마: 원정에 보낸 영웅은 밤 방어에 못 씀.

## 2. 전투 구조

- 7스탯 기반 자동 계산
  - 원정 전투: 힘(공격), 민첩(행동순서), 체력(생존), 지능(AI 판단)
  - 밤 방어: 힘(공격), 통솔(팀 버프), 체력(생존)
- 명중/회피 없음, 마나 없음
- 배치 없는 완전 자동

## 3. 사망 시스템 (TheSevenTactics 계승)

| 상태 | 조건 | 효과 |
|------|------|------|
| 기절 | HP 0 | 귀환 후 회복 (1~2턴) |
| 중상 | 기절 + 추가 피격 | 장기 치료 (3~5턴) |
| 사망 | 중상 + 추가 피격 | **영구 상실** |

---

## 5. 밤 습격 (거점 방어전)

- **매일 밤** 적 습격 발생
- 규모 가변: 소규모 정찰 ~ 대규모 공성 (챕터 진행에 따라 강화)
- 거점에 남은 영웅으로 자동 방어
- **감시탑** 업그레이드 → 습격 규모/적 구성 사전 정보 제공
- 방어 실패 시: 시설 피해, 자원 손실
