## 1. 개요

**X축 오토배틀 + 카드** — 영웅들이 자동으로 싸우고, 플레이어(바알)는 SP를 모아 카드를 사용하여 전장에 개입한다.

### 핵심 컨셉
- 영웅은 **알아서 싸운다** (자동 이동, 자동 공격)
- 플레이어는 **바알 = 지휘관** — 카드로 명령을 내림
- 전투 전: 파티 편성 + 덱 구성
- 전투 중: SP 관리 + 카드 사용 타이밍
- 전투 중 죄종 기반 **일기토** 자동 발동 (드라마 연출)

### 참고작
- **Crusaders Quest**: 횡스크롤 자동전투 + 스킬 블록
- **삼국지9/14**: 반자동 전투 — 지시 후 자동 진행
- **HoMM3**: 지휘관 버프, 파티 편성
- **Darkest Dungeon**: 로스터 관리, 영웅 상실 위험

### 시각 규격
- 해상도: 1280x720
- 스프라이트: LPC 64x64, 런타임 합성 (SpriteRenderer)
- 방향: East(오른쪽)만 — 단일 행 스프라이트시트
- 액션: walk(이동/대기), slash(공격)
- 병사: 스프라이트 없음, 카운터 텍스트로 표시

## 3. 전투 유형

| | 원정 전투 | 방어 전투 (밤 습격) | 사냥 |
|---|---|---|---|
| 편성 | 영웅 1~3명 | **방어 배치된 영웅만** | 영웅 1명 |
| 진행 | 파견 후 결과 받음 (리플레이 가능) | **실시간 관전** (카드 사용) | 실시간 1:1 전투 |
| 카드 | 사용 가능 | 사용 가능 | 사용 불가 |
| 증원 | 불가 (원정 중) | 불가 | 불가 |

핵심 딜레마: 낮에 영웅을 **어디에 배치할 것인가** — 건설/원정/사냥에 보내면 방어 인원이 줄어듦. 방어에 넣으면 경제 활동 불가.

## 4. SP 시스템

| 항목 | 값 |
|------|-----|
| SP 최대 | 100 |
| 매 라운드 획득 | +2 |
| 적 처치 시 | +10 |
| 카드 사용 시 | SP 소모 (카드별 다름) |

SP는 전투 시작 시 0에서 시작. 시간이 지나면 자연 회복 + 적 처치 보너스.

## 5. 카드 시스템

### 개요
- 거점에서 카드를 획득/제작 (연금술소, 상인, 원정 보상)
- 원정/방어전 전에 **덱 구성** (보유 카드 중 4~5장 선택)
- 전투 중 SP가 충분하면 카드 터치하여 사용
- 카드는 **소모품** — 사용하면 사라짐

### 프로토타입 카드 5종

| 카드 | SP | 대상 | 효과 | 지속 | 죄종 시너지 |
|------|-----|------|------|------|-----------|
| 🔥분노폭발 | 30 | 아군 전원 | ATK +50% | 3턴 | [분노] +80% |
| 🛡보호막 | 20 | 아군 전원 | 피해 -40% | 3턴 | [나태] 5턴 |
| 💀저주 | 40 | 적 전원 | ATK -30% | 5턴 | [시기] SPD도 -30% |
| ❤회복 | 25 | 아군 전원 | HP 20% 회복 | 즉시 | [폭식] 30% |
| ⚡속도증가 | 20 | 아군 전원 | SPD +50% | 3턴 | [교만] 첫타 2배 |

### 죄종 시너지
파티 내 해당 죄종 영웅이 있으면 카드 효과가 강화된다.
→ **파티 편성 + 덱 구성이 동시에 전략**

### 원정 연속 스테이지와의 연계
```
스테이지1: 카드 5장 가져감 → [분노폭발] 사용 → 남은 4장
스테이지2: 카드 4장으로 싸움 → [회복] 사용 → 남은 3장
"더 진행?" → 카드 3장으로 보스전 갈 수 있나?
```

## 6. 일기토 시스템

### 개요
전투 중 영웅의 **죄종 성격에 따라** 자동으로 적에게 일기토를 신청한다.
플레이어가 개입하지 않음 — 영웅 개인의 드라마 연출.

### 죄종별 일기토 성향

| 죄종 | 확률 | 대상 선택 |
|------|------|----------|
| 분노 | 50% | 랜덤 (싸우고 싶어 못 참음) |
| 교만 | 35% | 가장 강한 적 (자존심) |
| 시기 | 25% | 랜덤 (다른 영웅이 활약하면) |
| 탐욕 | 20% | 랜덤 |
| 폭식 | 15% | 가장 약한 적 (확실한 먹잇감) |
| 나태 | 5% | 거의 안 함 |
| 색욕 | 5% | 거의 안 함 |

### 적 수락/거부
- 수락 확률: 70%
- 거부 시: 적 전체 사기↓ + 신청 영웅 사기↑

### 일기토 시각 연출
1. 오토배틀 일시정지
2. 화면 어둡게 + 중앙에 1:1 배치
3. 선제(민첩 비교) → 한 대씩 주고받기
4. 승패 후 오토배틀 복귀

### 결과 파급

| 결과 | 효과 |
|------|------|
| 아군 승리 | 적 전체 사기↓, 오토배틀 유리 + 신청 영웅 사기↑ |
| 아군 패배 | 아군 사기↓ + 영웅 부상/사망 → 2명으로 전투 |
| 적 거부 | 적 사기↓ (소량) + 신청 영웅 사기↑ |

## 7. 전투 구조

### X축 오토배틀
- 영웅(좌) vs 적(우), X축 이동만
- 가장 가까운 적을 향해 자동 이동
- 교전 거리 도달 → 자동 공격
- 근접/원거리 사거리 구분 (스프라이트별)

### 7스탯 전투 역할
- **힘**: 공격 데미지
- **민첩**: 행동 순서 + 이동 속도
- **체력**: HP 풀
- **통솔**: 방어전 ATK 보정
- **지능/감각/매력**: 전투 미반영 (거점 행동용)

### 공식 (balance.csv 기반)
```
HP = hp_base(200) + 체력 × hp_vitality_mult(15) + 힘 × hp_strength_mult(5)
ATK(원정) = 힘 × atk_expedition_str_mult(0.7) + 민첩 × atk_expedition_agi_mult(0.4)
ATK(방어) = 힘 × atk_defense_str_mult(0.8) + 통솔 × atk_defense_lead_mult(0.4)
데미지 = ATK × (0.8~1.2 랜덤) × 버프배율 × (1 - 피해경감)
```

## 8. 사망 시스템

### 영웅

| 상태 | 조건 | 효과 |
|------|------|------|
| 기절 | HP 0 | 귀환 후 회복 (1~2턴) |
| 중상 | 기절 + 추가 피격 | 장기 치료 (3~5턴) |
| 사망 | 중상 + 추가 피격 | **영구 상실** |

### 병사
- HP 0 → **즉시 사망, 영구 소실**
- 병사는 소모품. 전투 후 생존 병사만 풀로 복귀

---

## 9. 병사 시스템

| 항목 | 값 |
|------|-----|
| 종류 | 민병 (1종) |
| HP | 80 |
| ATK | 8 |
| SPD | 3 |
| 시작 시 | 100명 |
| 모집 | 골드로 (비용 후일 기획) |

- 화면에 스프라이트 없음 (카운터 표시)
- 적은 병사를 우선 공격 → 병사 전멸 시 영웅이 직접 맞음
- 원정 배분 vs 방어 잔여 = 핵심 딜레마

## 10. 증원 시스템

방어전에서 대기 영웅을 전투 중 투입 가능.

| 항목 | 내용 |
|------|------|
| 대상 | 거점 대기 중인 영웅 |
| 비용 | 없음 (다음 턴 피로 페널티) |
| 등장 | 필드 좌측 가장자리 |
| 제한 | 대기 영웅 수만큼 |

---

## 11. 밤 습격 (거점 방어전)

- **3~5일 간격**으로 습격 발생 (매일 밤 아님, balance.csv: raid_interval_min/max)
- 습격 없는 밤: 평화롭게 결산으로 넘어감
- **MapScene 위 오버레이**로 전투 진행 (별도 씬 전환 없음, MapDefenseMode)
- 규모: 소규모 정찰 ~ 대규모 공성 (day + 챕터 기반 스케일링)
- **방어: 낮에 "방어" 임무에 배치된 영웅만 참전** (자동 참여 아님)
- 방어 배치 0명 = 무방비 → 습격 자동 패배
- SP/카드 사용 가능 (BattleSceneA와 동일 시스템)
- 감시탑 레벨에 따라 습격 정보 제공
- 방어 실패: 시설 피해, 자원 손실
