# 죄종 시스템 상세 설계

> 상태: 쌓임(누적) 프레임 + 7영역×7스탯 삼각 매핑 + 4구간 확정 (2026-04-20)
> 작성일: 2026-03-24 | 마지막 업데이트: 2026-04-20

---

## 1. 바알(플레이어)의 역할

바알은 **판결자이자 영주**다. 바알 자신의 유동 수치는 없고, 대신 **국시(國是, Edict)** 를 통해 통치 방향을 직접 결정한다:
- 7죄종 중 1개를 **국시로 선포** (기간형 5턴 지속)
- 선포한 국시의 효과가 거점 전반에 적용 (전투력/자원/생산성/영웅 수치 변동)
- 만료 후 공백 기간(1턴)을 거쳐 다음 국시 선포 가능
- 이벤트 선택지는 여전히 **영웅들의 죄종 수치**에 영향
- 상세 규칙: §5 국시(Edict) 시스템 참조

## 2. 7감정 모델 — 죄종 수치 = 영웅의 감정 상태

> **사기(Morale) 시스템 완전 제거.** 죄종 7수치 = 영웅이 현재 가진 7가지 감정의 누적치.

### 서사 프레이밍
- 죄종 수치 = **"영웅에게 들어간 나쁜 마음이 쌓이는 정도"** = 7가지 감정의 게이지
- 영웅은 0(깨끗한 상태)에서 시작, 바알 밑에서 행동할수록 각 감정이 쌓임
- 관리 = **"오염을 방지/정화"** — "행복하게 하는 게임"이 아니라 "덜 타락하게 지키는 게임"
- DD 스트레스 모델 + 7축 분해: 스트레스가 7개 채널로 분산되어 저마다 다른 드라마 생성

### 7죄종 = 7감정

| 죄종 | 감정 본질 |
|------|---------|
| 분노 | 화 / 적개심 |
| 교만 | 우월감 / 자존감 비대 |
| 탐욕 | 갈망 / 소유욕 |
| 시기 | 질투 / 박탈감 |
| 색욕 | 집착 / 갈구 |
| 폭식 | 결핍 / 공허 |
| 나태 | 무기력 / 의욕 상실 |

### 구조
- **죄종 7수치** (각 **0~20**) — 영웅 생성 시 **모두 0**, 행동마다 **쌓이는(단방향 누적)** 상태 게이지
- 사기(0~100) 완전 폐기 — 별도 관리 수치 없음
- 행동은 **주로 올림** — 쌓인 죄종은 **전용 정화 수단**(연회/야영/이벤트/시설/특성)으로만 감소
- **높은 수치 = 해당 영역 파워 보너스 + 문제 행동 위험** → 플레이어 딜레마
- **낮은 수치 = 깨끗한 상태(좋음)** — 저수치 페널티 없음

### 메인 죄종 개념 폐기
- 영웅의 "메인 죄종"을 생성 시 확정하지 않음
- `topSin()` = "현재 가장 많이 쌓인 죄종" 이라는 **상태** 표현일 뿐, 정체성이 아님
- 영웅 정체성은 **시작특성(trait)** 이 담당 (→ `hero_design.md`)

### 7죄종 × 7영역 × 7스탯 삼각 매핑 (2026-04-20 확정)

| 죄종 | 영역 (도메인) | 관련 스탯 | 주 게임 활동 |
|---|---|---|---|
| **분노** | 🗡️ 전투 | 힘 | 원정 전투, 사냥, 방어전 |
| **교만** | 👑 명성·지휘 | 통솔 | 파티 리더, 외교, 위계 |
| **탐욕** | 💰 자원·경제 | 지능 | 채집, 벌목, 거래, 전리품 |
| **시기** | 👁️ 정찰·정보 | 민첩 | 정찰, 약점 간파, 비밀 수집 |
| **색욕** | 💋 사회·관계 | 매력 | 주점, 고용, 관계 형성 |
| **폭식** | 🍖 식량·체력 | 건강 | 농장, 식량 조달, HP |
| **나태** | 🛌 회복·관조 | 감각 | 숙소, Stamina 회복, 지속력 |

모든 게임 활동은 이 7영역 중 하나에 속하며, 해당 죄종 쌓임·해당 스탯 활용과 연동된다.

### 수치 구간 (2026-04-20 확정)

| 구간 | 수치 | 스탯 보너스 | 효과 |
|---|---|---|---|
| **깨끗** | 0~4 | 없음 | 영향 없음 (정상 상태) |
| **발현** | 5~11 | 관련 스탯 **+1** | 감정이 드러남 (개성 표현), 대사 톤 변화만 |
| **고양** | 12~17 | 관련 스탯 **+2** | 상시 패시브(장점) + 상시 페널티(단점) 활성 |
| **폭주** | 18~19 | 관련 스탯 **+3** | 매 턴 강제 문제 행동 (`isRampaging=true`) |
| **임계** | **20 도달** | +3 유지 | **죄종별 결정적 순간 이벤트** 즉시 발동 |

- **엄격 구간 경계**: 수치 1 차이로 구간 전이 즉각 발생 (부드러운 전이 없음)
- **스탯 보너스 치환형**: 발현의 +1은 고양 진입 시 +2로 **교체** (누적 X)
- **깨끗 구간 넓게 (5칸)**: 대부분 시간은 여기 머물도록 설계

### 발현 구간 (5~11) — 스탯만 +1

발현 구간에는 고유 기능 없음. 단순히 **관련 스탯 +1**. "이 영웅의 감정이 드러난다" 수준의 개성 표현.

### 고양 구간 (12~17) — 장점·단점 매트릭스

> **설계 원칙**: 장점과 단점은 **서로 다른 영역**에 위치해야 함 (같은 차원에서 상쇄 X)
> **획득 방식**: C — 죄종당 장점 1개 + 단점 1개 **고정** (Phase 2에서 시작특성 연동 랜덤 편향 도입 예정)

### 폭주 구간 (18~19) — 스탯 +3 + 매 턴 강제 행동

| 죄종 | 영역 | 스탯 | 매 턴 강제 행동 |
|---|---|---|---|
| 분노 | 전투 | 힘 +3 | 도발 — 랜덤 영웅 분노 +2 |
| 교만 | 명성·지휘 | 통솔 +3 | 명령 거부 — 지시 20% 무시 |
| 탐욕 | 자원·경제 | 지능 +3 | 횡령 — 식량 -2 or 골드 -5 |
| 시기 | 정찰·정보 | 민첩 +3 | 독살 시도 — 강한 영웅 작업 방해 |
| 색욕 | 사회·관계 | 매력 +3 | 집착 — 대상 분리 거부 20% |
| 폭식 | 식량·체력 | 건강 +3 | 과식 — 식량 -3, HP +10 |
| 나태 | 회복·관조 | 감각 +3 | 태업 — 작업 20% 실패 |

- **해제**: 수치 17 이하로 내려가면 자동 폭주 해제
- **트리거 범위**: 어느 죄종이든 18+ 도달 시

### 임계 구간 (20 도달) — 결정적 순간 이벤트 (2026-04-21 세부 확정)

기존 "20 + 3턴 = 자동 이탈"을 **죄종별 분기 이벤트**로 승격. Darkest Dungeon의 Virtue/Affliction 구조.

#### 공통 구조

- 수치 20 도달 즉시 이벤트 발동 (3턴 대기 없음)
- 플레이어에게 선택지 3개 제시: **이탈 수용 / 구원 시도 / 극단 조치**
- **이탈 수용** → 결과 확정 (roll 없음)
- **구원 시도** → 성공 roll 필요. 실패 시 극단 분기로 귀결
- **극단 조치** → 결과 확정 (roll 없음)

#### 구원 성공 확률

| 상태 | 구원 성공 확률 |
|---|---|
| 기본 | **20%** |
| bonds 평균 ≥ 70 | +20% (동료들이 끌어줌) |
| bonds 평균 ≤ 30 | -10% |
| 시작특성 저항 계열 | +20% |
| 시작특성 편향 계열 | -10% |
| 후천 구원 특성 보유 | +15% |
| **최대** | 65% |
| **최소** | 5% |

## 2-A. 영웅 관계 시스템 (bonds) — 2026-04-20 도입 / 2026-04-24 판정·배율 확정

### 구조
- `hero.bonds = { [otherHeroId]: 0..100 }` — 영웅간 친밀도
- **Asymmetric** — A→B와 B→A가 다를 수 있음
- 초기값 50 (중립)
- **바알↔영웅 친밀도는 별도 축으로 두지 않음** — 후천 특성으로 이산 기록

### 수치 구간 해석

| 구간 | 의미 |
|---|---|
| 0~20 | 증오/원수 |
| 20~40 | 불신/껄끄러움 |
| 40~60 | 중립 |
| 60~80 | 호감 |
| 80~100 | 절친/혈맹 |

### 바알-영웅 관계를 별도 축으로 두지 않는 이유

- 영웅은 "결함 있는 인간"(용병/떠돌이) — 봉신이 아님
- 충성심이 핵심 축이면 CK3/삼국지 계보, 우리 게임은 **RimWorld/DD 계보 (감정 포화 서사)**
- 영웅의 바알에 대한 불만은 **죄종 수치**가 이미 담당 (돈 못 줌 → 탐욕·분노 쌓임)
- 장기 통치 성적은 **후천 특성** 획득(신뢰/원한 등)으로 이산 기록

---

### 성향 판정 규칙 (2026-04-24 확정)

bonds 변동 시 두 영웅의 **현재 성향**을 §3 관계 매트릭스에 대입해 배율을 결정한다.

#### 판정 본체: topSin

- 영웅의 **현재 sinStats 중 최고값 죄종**을 성향으로 본다
- **시작특성(trait)은 bonds 판정에 개입하지 않음** — 시작특성은 "sinStats 쌓임 속도의 편향"이라는 역할에만 집중
- **"메인 죄종 개념 폐기" 원칙 유지** — topSin은 "현재 상태"일 뿐, 고정 정체성이 아님

#### 동률 해체: 숨은 우선순위 순열

- 영웅 생성 시 **7죄종을 랜덤 셔플**하여 `hero.sinPriority`로 영구 저장
  ```
  hero.sinPriority = ['탐욕', '나태', '분노', '시기', '색욕', '폭식', '교만']  // 예시
  ```
- sinStats 동률(초반 모두 0 포함) 시 이 순서로 해체 → 1순위 죄종이 성향이 됨
- **플레이어에게 비공개** (영웅의 "숨은 기질")
- 동률 해체 시에도 성향이 영웅마다 다르게 나오므로 **초반 무력화 문제 자연 해결**

#### 판정 시점

- **사건 발생 순간의 스냅샷**으로 1회 계산
- **매 턴 자동 변동 없음** (순환 피드백 방지)
- 긍정 사건만 매트릭스 배율 적용. 부정 사건(배신/폭주 피해)은 원본 Δ 그대로

---

### bonds 변동 배율 계산 순서 (2026-04-24 확정)

```
1. 기본 Δ 산출 (사건별 수치 — 추후 확정)
2. 긍정 사건이면 × 관계배율 (동류 ×2 / 강화 ×1.5 / 중립 ×1 / 대립 ×-2)
3. × 색욕 고양 배율 (×2, 주체 색욕 12~17일 때)
4. × 열정령 국시 배율 (×1.5, 국시 활성 시)
5. Math.round
6. 분노 고양 가드 (주체 분노 12~17 && Δ>0 → Δ=0)
7. bonds clamp [0, 100]
```

### 매트릭스 적용 범위

| 사건 카테고리 | 매트릭스 적용? | 근거 |
|---|:-:|---|
| **긍정 사건** (공유 경험, 구조, 선행) | ✅ 적용 | 같은 경험도 성격이 다르면 반응이 다름 |
| **부정 사건** (폭주 피해, 배신, 횡령) | ❌ 미적용 | 배신은 누구에게나 배신. 원본 Δ 그대로 |
| **상시 특성 배경** (청빈, 대범 등 /턴) | ❌ 미적용 | 특성 자체가 성격 표현 — 이중 적용 방지 |

### 사건별 Δ 수치표

> **미확정** — 실제 게임 메커닉(원정 귀환·사냥·연회·방어전·이벤트·폭주 등)과 연동하여 추후 확정. 최소한 구현된 원정 귀환 시 변동(±3 중립 기준)은 유지.

### 관계 → 죄종 자동 반영 (기존 규칙 유지)

관계 변화는 별도 "감정 축"을 만들지 않고 **죄종 수치 쌓임의 증폭/억제**로 자동 반영된다.

| B가 죽음 | A.bonds[B] ≤ 30 | A 영향 없음 (혹은 교만 +1) |
| B가 원정 성공 | A.bonds[B] ≤ 30 | A의 시기 +2 |
| 같은 시설 + A.bonds[B] < 30 | — | A·B 모두 분노 +1/턴 |
| 같은 시설 + A.bonds[B] > 70 | — | 쌓임 없음 (친구 효과) |

|---|:-:|---|
| **동류** (같은 죄종 topSin) | **×2** | "같은 감정을 공유하는 자끼리" — 가장 강한 결속 |
| **강화** | **×1.5** | 한 짝의 성향 궁합 — 서로를 북돋움 |
| **중립** | **×1** | 특별한 상호작용 없음 |
| **대립** | **×-2** | 한 짝의 성향 충돌 — 부호 반전 |

### 페어 구성 요약

- **동류 ++** 7쌍 (대각선, 같은 죄종끼리)
- **강화** 3쌍: 분노↔시기 / 탐욕↔색욕 / 폭식↔나태
- **대립** 9쌍:
  - 교만 → 타 6개 죄종 모두 (6쌍) — 교만 완전 고립
  - 분노↔나태 / 탐욕↔폭식 / 시기↔색욕 (3쌍)

### 죄종별 관계 분포

| 죄종 | 강화 | 대립 | 중립 | 특징 |
|---|:-:|:-:|:-:|---|
| **교만** | 0 | 6 | 0 | **자기밖에 모른다** — 모든 타 죄종과 대립 |
| **시기** | 1 (분노) | 2 (교만, 색욕) | 3 | 박탈감 × 집착의 충돌 |
| **분노** | 1 (시기) | 2 (교만, 나태) | 3 | 공격성 × 무기력의 충돌 |
| **나태** | 1 (폭식) | 2 (교만, 분노) | 3 | 수동적 향락 계열 |
| **탐욕** | 1 (색욕) | 2 (교만, 폭식) | 3 | 쌓음 × 소비의 충돌 |
| **폭식** | 1 (나태) | 2 (교만, 탐욕) | 3 | 수동적 향락 계열 |
| **색욕** | 1 (탐욕) | 2 (교만, 시기) | 3 | 욕망 × 박탈감의 충돌 |

### 서사 근거

| 페어 | 관계 | 해석 |
|---|---|---|
| 분노 ↔ 시기 | 강화 | 박탈감 → 적개심 (카인/아벨 모티프) |
| 탐욕 ↔ 색욕 | 강화 | 물질·육체 욕망의 동맹 |
| 폭식 ↔ 나태 | 강화 | 수동적 향락 계열 동류 |
| 분노 ↔ 나태 | 대립 | 공격성 vs 무기력 |
| 탐욕 ↔ 폭식 | 대립 | 쌓음 vs 소비 |
| 시기 ↔ 색욕 | 대립 | 박탈감 vs 집착 대상 |
| 교만 vs 전 죄종 | 완전 대립 | **"모든 죄의 근원"** — 루시퍼 추락 모티프 |

### 관계가 게임에 미치는 영향

이 매트릭스는 두 방향으로 작용한다:

1. **bonds 변동 배율** (§2-A bonds 시스템 참조) — 사건 발생 시 두 영웅의 topSin을 매트릭스에 대입해 변동량 배율 결정
2. **죄종 쌓임 증폭/억제** (기존 규칙 유지) — 같은 파티 배치 시 관련 죄종 누적 가속/감속

> **방향성 없음**: 매트릭스는 대칭적으로 읽는다. "A가 교만 topSin, B가 분노 topSin"이든 반대든 **교만↔분노 = 대립**으로 일괄 처리.

## 4. 연쇄 반응 규칙

폭주 이벤트의 **결과**가 다른 영웅의 **트리거**가 되는 구조.
폭주 행동은 3단계 판정(특성 우선 → sinStats 가중 확률 → 이력 고정)으로 결정된다.

| 폭주 유형 | 문제 행동 | 영향 |
|----------|----------|------|
| 분노 | 랜덤 영웅 도발 | 대상 영웅 wrath+1 |
| 탐욕 | 거점 자원 횡령 | food-2 or gold-5 |
| 교만 | 명령 거부 / 이의 이벤트 | 배치 불가(20%) |
| 시기 | 강한 영웅 죄종 수치 -1 | 타 영웅 약화 |
| 폭식 | 식량 추가 소비 | food 감소 |
| 나태 | 행동 효율 감소 + 전파 | 30% 확률, 주변 sloth+1 |
| 색욕 | 집착 대상 배치 거부 | 20% 확률 |

### 특성에 의한 연쇄 반응

| 특성 | 연쇄 반응에서의 역할 |
|------|-------------------|
| 예민한 | 연쇄 반응에 참여 |
| 둔감한 | 연쇄 반응 비참여 |
| 의리있는 | 동료 폭주 영향 면역 |
| 카리스마 | 본인 폭주 시 주변 영향 범위↑ |
| 시비꾼 | 대립 이벤트 촉발 확률↑ |
| 방화광 | 폭주 시 시설 방화 |
| 파괴충동 | 수치 18+ 시 시설 파손 |

## 5. 국시(國是, Edict) 시스템 — 2026-04-17 도입

### 설계 철학

1. **단일 레버** — 바알은 누적 수치 없음. 7죄종 중 **1개 국시**만 선포하여 통치 색깔 표현
2. **기간형** — 선포 시 `edict_duration` 턴(기본 5턴) 지속, 이후 자동 만료
3. **공백 리듬** — 만료 후 `edict_cooldown_normal` 턴(기본 1턴) 공백 → 국시 없는 평범한 턴 존재
4. **자유 선포** — 해금 조건 없음. 플레이어가 원할 때 7종 중 선택
5. **대립 전환 비용** — 대립 죄종 국시 간 전환 시 `edict_cooldown_opposite_bonus` 추가 공백
6. **조기 해제 가능** — 만료 전 해제 시 `edict_cooldown_early`(기본 2턴) 공백 페널티
7. **교만은 독립** — 대립 국시 없음
