## 1. 바알(플레이어)의 역할

바알은 **판결자이자 영주**다. 바알 자신의 유동 수치는 없고, 대신 **국시(國是, Edict)** 를 통해 통치 방향을 직접 결정한다:
- 7죄종 중 1개를 **국시로 선포** (기간형 5턴 지속)
- 선포한 국시의 효과가 거점 전반에 적용 (전투력/자원/생산성/영웅 수치 변동)
- 만료 후 공백 기간(1턴)을 거쳐 다음 국시 선포 가능
- 이벤트 선택지는 여전히 **영웅들의 죄종 수치**에 영향
- 상세 규칙: §5 국시(Edict) 시스템 참조

## 2. 죄종 수치 누적(쌓임) 시스템

> **사기(Morale) 시스템 완전 제거.** 죄종 수치(0~20)가 유일한 영웅 관리 레버.

### 서사 프레이밍
- 죄종 수치 = **"영웅에게 들어간 나쁜 마음이 쌓이는 정도"**
- 영웅은 0(깨끗한 상태)에서 시작하여 바알 밑에서 행동할수록 죄종이 쌓임
- 관리 = **"오염을 방지/정화"** — 영웅을 "행복하게" 하는 게임이 아니라 "덜 타락하게" 지키는 게임
- DD 스트레스 모델과 구조 동일: 쌓이는 것이 기본, 정화는 수단을 통해

### 구조
- **죄종 7수치** (각 **0~20**) — 영웅 생성 시 **모두 0**, 행동마다 **쌓이는(단방향 누적)** 상태 게이지
- 사기(0~100) 완전 폐기 — 이제 별도 관리 수치 없음
- 행동은 **주로 올림** — 쌓인 죄종은 **전용 정화 수단**(연회/야영/이벤트/시설/특성)으로만 감소
- **높은 수치 = 죄종별 파워 보너스 + 문제 행동 위험** → 플레이어 딜레마
- **낮은 수치 = 깨끗한 상태(좋음)** — "의욕 상실" 같은 저수치 페널티 없음
- 이탈: HP 0(전투 사망) 또는 폭주 지속(이탈 이벤트) 두 가지

### 메인 죄종 개념 폐기
- 영웅의 "메인 죄종"을 **생성 시 확정하지 않는다**
- `topSin()` = "현재 가장 많이 쌓인 죄종" 이라는 **상태** 표현일 뿐, 정체성이 아님
- 영웅 정체성은 **특성(trait)** 이 담당 (→ `hero_design.md` 참조)

### 수치 구간 (상세 수치 TBD)

| 수치 구간(안) | 상태 | 효과 |
|-------------|------|------|
| 0~? | **깨끗** | 영향 없음 (좋은 상태) |
| ?~? | 발현 | 해당 죄종 소폭 효과 |
| ?~17 | **고양** | 해당 죄종 파워 보너스 활성 + 문제 행동 시작 |
| **18~19** | **폭주 중** | 매 낮 페이즈 문제 행동 1회, `isRampaging=true` |
| **20 + N턴** | **이탈** | 이탈 이벤트 → 영웅 손실 |

> 구체 경계값(0~7/8~14/15~17 등)은 다음 세션에서 확정.

### 죄종별 쌓임 규칙 (상세 변동량 TBD)

|------|-----------|---------------|-------------|
| **분노** | 전투/사냥/방어 배치 | TBD (야영/이벤트/특성) | 랜덤 영웅 도발 → 그 영웅 wrath+1 |
| **나태** | idle 1턴, 발병 | TBD | 행동 효율 감소(확률), 주변 sloth+1 전파 |
| **시기** | 타 영웅이 원정/보상 시 bench 중 | TBD | 강한 영웅 죄종 수치 -1 (깎아내리기) |
| **폭식** | 연회, 식량 풍부+idle | TBD | 매 낮 식량 추가 소비 |

> "정화 수단 카탈로그"는 다음 세션에서 확정 — 연회/야영/시설/이벤트/특성 각각이 어떤 죄종을 얼마나 감소시키는지.

### 폭주 방식 (Darkest Dungeon 스트레스 방식)

- **18 진입**: `hero.isRampaging = true`, 폭주 이벤트 강제 발동
- **18~19 유지**: 매 낮 페이즈 해당 죄종 문제 행동 1회 (`processRampageTick`)
- **20 도달 + N턴 지속**: 이탈 이벤트 → 영웅 손실 (N=3 현재 안)
- **해제**: 수치가 17 이하로 내려가면 자동 폭주 해제
- **트리거 범위**: 어느 죄종이든 18+ 도달 시 (메인 죄종 개념 제거 방침과 정합) — 세부 확정 TBD

### 이탈
sinStats 가중 확률로 이탈 방식 결정 (폭력적 탈영, 사보타주, 동료 꼬셔서 함께 이탈 등)

## 3. 죄종 간 관계

### 매트릭스 (TheSevenTactics 계승)

```
         교만  시기  분노  나태  탐욕  폭식  색욕
교만      ──   생성  강화   대립  강화   ·    강화
시기     전제   ──   생성   ·    강화   ·     ·
분노     강화  강화   ──   대립   강화   ·     ·
나태     대립   ·   대립   ──    대립  강화   대립
탐욕     강화  강화  강화   대립   ──   대립   강화
폭식      ·    ·    ·    강화   대립   ──   생성
색욕     강화   ·    ·    대립   강화  생성    ──
```

### 관계가 게임에 미치는 영향

쌓임 프레임에 따라, 영웅 간 관계는 **현재 쌓인 죄종 수치**를 기준으로 비례 판정한다. 고정된 "주 성향 라벨"은 존재하지 않으며, `topSin()`은 "현재 상태"일 뿐이다.

| 관계 | 영웅↔영웅 배치 시 |
|------|----------------|
| 생성 | 강한 시너지 + 관련 죄종 추가 쌓임 |
| 강화 | 시너지 + 관련 죄종 소폭 쌓임 |
| 대립 | 갈등 이벤트 + 상호 죄종 쌓임 |
| 중립 | 특별한 상호작용 없음 |

> 쌓인 수치가 높은 죄종끼리의 관계가 지배적으로 작용. 모든 영웅이 깨끗한 초반에는 관계 효과가 약하고, 쌓일수록 관계가 활성화됨.

## 4. 연쇄 반응 규칙

폭주 이벤트의 **결과**가 다른 영웅의 **트리거**가 되는 구조.
폭주 행동은 3단계 판정(특성 우선 → sinStats 가중 확률 → 이력 고정)으로 결정된다.

| 폭주 유형 | 문제 행동 | 영향 |
|----------|----------|------|
| 분노 | 랜덤 영웅 도발 | 대상 영웅 wrath+1 |
| 탐욕 | 거점 자원 횡령 | food-2 or gold-5 |
| 교만 | 명령 거부 / 이의 이벤트 | 배치 불가(20%) |
| 시기 | 강한 영웅 죄종 수치 -1 | 타 영웅 약화 |
| 폭식 | 식량 추가 소비 | food 감소 |
| 나태 | 행동 효율 감소 + 전파 | 30% 확률, 주변 sloth+1 |
| 색욕 | 집착 대상 배치 거부 | 20% 확률 |

### 특성에 의한 연쇄 반응

| 특성 | 연쇄 반응에서의 역할 |
|------|-------------------|
| 예민한 | 연쇄 반응에 참여 |
| 둔감한 | 연쇄 반응 비참여 |
| 의리있는 | 동료 폭주 영향 면역 |
| 카리스마 | 본인 폭주 시 주변 영향 범위↑ |
| 시비꾼 | 대립 이벤트 촉발 확률↑ |
| 방화광 | 폭주 시 시설 방화 |
| 파괴충동 | 수치 18+ 시 시설 파손 |

## 5. 국시(國是, Edict) 시스템 — 2026-04-17 도입

### 설계 철학

1. **단일 레버** — 바알은 누적 수치 없음. 7죄종 중 **1개 국시**만 선포하여 통치 색깔 표현
2. **기간형** — 선포 시 `edict_duration` 턴(기본 5턴) 지속, 이후 자동 만료
3. **공백 리듬** — 만료 후 `edict_cooldown_normal` 턴(기본 1턴) 공백 → 국시 없는 평범한 턴 존재
4. **자유 선포** — 해금 조건 없음. 플레이어가 원할 때 7종 중 선택
5. **대립 전환 비용** — 대립 죄종 국시 간 전환 시 `edict_cooldown_opposite_bonus` 추가 공백
6. **조기 해제 가능** — 만료 전 해제 시 `edict_cooldown_early`(기본 2턴) 공백 페널티
7. **교만은 독립** — 대립 국시 없음

### 죄종 × 정책 영역 매핑

| 죄종 | 국시명 | 정책 영역 | 대립 국시 |
|------|-------|----------|---------|
| 분노 | 강병령 | 군사 — 전쟁, 처벌 | 안식령 |
| 탐욕 | 축재령 | 경제 — 축적, 착취 | 풍요령 |
| 교만 | 위광령 | 위상 — 독선, 엘리트 | (없음) |
| 시기 | 경쟁령 | 경쟁 — 차등, 감시 | 열정령 |
| 나태 | 안식령 | 노동 — 방치, 안식 | 강병령 |
| 폭식 | 풍요령 | 복지 — 소비, 풍요 | 축재령 |
| 색욕 | 열정령 | 선동 — 관계, 열정 | 경쟁령 |

### 국시별 효과 (초안 — balance는 edicts.csv에서 관리)

| 국시 | 주요 효과 | 설계 의도 |
|------|---------|---------|
| **강병령** (분노) | 전투력 +15%, 매턴 영웅 분노 누적 가속 | 전쟁 강화 + 분노 오염 리스크 |
| **축재령** (탐욕) | 자원 획득 +20%, 매턴 영웅 탐욕 누적 가속 | 자원 경제 + 탐욕 오염 |
| **풍요령** (폭식) | 식량 소모 +30%, 특정 죄종 정화 효과 | 비용 증가 + 오염 완화 |
| **안식령** (나태) | 건설/연구 +20%, 전투력 -10%, 나태 누적 가속 | 생산 강화 + 나태 오염 |
| **경쟁령** (시기) | 영웅 경험치 +25%, 시기 누적 가속 | 성장 가속 + 시기 오염 |
| **열정령** (색욕) | 색욕 누적 가속, 관계 효과 증폭 | 드라마 촉진 |
| **위광령** (교만) | Lv5+ 영웅 능력 +10%, 교만 누적 가속 | 엘리트 강화 + 교만 오염 |

> 효과 수치/키는 `src/data/edicts.csv`에서 로드. 국시별 구체 "누적 가속량"과 "정화 효과"는 정화 수단 카탈로그 확정 후 일괄 조정 예정.

### 상태 전이 규칙

```
무(無) 국시 → [선포] → 선포 중(N턴) → [자동 만료] → 쿨다운(1턴) → 무(無) 국시
                            │
                            └─ [조기 해제] → 쿨다운(2턴) → 무(無) 국시

대립 국시 전환 시: 쿨다운 + 1턴 추가
```

### 데이터 구조

```
Store.edict:
  null                                                    — 무 국시, 쿨다운 없음
  { type: 'wrath', activatedDay: 10, expireDay: 15,
    cooldownUntilDay: null }                              — 선포 중
  { type: null, activatedDay: null, expireDay: null,
    cooldownUntilDay: 17 }                                — 쿨다운 중
```
