# 죄종 시스템 상세 설계

> 상태: 확정 (죄종 7수치 + 주 성향 고정 + 특성 예외 + 바알 업보)
> 작성일: 2026-03-24

---

## 1. 바알(플레이어)의 역할

바알은 죄종 수치를 가지지 않는다. 바알의 역할은 **판결자**:
- 이벤트에 대해 선택지를 고르고, 그 결과가 영웅들의 사기에 반영됨
- 어떤 선택이든 트레이드오프 — 누군가를 만족시키면 다른 누군가가 불만

---

## 2. 사기 시스템

### 구조
- 영웅당 수치 **1개**: 사기 (0~100)
- **죄종 7수치** (각 1~20) — 성격 차원. 만족/불만 강도는 수치에 비례
- **주 성향 = 최고 죄종 수치** — 폭주/이탈 행동을 고정 결정
- 양쪽 극단이 모두 위험: 0 = 이탈, 100 = 폭주

| 구간 | 상태 | 효과 |
|------|------|------|
| 0 | **이탈** | 거점을 떠남 (주 성향에 따라 방식 고정) |
| 1~25 | 불만 | 능력 저하 + **매 턴 이탈 확률 체크** |
| 26~75 | 안정 | 정상 운용 |
| 76~99 | 고양 | 능력 강화 + **구원(Inspiration) 확률** + **매 턴 폭주 확률 체크** |
| 100 | **폭주** | 통제 불능 (주 성향에 따라 행동 고정) |

> 설계 의도: 사기는 **높을수록 좋지만, 너무 높으면 위험**. 고양 구간은 강하지만 폭주 리스크가 있어서 "사기를 올릴까 말까" 고민이 생긴다. 양극단이 모두 나쁜 구조(=안정 유지 게임)를 해소.

### 사기 변동 요인

| 요인 | 예시 | 사기 변동 |
|------|------|----------|
| 죄종 조건 만족 | 분노 높은 영웅을 원정 파견 | +변동량 (해당 죄종 수치에 비례) |
| 죄종 조건 불만 | 분노 높은 영웅 3일 미파견 | -변동량 (해당 죄종 수치에 비례) |
| 복수 죄종 동시 | 여러 죄종 조건이 동시 충족/미충족 | 각 수치에 비례하여 합산 |
| 유리한 판결 | 해당 성향을 편드는 판결 | +10 |
| 불리한 판결 | 해당 성향에 불리한 판결 | -10 |
| 영웅 간 대립 | 죄종 수치 대립 관계 영웅과 배치 | -10 |
| 특성 보정 | 특성에 따라 사기 변동폭 변조 | 특성별 상이 |
| 방어전 승리 | 밤 습격 방어 성공 | 전체 +5 |
| 방어전 패배 / 동료 사망 | 습격 실패 or 사망 발생 | 전체 -10 |

### 폭주/이탈 행동 결정 — 주 성향 고정 + 특성 예외

사기가 극단(0 또는 100)에 도달하면, 다음 규칙으로 행동을 결정한다:

1. **후천 폭주 특성 우선** — 폭주 반복으로 후천 폭주 특성("파괴자", "약탈자" 등)을 획득한 영웅은 해당 강박 행동이 **우선 발동**
2. **주 성향 고정** — 후천 폭주 특성이 없으면, **최고 죄종 수치(주 성향)**에 따라 행동이 **고정 결정**

> 설계 의도: 플레이어가 영웅 프로필을 보고 "이 영웅이 폭주하면 뭘 할지" 예측할 수 있어야 관리가 재미있다. 확률 판정은 전략적 대비를 불가능하게 만들어 경영 시뮬의 핵심 재미를 해친다.

### 폭주 (사기 100)

주 성향에 따라 폭주 행동이 결정된 뒤:
- 70% 확률 → **타락** (영구 강화 + 영구 리스크, 후천 특성 "타락자" 획득)
- 30% 확률 → **구원** (안정화 + 고유 능력, 후천 특성 "개심자" 획득)

| 주 성향 | 폭주 행동 |
|---------|----------|
| 분노 | 가장 가까운 건물을 **부숨** |
| 교만 | 명령을 **전부 거부**, 독자 행동 |
| 탐욕 | 창고에서 자원을 **훔쳐서 도주** 시도 |
| 시기 | 가장 강한 영웅의 장비를 **파괴** |
| 폭식 | 식량 저장소를 **비움** |
| 나태 | **아무것도 안 함** + 주변 영웅 사기 저하 |
| 색욕 | 영웅 2명을 **분쟁에 휘말리게** 함 |

### 이탈 (사기 0)

주 성향에 따라 이탈 방식이 고정 결정:

| 주 성향 | 이탈 방식 |
|---------|---------|
| 분노 | 폭력적 탈영 |
| 교만 | "너 밑에 있을 수준이 아니다" 선언 후 이탈 |
| 탐욕 | 창고 털고 도주 |
| 시기 | 뒤에서 사보타주 후 이탈 |
| 폭식 | 보급품 갖고 이탈 |
| 나태 | 조용히 사라짐 |
| 색욕 | 다른 영웅 꼬여서 같이 떠남 |

### 고유 특성과의 상호작용
- 고유 특성은 **장점 전용** — 사기/폭주에 직접 영향하지 않음
- 단, 일부 고유 특성이 간접적으로 사기 안정에 기여:
  - "낙천적": 사기 하락폭 -30% → 이탈 구간 진입 느림
  - "철의 의지": 폭주/이탈 임계값 5씩 완화
  - "카리스마": 파티 사기↑ → 동료 안정화
- 후천 폭주 특성("파괴자", "약탈자" 등): 폭주 2회 반복 시 획득 → 이후 고정 강박 행동

## 3. 죄종 간 관계

### 매트릭스 (TheSevenTactics 계승)

```
         교만  시기  분노  나태  탐욕  폭식  색욕
교만      ──   생성  강화   대립  강화   ·    강화
시기     전제   ──   생성   ·    강화   ·     ·
분노     강화  강화   ──   대립   강화   ·     ·
나태     대립   ·   대립   ──    대립  강화   대립
탐욕     강화  강화  강화   대립   ──   대립   강화
폭식      ·    ·    ·    강화   대립   ──   생성
색욕     강화   ·    ·    대립   강화  생성    ──
```

### 관계가 게임에 미치는 영향

영웅 간 관계는 **주 성향(최고 죄종 수치)**을 기준으로 판정한다. 부 성향(2위 이하)도 약한 강도로 영향.

| 관계 | 영웅↔영웅 배치 시 |
|------|----------------|
| 생성 | 강한 시너지 + 사기 상승 |
| 강화 | 시너지 + 사기 소폭 상승 |
| 대립 | 갈등 이벤트 + 사기 하락 |
| 중립 | 특별한 상호작용 없음 |

> 주 성향끼리의 관계가 지배적으로 작용. 부 성향은 보조적 영향.

## 4. 연쇄 반응 규칙

폭주 이벤트의 **결과**가 다른 영웅의 **트리거**가 되는 구조.
폭주 행동은 주 성향에 따라 고정 결정된다 (행동 강박 특성이 있으면 특성 우선).

| 폭주 유형 | 기본 행동 | 영향받는 영웅 | 사기 변동 |
|----------|----------|-------------|----------|
| 분노 | 건물 파괴 | 전체 | 사기 -10 |
| 탐욕 | 자원 도난 | 폭식 주 성향 영웅 (자원 부족) | 사기 -20 |
| 교만 | 명령 거부 | 분노 주 성향 영웅 (화남) | 사기 +15 (폭주 방향) |
| 시기 | 장비 파괴 | 탐욕 주 성향 영웅 (손실) | 사기 -15 |
| 폭식 | 식량 소진 | 전체 | 사기 -10 |
| 나태 | 행동 거부 | 전체 (전염) | 사기 -10 |
| 색욕 | 분쟁 유발 | 관련 영웅 2명 | 사기 +20 (폭주 방향) |

### 특성에 의한 연쇄 반응 증폭

| 특성 | 연쇄 반응에서의 역할 |
|------|-------------------|
| 예민한 | 연쇄 반응에 참여 (사기 변동 받음) |
| 둔감한 | 연쇄 반응 비참여 (사기 변동 면역) |
| 의리있는 | 동료 폭주 시 사기하락 면역 |
| 카리스마 | 본인 폭주 시 주변 영향 범위↑ |
| 시비꾼 | 대립 이벤트 촉발 확률↑ |
| 중재자 | 대립 이벤트 완화 (본인 사기 소모) |
| 방화광 | 폭주 시 시설 방화 (추가 피해) |
| 파괴충동 | 불만 구간에서 시설 파손 (폭주 전에도 발동) |

## 5. 바알 업보 (거점 죄종 수치)

> 상태: 초안 확정 (2026-04-14)

### 개요

바알은 개인 죄종 수치를 갖지 않지만, **경영 판단의 결과**가 거점(림보)에 7죄종 업보로 누적된다.
영웅 개인의 죄종 = 태어난 성격(고정). 바알의 업보 = 판결로 쌓인 결과(누적). 두 시스템은 별개 레이어.

### 수치 구조

- **범위**: 0~100
- **시작값**: 50 (중립)
- **임계점**: 70 이상 = 高 효과 발동, 30 이하 = 低 효과 발동
- **31~69**: 중립 (효과 없음, 심상만 서서히 변화)

### 입력 (어떻게 쌓이는가)

다양한 플레이 행동이 업보에 영향:

| 입력 | 설명 |
|------|------|
| 판결 (이벤트 선택) | 선택지에 죄종 태그 → 해당 업보 증감 |
| 거점 운영 방식 | 정책/시설 운영 방식에 따라 특정 업보 변동 |
| 전투 빈도 | 전투를 많이 할수록 분노 업보 상승 등 |
| 로스터 구성 | 영웅들의 주 성향 분포가 거점 분위기에 영향 |

> 설계 의도: 플레이어가 의식적으로 업보를 "관리"하는 게 아니라, 자연스러운 플레이 방식의 결과로 업보가 쌓인다.

### 피드백 (어떻게 보여주는가)

수치를 전면에 내세우지 않고 **심상(Ambience)**으로 체감:

| 피드백 계층 | 방식 |
|------------|------|
| **1차: 심상** | 1위 죄종의 화면 색조, 배경음악, 영웅 대사 변화 |
| **2차: 확인용** | 메뉴 안에서 수치 열람 가능 (궁금한 플레이어를 위해) |
| **3차: 정산 힌트** | 챕터 정산 시 "림보에 ○○의 기운이 짙다" 텍스트 |

> 원칙: "스프레드시트가 아니라 드라마" — 수치를 보고 최적화하는 게 아니라, 분위기로 느끼는 게임

### 거점 정책 효과 (각 죄종 = 정책 영역 하나)

각 죄종은 거점의 **정책 영역**을 대변. 임계점(70+/30-)을 넘으면 효과 발동.
**영웅 개인 행동(폭주/이탈/명령 거부)은 영웅 죄종 시스템에서 처리** — 바알 업보는 거점 환경/시설/자원 레벨에만 영향.

#### 분노 — 군사 정책
```
低 평화 국가 ◄────────► 高 군사 국가
```
| 低 (30-) | 高 (70+) |
|----------|----------|
| 주변 세력 우호적 → 습격 빈도 감소 | 영웅 전투 보정 증가 |
| 교역 이벤트 발생 확률 증가 | 주변 세력 적대적 → 습격 빈도 증가 |
| 영웅 전투 보정 감소 | 교역 차단 |

#### 탐욕 — 경제 정책
```
低 검소 국가 ◄────────► 高 착취 국가
```
| 低 (30-) | 高 (70+) |
|----------|----------|
| 건설/유지비 감소 | 토벌/채집 보상 증가 |
| 자원 수입 감소 | 매 턴 고정 지출 증가 (사치 비용) |

#### 교만 — 위상 정책
```
低 협력 국가 ◄────────► 高 패권 국가
```
| 低 (30-) | 高 (70+) |
|----------|----------|
| 동맹/협력 이벤트 발생 | 거점 위상 상승 → 강한 영웅이 찾아옴 |
| 초청 영웅 등급 낮음 | 동맹 거부, 외교 단절 |
| 기존 시설 업그레이드 효율 증가 | 고급 시설/연구 해금 가속, 기본 시설 소홀 |

#### 시기 — 경쟁 정책
```
低 협동 문화 ◄────────► 高 경쟁 문화
```
| 低 (30-) | 高 (70+) |
|----------|----------|
| 방어전 파티 시너지 증가 (팀워크) | 영웅 개인 성장 속도 증가 (경쟁 동기부여) |
| 개인 영웅 성장 느림 | 방어전 파티 시너지 감소 (협력 약화) |

#### 나태 — 노동 정책
```
低 근면 국가 ◄────────► 高 복지 국가
```
| 低 (30-) | 高 (70+) |
|----------|----------|
| 건설/연구 속도 증가 | 영웅 회복 속도 증가 |
| 영웅 컨디션 회복 느림 (과로) | 건설/연구 속도 감소 |

#### 폭식 — 복지 정책
```
低 긴축 ◄────────► 高 복지 국가
```
| 低 (30-) | 高 (70+) |
|----------|----------|
| 턴당 자원 소모 최소 | 영웅 사기 기본값 상승, 질병 저항 |
| 영웅 컨디션 저하, 질병 확률 증가 | 식량+골드 소모 대폭 증가 |

#### 색욕 — 선동/열정 정책
```
低 이성적 통치 ◄────────► 高 선동 국가
```
| 低 (30-) | 高 (70+) |
|----------|----------|
| 사기 변동폭 축소 (안정적) | 사기 변동폭 증가 (극적 상황 빈발) |
| 이벤트 빈도 감소, 잔잔한 운영 | 이벤트 빈도 증가, 드라마 많음 |

> 高색욕 = 게임이 극적으로 / 低색욕 = 게임이 조용하게 — 플레이어가 드라마 강도를 간접 조절

### 업보와 엔딩

게임 종료 시 가장 높은 업보 죄종이 엔딩을 결정 (7개 엔딩). 상세는 엔딩 시스템 참조.
