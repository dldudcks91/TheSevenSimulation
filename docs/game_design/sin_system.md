# 죄종 시스템 상세 설계

> 상태: 컨셉 확정, 수치 미확정
> 작성일: 2026-03-24

---

## 1. 바알(플레이어)의 역할

바알은 죄종 수치를 가지지 않는다. 바알의 역할은 **판결자**:
- 이벤트에 대해 선택지를 고르고, 그 결과가 영웅들의 사기에 반영됨
- 어떤 선택이든 트레이드오프 — 누군가를 만족시키면 다른 누군가가 불만

---

## 2. 사기 시스템

### 구조
- 영웅당 수치 **1개**: 사기 (0~100)
- 죄종은 고정 성격 태그 — 사기가 극단에 갔을 때 **어떤 행동을 하는지** 결정
- 양쪽 극단이 모두 위험: 0 = 이탈, 100 = 폭주

| 구간 | 상태 | 효과 |
|------|------|------|
| 0 | **이탈** | 거점을 떠남 (죄종에 따라 방식 다름) |
| 1~30 | 불만 | 능력 저하, 명령 거부 위험 |
| 31~70 | 안정 | 정상 운용 |
| 71~99 | 고양 | 능력 강화, 충성적 |
| 100 | **폭주** | 통제 불능 (죄종에 따라 사고 다름) |

### 사기 변동 요인

| 요인 | 예시 | 사기 변동 |
|------|------|----------|
| 죄종 만족 | 분노 영웅을 원정 파견 | +10 |
| 죄종 불만 | 분노 영웅 3일 미파견 | -15 |
| 유리한 판결 | [분노]를 편드는 판결 | +10 |
| 불리한 판결 | [분노]에게 불리한 판결 | -10 |
| 영웅 간 대립 | 대립 죄종과 같은 시설 배치 | -10 |
| 방어전 승리 | 밤 습격 방어 성공 | 전체 +5 |
| 방어전 패배 / 동료 사망 | 습격 실패 or 사망 발생 | 전체 -15 |

### 폭주 (사기 100)
70% 확률 → **타락** (영구 강화 + 영구 리스크)
30% 확률 → **구원** (안정화 + 고유 능력)

### 이탈 (사기 0)
죄종에 따라 이탈 방식이 다름 (폭력적 탈영, 사보타주, 동료 꼬셔서 함께 이탈 등)

## 3. 죄종 간 관계

### 매트릭스 (TheSevenTactics 계승)

```
         교만  시기  분노  나태  탐욕  폭식  색욕
교만      ──   생성  강화   대립  강화   ·    강화
시기     전제   ──   생성   ·    강화   ·     ·
분노     강화  강화   ──   대립   강화   ·     ·
나태     대립   ·   대립   ──    대립  강화   대립
탐욕     강화  강화  강화   대립   ──   대립   강화
폭식      ·    ·    ·    강화   대립   ──   생성
색욕     강화   ·    ·    대립   강화  생성    ──
```

### 관계가 게임에 미치는 영향

| 관계 | 영웅↔영웅 배치 시 |
|------|----------------|
| 생성 | 강한 시너지 + 사기 상승 |
| 강화 | 시너지 + 사기 소폭 상승 |
| 대립 | 갈등 이벤트 + 사기 하락 |
| 중립 | 특별한 상호작용 없음 | 무반응 |

---

## 4. 연쇄 반응 규칙

폭주 이벤트의 **결과**가 다른 영웅의 **트리거**가 되는 구조:

| 폭주 결과 | 영향받는 영웅 | 사기 변동 |
|---------|------------|----------|
| 분노: 건물 파괴 | 전체 | 사기 -10 |
| 탐욕: 자원 도난 | 폭식 (자원 부족) | 사기 -20 |
| 교만: 명령 거부 | 분노 (화남) | 사기 +15 (폭주 방향) |
| 시기: 장비 파괴 | 탐욕 (손실) | 사기 -15 |
| 폭식: 식량 소진 | 전체 | 사기 -10 |
| 나태: 행동 거부 | 전체 (전염) | 사기 -10 |
| 색욕: 분쟁 유발 | 관련 영웅 2명 | 사기 +20 (폭주 방향) |
