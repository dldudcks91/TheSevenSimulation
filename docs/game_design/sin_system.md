# 죄종 시스템 상세 설계

> 상태: 재설계 완료 (죄종 태그 → 죄종 7수치 → primarySin 라벨 제거, 특성 우선/확률 판정)
> 작성일: 2026-03-24

---

## 1. 바알(플레이어)의 역할

바알은 영웅과 마찬가지로 **7개 죄종 수치**(각 1~100, 시작 50)를 가진다. 바알의 역할은 **판결자**:
- 이벤트에 대해 선택지를 고르고, 그 결과가 영웅들의 사기 + 바알의 죄종 수치에 반영됨
- 바알의 죄종 수치는 영웅들과 함께 움직인다 — 같은 이벤트, 같은 판결이 동시에 변동
- 어떤 선택이든 트레이드오프 — 누군가를 만족시키면 다른 누군가가 불만
- 임계점(70+/30-)에서 거점 환경 효과 발동, 가장 높은 죄종이 엔딩 결정
- 상세 변동 규칙: §5 바알 죄종 수치 변동 시스템 참조

## 2. 사기 시스템

### 구조
- 영웅당 수치 **1개**: 사기 (0~100)
- **죄종 7수치** (각 1~20) — 성격 차원. 7수치가 확률/비례로 행동을 결정
- **죄종 수치는 영웅 생성 시 고정** — 게임 진행 중 어떤 이벤트/특성/폭주로도 변동하지 않는다. 성격은 운명이다.
- **주 성향 라벨(primarySin) 없음** — 고정 라벨 대신 수치 비례 확률 판정
- 양쪽 극단이 모두 위험: 0 = 이탈, 100 = 폭주

| 구간 | 상태 | 효과 |
|------|------|------|
| 0 | **이탈** | 거점을 떠남 (특성 우선 → sinStats 가중 확률로 방식 결정) |
| 1~30 | 불만 | 능력 저하, 명령 거부 위험 |
| 31~70 | 안정 | 정상 운용 |
| 71~99 | 고양 | 능력 강화, 충성적 |
| 100 | **폭주** | 통제 불능 (특성 우선 → sinStats 가중 확률로 행동 결정) |

### 사기 변동 요인

| 요인 | 예시 | 사기 변동 |
|------|------|----------|
| 죄종 조건 만족 | 분노 높은 영웅을 원정 파견 | +변동량 (해당 죄종 수치에 비례) |
| 죄종 조건 불만 | 분노 높은 영웅 3일 미파견 | -변동량 (해당 죄종 수치에 비례) |
| 복수 죄종 동시 | 여러 죄종 조건이 동시 충족/미충족 | 각 수치에 비례하여 합산 |
| 유리한 판결 | 해당 성향을 편드는 판결 | +10 |
| 불리한 판결 | 해당 성향에 불리한 판결 | -10 |
| 영웅 간 대립 | 죄종 수치 대립 관계 영웅과 배치 | -10 |
| 특성 보정 | 특성에 따라 사기 변동폭 변조 | 특성별 상이 |
| 방어전 승리 | 밤 습격 방어 성공 | 전체 +5 |
| 방어전 패배 / 동료 사망 | 습격 실패 or 사망 발생 | 전체 -15 |
| **Stamina 탈진** | Stamina 0 | -10 즉시 (`morale_penalty_exhausted`) |
| **Stamina 과로 지속** | Stamina 1~25 | -2 /턴 (`morale_penalty_overwork_tick`) |
| **발병** | 과로 상태에서 발병 트리거 | -8 (`sickness_morale_delta`) |

### 폭주/이탈 행동 결정 — 3단계 판정

사기가 극단(0 또는 100)에 도달하면, 다음 3단계로 구체적 행동을 결정한다:

1. **특성 우선** — 행동 강박 특성("방화광", "파괴충동", "도벽" 등)이 있으면 해당 특성의 강박 행동이 **우선 발동**
2. **sinStats 가중 확률** — 특성으로 결정되지 않으면, 7개 죄종 수치를 가중치로 한 **확률 판정**으로 행동 결정 (예: 분노 15, 탐욕 8이면 분노형 폭주가 나올 확률이 약 2배)
3. **이력 누적 → 레벨업 시 폭주 특성 우선** — 같은 유형의 폭주가 2회 반복되면, 다음 레벨업(Lv5 or Lv10)에서 행동 특성 대신 폭주 특성이 부여됨. 이후 해당 유형이 **고정 발동**

### 폭주 (사기 100)
70% 확률 → **타락** (영구 강화 + 영구 리스크, 후천 특성 "타락자" 획득)
30% 확률 → **구원** (안정화 + 고유 능력, 후천 특성 "개심자" 획득)

### 이탈 (사기 0)
sinStats 가중 확률로 이탈 방식이 결정됨 (폭력적 탈영, 사보타주, 동료 꼬셔서 함께 이탈 등)

### 특성과의 상호작용
- "둔감한" 특성: 폭주/이탈 임계값 넓음 → 안정적, 사기 변동에 잘 반응하지 않음
- "예민한" 특성: 임계값 좁음 → 사기 변동 민감, 관리 어려움
- "변덕쟁이" 특성: 사기 변동폭 ×1.5 → 빠르게 폭주/이탈에 도달
- "철의 의지" 특성: 임계값 5씩 완화
- 행동 강박 특성("방화광", "파괴충동"): 폭주/불만 시 고유 강박 행동 추가 발동

## 3. 죄종 간 관계

### 매트릭스 (TheSevenTactics 계승)

```
         교만  시기  분노  나태  탐욕  폭식  색욕
교만      ──   생성  강화   대립  강화   ·    강화
시기     전제   ──   생성   ·    강화   ·     ·
분노     강화  강화   ──   대립   강화   ·     ·
나태     대립   ·   대립   ──    대립  강화   대립
탐욕     강화  강화  강화   대립   ──   대립   강화
폭식      ·    ·    ·    강화   대립   ──   생성
색욕     강화   ·    ·    대립   강화  생성    ──
```

### 관계가 게임에 미치는 영향

죄종 수치화에 따라, 영웅 간 관계는 **각 죄종 수치**를 기준으로 비례 판정한다 (고정 주 성향 라벨 없음).

| 관계 | 영웅↔영웅 배치 시 |
|------|----------------|
| 생성 | 강한 시너지 + 사기 상승 |
| 강화 | 시너지 + 사기 소폭 상승 |
| 대립 | 갈등 이벤트 + 사기 하락 |
| 중립 | 특별한 상호작용 없음 |

> 모든 죄종 수치가 비례 강도로 관계에 영향. 높은 수치끼리의 관계가 지배적으로 작용.

## 4. 연쇄 반응 규칙

폭주 이벤트의 **결과**가 다른 영웅의 **트리거**가 되는 구조.
폭주 행동은 3단계 판정(특성 우선 → sinStats 가중 확률 → 이력 고정)으로 결정된다.

| 폭주 유형 | 기본 행동 | 영향받는 영웅 (해당 죄종 수치 최고) | 사기 변동 |
|----------|----------|-------------------------------|----------|
| 분노 | 건물 파괴 | 전체 | 사기 -10 |
| 탐욕 | 자원 도난 | 폭식 수치 최고 영웅 (자원 부족) | 사기 -20 |
| 교만 | 명령 거부 | 분노 수치 최고 영웅 (화남) | 사기 +15 (폭주 방향) |
| 시기 | 장비 파괴 | 탐욕 수치 최고 영웅 (손실) | 사기 -15 |
| 폭식 | 식량 소진 | 전체 | 사기 -10 |
| 나태 | 행동 거부 | 전체 (전염) | 사기 -10 |
| 색욕 | 분쟁 유발 | 관련 영웅 2명 | 사기 +20 (폭주 방향) |

### 특성에 의한 연쇄 반응 증폭

| 특성 | 연쇄 반응에서의 역할 |
|------|-------------------|
| 예민한 | 연쇄 반응에 참여 (사기 변동 받음) |
| 둔감한 | 연쇄 반응 비참여 (사기 변동 면역) |
| 의리있는 | 동료 폭주 시 사기하락 면역 |
| 카리스마 | 본인 폭주 시 주변 영향 범위↑ |
| 시비꾼 | 대립 이벤트 촉발 확률↑ |
| 중재자 | 대립 이벤트 완화 (본인 사기 소모) |
| 방화광 | 폭주 시 시설 방화 (추가 피해) |
| 파괴충동 | 불만 구간에서 시설 파손 (폭주 전에도 발동) |

## 5. 바알 죄종 수치 변동 시스템

### 설계 철학

1. **양쪽 다 의미 있음** — 70+도 30-도 각각 버프+리스크. 50은 무색무취
2. **행동 누적 → 세계 반응** — 통치 스타일이 거점 환경, NPC, 영웅에 반영
3. **수치 반공개** — 심상(텍스트+톤)으로 체감, 챕터 정산 또는 특정 시설에서 수치 확인 가능
4. **1:1 비중** — 올리는 양 = 내리는 양 (다중 태깅에서 복잡성 발생)
5. **고정 대립 쌍 없음** — 행동별 개별 태깅
6. **교만은 독립적** — 자기 자신에 대한 죄, 별도 움직임
7. **포고령 재설계** — 바알 죄종 수치와 연동. 포고령 = 단기 즉시 정책, 죄종 = 장기 누적 성향. 포고령 선택이 바알 죄종도 변동시키고, 죄종 임계 시 새 포고령 해금

### 정책 영역 매핑

| 죄종 | 정책 영역 | 의미 |
|------|----------|------|
| 분노 | 군사 | 전쟁, 처벌, 공격성 |
| 탐욕 | 경제 | 축적, 착취, 자원 독점 |
| 교만 | 위상 | 자기확신, 독선, 엘리트주의 |
| 시기 | 경쟁 | 차등, 감시, 경쟁 부추기기 |
| 나태 | 노동 | 방치, 무기력, 안식 |
| 폭식 | 복지 | 소비, 풍요, 낭비 |
| 색욕 | 선동 | 감정, 관계, 열정 |

### 행동 범주

플레이어 행동을 6개 범주로 분류. 각 행동에 죄종 태그를 부여하여 변동을 추적한다.

| 범주 | 시점 | 성격 | 빈도 |
|------|------|------|------|
| 판결 | 아침 | 이벤트 선택지 응답 | 매턴 1~2회 |
| 작업 지시 | 낮 | 영웅에게 작업 배정 | 매턴 다수 |
| 편성 | 낮 | 누구를 어디에 | 매턴 (미정) |
| 소비 | 낮 | 자원 사용 결정 | 비정기 |
| 외교 | 이벤트 | 외부 세력 대응 | 드물게 |
| 전투 판단 | 저녁/밤 | 전투 중 결정 | 비정기 |

### 범주별 변동 테이블

#### 판결 (아침 — 이벤트 선택지)

이벤트 선택지마다 개별 태깅. 선택지의 **톤**에 따른 패턴:

| 판결 톤 | 올라가는 죄종 | 내려가는 죄종 |
|---------|-------------|-------------|
| 처벌/강경 | 분노 +2 | 색욕 -2 |
| 용서/화해 | 색욕 +2 | 분노 -2 |
| 독점/거절 | 탐욕 +2 | 폭식 -2 |
| 분배/포상 | 폭식 +2 | 탐욕 -2 |
| 무시/방치 | 나태 +2 | 교만 -2 |
| 독단 결정 | 교만 +2 | 시기 -2 |
| 의견 수렴 | 시기 +2 | 교만 -2 |
| 경쟁 부추기기 | 시기 +2 | 색욕 -2 |
| 감정적 선택 | 색욕 +2 | 나태 -2 |

> 복합 선택지는 여러 톤이 겹침. 예: "도둑을 처벌하고 훔친 자원 회수" → 분노 +2, 색욕 -2 (처벌) + 탐욕 +2, 폭식 -2 (독점)

#### 작업 지시 (낮 — 영웅을 뭘 시킬지)

매턴 반복되므로 변동량 작게 (±1~2).

| 작업 | 변동 | 논리 |
|------|------|------|
| 원정 파견 | 분노 +2, 나태 -2 | 전쟁 = 군사↑ 무기력↓ |
| 사냥 | 분노 +1, 나태 -1 | 소규모 전투 |
| 채집 | 탐욕 +1, 폭식 -1 | 자원 모으기 = 축적 |
| 벌목 | 탐욕 +1, 나태 -1 | 노동 지시 |
| 건설 | 교만 +1, 나태 -1 | 거점 키우기 = 위상 |
| 연구 | 교만 +1, 나태 -1 | 발전 추구 |
| 개척 | 탐욕 +1, 분노 +1, 나태 -2 | 영토 확장 |
| 휴식 지시 | 나태 +2, 분노 -2 | 쉬게 함 |
| 치료 | 폭식 +1, 탐욕 -1 | 복지 = 자원 소비 |
| 외교 | 색욕 +1, 분노 -1 | 관계 = 선동↑ 군사↓ |

#### 편성 (낮 — 누구를 어디에)

> 미정. 매턴 판정 vs 패턴 판정(3~5턴 누적) 방식 미확정.

#### 소비 (낮 — 자원 사용)

| 소비 행동 | 변동 | 논리 |
|----------|------|------|
| 연회 개최 | 폭식 +2, 탐욕 -2 | 자원 소비 = 복지↑ 축적↓ |
| 장비 구매 | 탐욕 -1, 시기 +1 | 자원 씀 + 특정 영웅 투자 |
| 영웅 영입 | 색욕 +1, 탐욕 -1 | 사람 모음 + 자원 소비 |
| 교역 (이익 추구) | 탐욕 +2, 폭식 -2 | 이익 = 축적↑ 복지↓ |
| 교역 (필요물자 구매) | 폭식 +1, 탐욕 -1 | 소비 = 복지↑ 축적↓ |
| 자원 비축 (안 씀) | 탐욕 +1, 폭식 -1 | 매턴 자동, 잉여 자원 시 |

#### 외교 (이벤트 — 외부 세력 대응)

빈도 낮으므로 다중 태깅 허용 (임팩트 큼).

| 외교 행동 | 변동 | 논리 |
|----------|------|------|
| 선제 공격 | 분노 +2, 교만 +2, 색욕 -2, 나태 -2 | 공격적 + 자신감 |
| 외교적 타협 | 색욕 +2, 분노 -2 | 관계 중시 |
| 겸손한 외교 | 교만 -2, 시기 +2 | 몸 낮춤 = 비교심 자극 |
| 세력 초청 참석 | 색욕 +1, 교만 +1, 나태 -1, 탐욕 -1 | 관계 + 과시 |
| 초청 거절 | 교만 +1, 색욕 -1 | 독단 |
| 의뢰 수행 | 탐욕 +1, 나태 -1 | 보상 목적 노동 |
| 무시/방치 | 나태 +2, 교만 -2 | 무기력 + 위상 하락 |

#### 전투 판단 (저녁/밤 — 전투 중 결정)

| 전투 판단 | 변동 | 논리 |
|----------|------|------|
| 원정 계속 진행 | 분노 +1, 나태 -1 | 밀어붙이기 |
| 원정 철수 | 나태 +1, 분노 -1 | 물러남 |
