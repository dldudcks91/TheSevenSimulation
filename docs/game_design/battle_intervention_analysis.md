# 전투 중 유저 개입 방식 — 비교 분석 및 논의

> 작성일: 2026-03-29
> 목적: 현재 카드 시스템 냉정 평가 + 레퍼런스 게임 15종 분석 + TheSevenSimulation에 적합한 개입 방식 도출

---

## 1. 현재 카드 시스템 냉정 평가

### 현재 설계 요약
- 오토배틀 중 SP(0→100)가 차면 소모품 카드를 사용
- 카드 5종 (ATK↑, 방어↑, 저주, 회복, 속도↑) — 전원 대상, 전투 버프/디버프
- 카드는 소모품 — 연금술소/상인/원정에서 획득
- 죄종 시너지: 파티 내 해당 죄종 영웅이 있으면 효과 강화

### 문제점

| # | 문제 | 설명 |
|---|------|------|
| 1 | **판단이 단순함** | "SP 차면 가장 좋은 카드 누르기" — 타이밍 고민이 거의 없음. 적 HP가 낮을 때 ATK↑? 아군 HP가 낮을 때 회복? 이 정도가 전부 |
| 2 | **전원 대상 = 선택지 없음** | 모든 카드가 "아군 전원" 또는 "적 전원" → 누구에게 쓸지 고민이 없음 |
| 3 | **소모품 = 안 쓰고 모으기** | 소모품이면 아끼게 됨. "이번 전투에서 쓸까 다음에 쓸까" → 대부분 안 씀. 소모품 시스템의 고질병 |
| 4 | **경제 시스템과 충돌** | 카드 획득/제작 경로가 미확정. 거점 경제(건설/연구/고용)와 카드 제작이 자원을 놓고 경쟁하면 복잡도만 올라감 |
| 5 | **게임 정체성과 불일치** | 이 게임의 핵심은 "거점 경영 + 죄종 드라마". 전투 중 카드 조작은 **액션 게임의 재미**를 추구하는 것 — 경영 시뮬과 어울리지 않음 |
| 6 | **관전 몰입 방해** | 오토배틀의 재미는 "내가 준비한 영웅들이 알아서 싸우는 걸 보는 것". 카드 사용 타이밍을 재느라 관전에 집중 못함 |
| 7 | **구현 비용 대비 효과** | 카드 풀 20종+ 설계, 밸런싱, 획득 경로, UI — 투자 대비 전투 체감 향상이 크지 않음 |

### 핵심 질문
> "바알(플레이어)이 전투에 개입하는 방식으로 **카드**가 최선인가?"

## 2. 레퍼런스 게임 15종 분석

### 분류: 전투 자동화 수준별

```
완전 수동 ◀────────────────────────────────────────▶ 완전 자동

Slay the Spire   HoMM3   Battle Brothers   Epic Seven   삼국지9   Kenshi
   DD            Total War(수동)                CQ       삼국지14   AFK Arena
                                           Arknights   Monster Train   RimWorld
```

### A. 완전 자동 — 관전형 (삼국지9, Kenshi, AFK Arena)

| 게임 | 전투 중 유저 행동 | 핵심 재미 |
|------|----------------|----------|
| **삼국지9** | 없음. 진형 선택 후 관전 | 확률 기반 전법 발동 → "내 준비가 맞았나?" 검증의 쾌감 |
| **Kenshi** | 이동 명령만 가능. 전투 AI가 알아서 싸움 | 0에서 키운 캐릭터가 성장하는 내러티브. 자동전투이기에 더 감동적 |
| **AFK Arena** | 궁극기 타이밍만 수동 (옵션) | 편성 퍼즐이 본체. 전투는 "답 확인" |

**공통점**: 전투 전 준비가 게임의 95%. 전투 중 개입이 적을수록 준비의 가치가 올라감.

### B. 반자동 — 제한적 개입 (삼국지14, Monster Train, Arknights)

| 게임 | 전투 중 유저 행동 | 핵심 재미 |
|------|----------------|----------|
| **삼국지14** | 이동 명령. 장수가 성격대로 독단 행동 | 장수 성격이 만드는 창발적 서사 (장비가 돌격, 참모가 회피) |
| **Monster Train** | 카드로 유닛 배치 + 스펠 사용 → 전투는 자동 | "기계를 조립하고 작동 보기". 3층 공간 전략 |
| **Arknights** | 실시간 배치 + 스킬 타이밍 | 배치 타이밍과 DP 경제 관리 |

**공통점**: 유저는 "배치/설정"을 하고, 실행은 시스템이. 삼국지14의 장수 독단은 **이 게임의 죄종 시스템과 직접 대응**.

### C. 수동 전투 — 전략형 (HoMM3, Battle Brothers, DD, Slay the Spire)

| 게임 | 전투 중 유저 행동 | 핵심 재미 |
|------|----------------|----------|
| **HoMM3** | 유닛 이동 + 영웅 마법 1회/턴 | 전략 맵 경영 + 전술 전투의 이중 레이어 |
| **Battle Brothers** | 완전 수동 턴제 전술 | 피로/사기/지형 관리. 용병 영구사망 |
| **Darkest Dungeon** | 완전 수동 턴제. 스트레스로 통제력 상실 | 스트레스 연쇄 → 영웅이 명령 거부 → 드라마 |
| **Slay the Spire** | 완전 수동 카드 전투 | 덱빌딩 + 적 의도 읽기 퍼즐 |

**공통점**: 전투 자체가 게임의 핵심 콘텐츠. 하지만 이 게임들은 **전투 비중이 50% 이상**. TheSevenSimulation은 거점 경영이 핵심이므로 이 방향은 과함.

### D. RimWorld — 특수 케이스

- 평소엔 경영. 습격 시 일시정지 + 징집 + 위치 지정
- 전투 준비(킬박스, 함정, 포탑)가 진짜 게임
- 전투 중 조작은 "위치 지정"이 대부분 — 공격 자체는 자동

- 가장 가까운 모델: **삼국지9/14 + Kenshi**

## 4. 카드 대안 — 5가지 방향

### 안 A: 카드 유지 (현행 개선)
- SP/카드 시스템 그대로, 카드 대상을 단일 영웅으로 변경
- 소모품 → 영구 보유 + 쿨타임으로 전환
- **장점**: 이미 구현됨
- **단점**: 근본적 문제(경영 시뮬에 액션 요소) 미해결

### 안 B: 전투 전 "작전 지시" (삼국지9 모델)
- 전투 시작 전에 3가지 지시 중 택1
  - 공격 중시: ATK↑, 방어↓
  - 방어 중시: 방어↑, ATK↓
  - 균형: 기본
- 전투 중 개입 없음. 완전 관전
- **장점**: 구현 단순, 경영 시뮬에 적합
- **단점**: 너무 단순할 수 있음. 관전이 지루할 수 있음

### 안 C: "바알의 명령" — 전투 중 1회 개입 (HoMM3 영웅 마법 모델)
- 전투 중 바알이 **1회만** 개입 가능 (쿨타임 or SP 기반)
- 선택지 3~4개: "분노를 부추겨라" / "후퇴하라" / "전력을 다해라" / "약점을 노려라"
- 효과는 강력하지만 1회라 타이밍이 중요
- **장점**: 관전하다가 결정적 순간에 한 번 개입 → 극적 + 단순
- **단점**: 1회가 너무 적게 느껴질 수 있음

### 안 D: "죄종 반응" — 자동 이벤트 + 유저 선택 (DD 스트레스 모델 + 삼국지14 성격)
- 전투 중 영웅의 사기/죄종에 따라 **이벤트가 자동 발생**
- 유저는 발생한 이벤트에 **반응 선택**만 함
- 예시:
  ```
  [분노] 영웅 "타론"이 폭주 직전!
  → ① 부추긴다 (ATK 2배, 제어 불능)
  → ② 진정시킨다 (사기 -10, 정상 유지)

  [교만] 영웅 "리아"가 일기토를 신청하려 한다!
  → ① 허락한다 (일기토 돌입)
  → ② 금지한다 (사기 -15)

  [나태] 영웅 "밀드"가 전의를 잃고 있다
  → ① 독려한다 (사기 +5, 피로↑)
  → ② 내버려둔다 (SPD -30%)
  ```
- **장점**:
  - 죄종 시스템과 완벽 통합 — 전투가 죄종 드라마의 무대가 됨
  - 매 전투가 다른 이벤트 → 반복감 없음
  - "판결자 바알" 역할과 일치 — 카드를 쓰는 마법사가 아니라 판단하는 영주
  - 관전 중 간헐적 선택 → 몰입 유지
  - 별도 카드 경제 시스템 불필요
- **단점**: 이벤트 종류를 다양하게 만들어야 함

### 안 E: 완전 관전 + 사전 전략 강화 (Kenshi 모델)
- 전투 중 개입 완전 제거
- 대신 전투 전 전략 옵션 확대: 진형 선택, 우선 공격 대상 지정, 후퇴 조건 설정
- **장점**: 가장 단순, 경영에 집중
- **단점**: 관전이 지루할 수 있음 (방어전 3~5일 간격이므로 감수 가능?)

## 5. 추천: 안 D "죄종 반응" 시스템

### 이유

| 평가 기준 | 카드 (현행) | 죄종 반응 (안 D) |
|---------|:---------:|:-------------:|
| 게임 정체성 (경영 시뮬) | ❌ 액션적 | ✅ 판결/경영적 |
| 죄종 시스템 활용 | △ 시너지만 | ✅ 핵심 연동 |
| "바알 = 판결자" 역할 | ❌ 마법사처럼 | ✅ 판단하는 영주 |
| 관전 몰입 | ❌ 카드 UI 주시 | ✅ 전투 보다가 선택 |
| 구현/밸런싱 복잡도 | 높음 (카드 20종+) | 중간 (죄종 7×이벤트 N개) |
| 반복 플레이 다양성 | 낮음 (최적 카드 고정) | 높음 (사기/죄종 조합마다 다름) |
| 별도 경제 시스템 | 필요 (카드 제작) | 불필요 |
| 드라마 생산 | 낮음 | 높음 (매 전투가 이야기) |

### 죄종 반응 시스템 기본 구조

```
전투 진행 (오토배틀)
  ↓
영웅 사기/죄종 조건 충족 시 → 이벤트 발생 (확률 기반)
  ↓
전투 일시정지 + 이벤트 팝업 (선택지 2개)
  ↓
플레이어 선택 → 효과 적용 → 오토배틀 재개
```

### 이벤트 발동 조건 예시

| 죄종 | 발동 조건 | 이벤트 |
|------|---------|--------|
| 분노 | HP < 50% | 폭주 직전 — 부추기기 vs 진정 |
| 교만 | 적 보스 존재 | 일기토 신청 — 허락 vs 금지 |
| 시기 | 다른 영웅이 처치 누적 | 독단 행동 — 방치 vs 제지 |
| 탐욕 | 적 처치 시 | 전리품 집착 — 허용 vs 명령 |
| 폭식 | 전투 장기화 (10라운드+) | 무모한 돌진 — 격려 vs 제지 |
| 색욕 | 동료 HP < 30% | 동료 보호 집착 — 허용 vs 금지 |
| 나태 | 전투 시작 시 | 전의 상실 — 독려 vs 방치 |

### 기존 일기토 시스템과의 통합
- 현재 일기토는 "자동 발동" → 죄종 반응으로 흡수
- [교만] 일기토 신청 → 유저가 허락/금지 선택 → 허락 시 일기토 씬 진행
- 일기토가 "유저의 판단"이 되면서 드라마 강화

## 6. 병행 가능: 안 B + 안 D 조합

"작전 지시"(전투 전) + "죄종 반응"(전투 중)을 결합하면:

```
[전투 전] 작전 지시 선택 (공격/방어/균형)
     ↓
[전투 중] 오토배틀 관전
     ↓
[전투 중] 죄종 이벤트 발생 시 → 판결 (선택지 2개)
     ↓
[전투 후] 결과 + 사기 변동
```

- 전투 전 준비 = 파티 편성 + 작전 지시 (안 B)
- 전투 중 개입 = 죄종 반응 이벤트 (안 D)
- 카드 시스템 = **제거**

이 구조가 "거점 경영 시뮬 + 죄종 드라마"라는 게임 정체성에 가장 부합.
