# 세력 시스템 & 초청 이벤트 설계

> 상태: 초안
> 작성일: 2026-04-06
> 참고: RimWorld(파벌), 프린세스메이커(대회), CK3(이벤트 체인), Darkest Dungeon(영웅 관리)

---

## 1. 개요

### 핵심 컨셉

세력은 **자율 AI가 아니라 이벤트 소스**. 독자적으로 행동하지 않고, 이벤트/의뢰/초청을 통해 거점에 영향을 미친다. RimWorld의 파벌 시스템과 동일한 접근.

| 삼국지식 (자율 국가) | 이 게임 (이벤트 소스) |
|---|---|
| 국가 AI가 매 턴 의사결정 | **없음** — 스크립트/확률 기반 |
| 각 국가 자원/경제 관리 | **없음** — "강한/약한" 태그만 |
| 구현 비용 3~5배 | **현재의 +30~40%** |
| 죄종 드라마 묻힘 | **보존** — 세력이 드라마의 연료 |

### 설계 원칙

- 세력은 **죄종 드라마에 연료를 공급하는 외부 장치**
- 새 메카닉 최소화, 기존 시스템(이벤트/전투/스탯체크) 재활용
- "살아있는 세계" 느낌은 이름과 맥락으로 충분 — AI 불필요

### 관계 시스템

세력마다 관계도(0~100)를 가진다.

| 구간 | 상태 | 가능한 상호작용 |
|---|---|---|
| 0~20 | 적대 | 침공 이벤트 발생 |
| 21~40 | 경계 | 제한적 교역 |
| 41~60 | 중립 | 의뢰, 교역 |
| 61~80 | 우호 | 초청 이벤트, 지원 |
| 81~100 | 동맹 | 방어전 지원군, 특수 보상 |

### 관계 변동 요인

| 트리거 | 관계 변화 |
|--------|----------|
| 챕터 보스 격파 | 주변 우호 세력 +10 |
| 방어전 연패 | 적대 세력 -10 (약해 보임) |
| 초청 이벤트 수락/우승 | 해당 세력 +10~20 |
| 초청 이벤트 거절 | 해당 세력 -5 |
| 의뢰 완료 | 해당 세력 +5~15 |
| 의뢰 실패 | 해당 세력 -10 |
| 이벤트 선택지 (약탈 등) | 해당 세력 -20~30 |
| 포고령: 긴축 | 교역 세력 -5 |

## 3. 세력과의 상호작용 — 3가지 채널

| 채널 | 트리거 | 형식 | 전제 |
|---|---|---|---|
| **침공** | 적대 관계 + 확률 | 밤 습격 확장 (이름 있는 적) | 없음 |
| **의뢰** | 의뢰소 건설 후 상시 | 건물 클릭 → 의뢰 목록 팝업 | 의뢰소 (Tier 2) |
| **초청** | 역마차 건설 후 주기적 | 아침 이벤트 → 수락 시 낮에 팝업 | 역마차 (Tier 2) |

## 4. 침공 (밤 습격 확장)

기존 밤 습격 시스템에 **세력 이름을 씌우는 것**만으로 체감이 달라진다.

| 현재 밤 습격 | 세력 침공 |
|---|---|
| "적이 쳐들어온다" | **"그림자 교단이 보복 침공을 시작했다"** |
| 이름 없는 적 | 세력 소속 적 (이름, 배경) |
| day 기반 규모 | 세력 강도 + 관계도 기반 규모 |
| 방어만 가능 | 방어 or 보복 원정 선택 |

### 침공 이벤트 (아침 경고)

```
[세력 침공 예고]
트리거: 적대 세력 관계 20 이하 + 확률
장면: "그림자 교단이 대규모 습격을 예고합니다."

▶ 방어를 준비한다     → 방어 배치 강화, 밤에 강화된 습격
▶ 선제 공격한다       → 영웅 2명 원정으로 세력 거점 공격
▶ 외교로 막는다       → 영웅 1명 파견, 매력+지능 체크
```

### 의뢰 결과 판정

```
성공 확률 = 50% + (적합 스탯 합 - 기준값 20) × 5%
  예: 힘14 + 민13 = 27, 기준 20 → 50% + 35% = 85%

성공 → 골드 + 관계↑ + 영웅 사기 +5
실패 → 관계↓ + 영웅 사기 -10 + 부상 가능 (risk=injury일 때)
```

### 의뢰 중 죄종 특수 반응

| 의뢰 유형 | 죄종 | 특수 반응 |
|----------|------|---------|
| 호위 | [탐욕] | 상단 물건 슬쩍 → 보상↑, 관계↓ |
| 토벌 | [분노] | 과잉 진압 → 보상↑, 부상 확률↑ |
| 배달 | [나태] | 게을러서 지연 → 관계 보상 절반 |
| 중재 | [교만] | "내가 판관이다" → 성공률↑, 관계 보상 절반 |
| 정찰 | [색욕] | 적 진영에서 엮임 → 기간 +1턴 |

## 6. 초청 이벤트 (카테고리 F)

### 구조

프린세스메이커의 대회 시스템을 참고. 세력이 **초청장**을 보내고, 영웅을 파견하면 플레이어가 **라운드별로 직접 조작**한다.

```
[전제] 역마차 건설 완료
  ↓
[아침] EventScene: "초청장이 도착했습니다"
  → ▶ 영웅을 보낸다 → 영웅 선택 팝업 (적합 스탯 ★ 표시)
  → ▶ 거절한다 → 세력 관계 -5
  → ▶ "다음에 보내겠다" → 관계 변동 없음, 기회 소멸
  ↓
[낮] 대회 팝업 (전투형: MapHuntPopup / 비전투형: 선택지 체인)
  → 3라운드, 매 라운드 플레이어 선택
  → 라운드 사이에 죄종 반응 이벤트 삽입
  ↓
[결과] 순위 판정 → 보상 + 귀환 후 죄종 연쇄
```

### 프린세스메이커와의 매핑

| 프린세스메이커 | TheSevenSimulation | 차이점 |
|---|---|---|
| 딸 1명 육성 | **영웅 7명 중 선택** | 누구를 보내느냐가 전략 |
| 스탯으로 자동 판정 | **라운드별 플레이어 선택** | 직접 조작감 |
| 축제 날짜 고정 | **역마차 이후 랜덤 초청** | 예고 없이 찾아옴 |
| 결과: 순위 + 상금 | 결과: 보상 + **죄종 드라마** | 귀환 후 연쇄반응 |
| 부작용 없음 | **보내는 동안 방어 인원↓** | 트레이드오프 |

### 순위 시스템

| 순위 | 조건 | 보상 배율 |
|---|---|---|
| 🏆 우승 | 3라운드 모두 성공 | ×1.0 |
| 🥈 준우승 | 2라운드 성공 | ×0.5 |
| 참가 | 1라운드만 성공 | ×0.2 |
| 실격 | 사고 or 허세 실패 | 보상 없음, 관계↓ |

### 귀환 후 죄종 연쇄 (공통)

| 결과 | 파견 영웅 | 미참가 영웅 |
|------|---------|-----------|
| 우승 | 사기 +20 | [시기] 사기 -15, [교만] 사기 -10 |
| 준우승 | 사기 +5 | 변동 없음 |
| 탈락 | 사기 -10 ([교만]이면 -20) | 변동 없음 |

### F-1. 무투회 — 「피의 왕관」

> 세력: 용병단
> 핵심 스탯: **힘 + 민첩**
> 방식: **전투 3라운드** (MapHuntPopup 재활용)
> 빈도: ~12턴 간격

**초청장:**
*"용병단이 무투회를 엽니다. 우승자에게 '피의 왕관'과 상금을 수여합니다. '죽이지만 않으면 됩니다. 아마도.'"*

**1라운드 — 예선:** 약한 상대와 1:1 전투 (MapHuntPopup)

승리 후 인터벌 선택지:
| 선택지 | 결과 |
|--------|------|
| ▶ 정면 승부한다 | 다음 전투 정상 진행 |
| ▶ 도발한다 "겁먹었냐" | 적 ATK↑ SPD↓. [교만] 사기+5 |
| ▶ 군중에게 어필한다 | 매력 체크. 성공: 적 사기↓ 보너스 |
| ▶ 기권한다 | 예선 보상만 획득 |

**2라운드 — 본선:** 중간 상대. 인터벌에 회복약 사용 선택지 추가.

**3라운드 — 결승:** 강한 상대. 전투 후 결과 정산.

**보상:**

| 결과 | 보상 |
|------|------|
| 우승 | 장비 + 💰150 + 용병단 관계+20 |
| 준우승 | 💰80 + 관계+10 |
| 예선 탈락 | 💰30 |

### F-2. 교역 경매 — 「황금의 밤」

> 세력: 교역 도시
> 핵심 스탯: **지능 + 매력**
> 방식: **선택지 3라운드** (스탯 체크)
> 빈도: ~15턴 간격

**초청장:**
*"교역 도시에서 대경매를 엽니다. 희귀 물품이 나온다는 소문입니다. '좋은 눈과 좋은 혀가 필요합니다.'"*

**매 라운드:** 물건이 나오고 입찰 방법을 선택.

| 선택지 | 결과 |
|--------|------|
| ▶ 정가에 입찰한다 | 50% 낙찰 |
| ▶ 높게 부른다 (골드 1.5배) | 90% 낙찰 |
| ▶ 허세를 부린다 (매력 체크) | 성공: 할인 낙찰 / 실패: 퇴장 |
| ▶ 패스 | 다음 물건으로 |

**죄종 자동 반응 (2라운드 시작 시 삽입):**

| 주 성향 | 자동 사건 | 플레이어 대응 |
|---------|---------|-------------|
| [탐욕] | 옆 상인 지갑에 손이 간다 | 말리기/눈감기 |
| [교만] | "이 정도 물건에 이 값?" 코웃음 | 사과시키기/방치 |
| [폭식] | 경매장 다과를 혼자 먹어치움 | 말리기/같이 먹기 |

### F-3. 사냥 대회 — 「짐승왕의 뿔」

> 세력: 우호 마을
> 핵심 스탯: **힘 + 감각**
> 방식: **탐색 선택지 2라운드 + 보스 전투 1라운드**
> 빈도: ~12턴 간격

**초청장:**
*"인근 숲에 거대 짐승이 나타났습니다. 여러 세력이 사냥단을 보내고 있습니다. '먼저 잡는 쪽이 뿔과 명예를 가집니다.'"*

**1라운드 — 추적:**

| 선택지 | 결과 |
|--------|------|
| ▶ 발자국을 따른다 (감각 체크) | 성공: 선제권 / 실패: 라이벌이 앞서감 |
| ▶ 높은 곳에서 살핀다 (민첩 체크) | 성공: 짐승+라이벌 위치 파악 / 실패: HP 소모 |
| ▶ 함정을 설치한다 (지능 체크) | 성공: 전투에서 짐승 HP -30% / 실패: 함정 낭비 |

**2라운드 — 조우:**

| 선택지 | 결과 |
|--------|------|
| ▶ 돌진한다 (힘 체크) | 성공: 전투 유리 보정 |
| ▶ 라이벌을 방해한다 | 라이벌 탈락, 승리 확정적. 세력 관계-10 |
| ▶ 라이벌과 협력한다 | 전투 쉬워짐. 보상 절반, 관계+10 |

**3라운드 — 전투:** 보스급 야수와 1:1 (MapHuntPopup). 1~2라운드 선택에 따라 조건 변동.

---

### F-4. 변론 대회 — 「진실의 법정」

> 세력: 교역 도시 / 종교 세력
> 핵심 스탯: **지능 + 매력**
> 방식: **선택지 3라운드** (답변이 전원 사기에 영향)
> 빈도: ~20턴 간격

**초청장:**
*"교역 도시에서 연례 변론 대회를 엽니다. 주제: '죄는 선택인가, 숙명인가.' '승자에게 도시의 지지를 약속합니다.'"*

이 대회의 핵심: **플레이어의 답변이 거점 영웅 전원의 사기에 영향**.
바알(판결자)의 철학이 드러나는 이벤트.

**1라운드 — 논제: "약자를 지키는 것은 의무인가"**

| 선택지 | 스탯 체크 | 사기 영향 |
|--------|---------|---------|
| ▶ "의무다. 힘은 책임을 동반한다" | 지능 체크 | [나태] 사기-5, [교만] 사기+5 |
| ▶ "동의한다. 그래서 대가를 받는다" | 매력 체크 | [탐욕] 사기+5 |
| ▶ "약한 것이 죄다" | 자동 성공 | [분노] 사기+10, 전체 사기-5 |
| ▶ 영웅에게 맡긴다 | 주 성향 기반 자동 | 예측 불가 |

**2라운드 — 논제: "복수는 정당한가"**
바알의 여정 자체를 묻는 주제. 선택이 챕터 7 과거 결산에 기록됨.

**3라운드 — 논제: "용서할 수 없는 죄가 있는가"**
가장 무거운 주제. 모든 선택지가 강한 사기 변동.

**"영웅에게 맡긴다" 선택지 — 주 성향별 자동 답변:**

| 주 성향 | 답변 경향 | 결과 |
|---------|---------|------|
| 분노 | 공격적, 단정적 | 점수 낮지만 군중 열광 |
| 교만 | 웅변적, 독선적 | 점수 높지만 거점 영웅 불만 |
| 탐욕 | 실리적, 거래 제안 | 점수 중간, 골드 보너스 |
| 나태 | 대충 대답 | 점수 최하 |
| 시기 | 상대 깎아내리기 | 점수 중간, 상대 세력 관계↓ |
| 폭식 | 주제 이탈 | 관중 호감 |
| 색욕 | 감정 호소 | 점수 높음, 관계 이벤트 |

---

### F-5. 대연회 — 「탐닉의 밤」

> 세력: 교역 도시 / 우호 마을
> 핵심 스탯: **매력 + 감각**
> 방식: **선택지 3라운드** (사교 + 사건 + 귀환)
> 빈도: ~15턴 간격

**초청장:**
*"교역 도시에서 대연회를 엽니다. 좋은 음식, 좋은 술, 좋은 인연. '단, 분별을 잃지 않는다면.'"*

이 대회의 핵심: **죄종이 가장 폭발하는 이벤트**. 사교 장면에서 죄종이 사고를 친다.

**1라운드 — 입장:**

| 선택지 | 결과 |
|--------|------|
| ▶ 용병단 수장과 대화한다 | 매력 체크. 성공: 용병단 관계+10 |
| ▶ 교역 상인과 거래한다 | 지능 체크. 성공: 다음 교역 할인 |
| ▶ 술과 음식을 즐긴다 | 사기+10. [폭식] 과식 위험 |
| ▶ 조용히 관찰한다 | 감각 체크. 성공: 세력 정보 획득 |

**2라운드 — 사건 발생 (영웅 죄종 기반 자동 분기):**

영웅의 주 성향에 따라 **자동으로 사건이 터진다**:

| 주 성향 | 자동 사건 | 플레이어 대응 선택지 |
|---------|---------|-------------------|
| 분노 | 다른 세력 대표와 말다툼 | 말리기/방치/편들기 |
| 시기 | 다른 참가자 장비를 흉봄 | 질책/무시/장비 약속 |
| 탐욕 | 연회장 은식기를 슬쩍 챙김 | 말리기/눈감기/돌려놓기 |
| 나태 | 구석에서 잠들어버림 | 깨우기/그냥 두기 |
| 폭식 | 뷔페를 혼자 비움 | 말리기/같이 먹기/사과 |
| 색욕 | 다른 세력 인물에게 접근 | 말리기/방치/이용(정보) |
| 교만 | 아무도 안 부탁했는데 연설 | 말리기/박수치기/퇴장 |

**3라운드 — 귀환:** 2라운드 대응 결과에 따라 보상/관계/사기 정산.

## 7. 챕터별 세력 이벤트 강조

세력 이벤트도 챕터 환경 변조와 연동한다:

| 챕터 | 강조 세력 이벤트 | 이유 |
|------|----------------|------|
| 1 분노 | **무투회 빈도↑**, 침공 | 전쟁 분위기 |
| 2 시기 | **사냥 대회** (비교 유발) | "왜 쟤를 보냄" 연쇄 |
| 3 탐욕 | **교역 경매 빈도↑**, 의뢰 보상↑ | 거래와 유혹 |
| 4 나태 | 초청 감소, 침공 감소 | 세계도 멈추는 느낌 |
| 5 폭식 | **대연회** (식량 부족 중 초대) | 나눌 것인가 |
| 6 색욕 | **변론 대회** (신뢰 주제) | 불신 분위기 |
| 7 교만 | **합동 훈련 이벤트** | 동맹 총결산 |

## 8. 죄종 시스템과의 시너지

세력 이벤트가 **죄종 드라마를 증폭**시켜야 한다:

| 세력 이벤트 | 죄종 연쇄 |
|------------|----------|
| 교역 경매에서 사치품 도착 | [탐욕] 사기↑(갖고 싶다) → [시기] 사기↓(왜 쟤만) |
| 우호 마을이 도움 요청 | [교만] "그깟 마을" 사기↓ / [나태] "귀찮다" 사기↓ |
| 무투회 우승 | 우승 영웅 사기↑↑ → [시기] 다른 영웅 사기↓ → 연쇄반응 |
| 대연회 초대 | [폭식] 과식 → [탐욕] 은식기 → [색욕] 분쟁 → 다중 연쇄 |
