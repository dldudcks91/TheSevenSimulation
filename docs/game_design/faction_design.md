> 상태: 초청 이벤트 상세 완성 (F-1~F-8) — **Phase 2 예정**
> 작성일: 2026-04-06
> 참고: RimWorld(파벌), 프린세스메이커(대회), CK3(이벤트 체인), Darkest Dungeon(영웅 관리)

## 1. 개요

### 핵심 컨셉

세력은 **자율 AI가 아니라 이벤트 소스**. 독자적으로 행동하지 않고, 이벤트/의뢰/초청을 통해 거점에 영향을 미친다. RimWorld의 파벌 시스템과 동일한 접근.

| 삼국지식 (자율 국가) | 이 게임 (이벤트 소스) |
|---|---|
| 국가 AI가 매 턴 의사결정 | **없음** — 스크립트/확률 기반 |
| 각 국가 자원/경제 관리 | **없음** — "강한/약한" 태그만 |
| 구현 비용 3~5배 | **현재의 +30~40%** |
| 죄종 드라마 묻힘 | **보존** — 세력이 드라마의 연료 |

### 설계 원칙

- 세력은 **죄종 드라마에 연료를 공급하는 외부 장치**
- 새 메카닉 최소화, 기존 시스템(이벤트/전투/스탯체크) 재활용
- "살아있는 세계" 느낌은 이름과 맥락으로 충분 — AI 불필요

### 관계 시스템

세력마다 관계도(0~100)를 가진다.

| 구간 | 상태 | 가능한 상호작용 |
|---|---|---|
| 0~20 | 적대 | 침공 이벤트 발생 |
| 21~40 | 경계 | 제한적 교역 |
| 41~60 | 중립 | 의뢰, 교역 |
| 61~80 | 우호 | 초청 이벤트, 지원 |
| 81~100 | 동맹 | 방어전 지원군, 특수 보상 |

### 관계 변동 요인

| 트리거 | 관계 변화 |
|--------|----------|
| 챕터 보스 격파 | 주변 우호 세력 +10 |
| 방어전 연패 | 적대 세력 -10 (약해 보임) |
| 초청 이벤트 수락/우승 | 해당 세력 +10~20 |
| 초청 이벤트 거절 | 해당 세력 -5 |
| 의뢰 완료 | 해당 세력 +5~15 |
| 의뢰 실패 | 해당 세력 -10 |
| 이벤트 선택지 (약탈 등) | 해당 세력 -20~30 |

## 3. 세력과의 상호작용 — 3가지 채널

| 채널 | 트리거 | 형식 | 전제 |
|---|---|---|---|
| **침공** | 적대 관계 + 확률 | 밤 습격 확장 (이름 있는 적) | 없음 |
| **의뢰** | 의뢰소 건설 후 상시 | 건물 클릭 → 의뢰 목록 팝업 | 의뢰소 (Tier 2) |
| **초청** | 역마차 건설 후 주기적 | 아침 이벤트 → 수락 시 낮에 팝업 | 역마차 (Tier 2) |

## 4. 침공 (밤 습격 확장)

기존 밤 습격 시스템에 **세력 이름을 씌우는 것**만으로 체감이 달라진다.

| 현재 밤 습격 | 세력 침공 |
|---|---|
| "적이 쳐들어온다" | **"그림자 교단이 보복 침공을 시작했다"** |
| 이름 없는 적 | 세력 소속 적 (이름, 배경) |
| day 기반 규모 | 세력 강도 + 관계도 기반 규모 |
| 방어만 가능 | 방어 or 보복 원정 선택 |

### 침공 이벤트 (아침 경고)

```
[세력 침공 예고]
트리거: 적대 세력 관계 20 이하 + 확률
장면: "그림자 교단이 대규모 습격을 예고합니다."

▶ 방어를 준비한다     → 방어 배치 강화, 밤에 강화된 습격
▶ 선제 공격한다       → 영웅 2명 원정으로 세력 거점 공격
▶ 외교로 막는다       → 영웅 1명 파견, 매력+지능 체크
```

### 의뢰 결과 판정

```
성공 확률 = 50% + (적합 스탯 합 - 기준값 20) × 5%
  예: 힘14 + 민13 = 27, 기준 20 → 50% + 35% = 85%

성공 → 골드 + 관계↑ + 죄종 수치 소폭 감소 (정화, TBD)
실패 → 관계↓ + 죄종 수치 소폭 증가 (TBD) + 부상 가능 (risk=injury일 때)
```

## 6. 초청 이벤트 (카테고리 F)

### 구조

프린세스메이커의 대회 시스템을 참고. 세력이 **초청장**을 보내고, 영웅을 파견하면 플레이어가 **라운드별로 직접 조작**한다.

```
[전제] 역마차 건설 완료
  ↓
[아침] EventScene: "초청장이 도착했습니다"
  → ▶ 영웅을 보낸다 → 영웅 선택 팝업 (적합 스탯 ★ 표시)
  → ▶ 거절한다 → 세력 관계 -5
  → ▶ "다음에 보내겠다" → 관계 변동 없음, 기회 소멸
  ↓
[낮] 대회 팝업 (전투형: MapHuntPopup / 비전투형: 선택지 체인)
  → 3라운드, 매 라운드 플레이어 선택
  → 라운드 사이에 죄종 반응 이벤트 삽입
  ↓
[결과] 순위 판정 → 보상 + 귀환 후 죄종 연쇄
```

### 프린세스메이커와의 매핑

| 프린세스메이커 | TheSevenSimulation | 차이점 |
|---|---|---|
| 딸 1명 육성 | **영웅 7명 중 선택** | 누구를 보내느냐가 전략 |
| 스탯으로 자동 판정 | **라운드별 플레이어 선택** | 직접 조작감 |
| 축제 날짜 고정 | **역마차 이후 랜덤 초청** | 예고 없이 찾아옴 |
| 결과: 순위 + 상금 | 결과: 보상 + **죄종 드라마** | 귀환 후 연쇄반응 |
| 부작용 없음 | **보내는 동안 방어 인원↓** | 트레이드오프 |

### 순위 시스템

| 순위 | 조건 | 보상 배율 |
|---|---|---|
| 🏆 우승 | 3라운드 모두 성공 | ×1.0 |
| 🥈 준우승 | 2라운드 성공 | ×0.5 |
| 참가 | 1라운드만 성공 | ×0.2 |
| 실격 | 사고 or 허세 실패 | 보상 없음, 관계↓ |

### 귀환 후 죄종 연쇄 (공통)

| 결과 | 파견 영웅 | 미참가 영웅 |
|------|---------|-----------|
| 우승 | 죄종 수치 감소 (정화, TBD) | [시기] 수치 증가 (TBD), [교만] 수치 증가 (TBD) |
| 준우승 | 죄종 수치 소폭 감소 (TBD) | 변동 없음 |
| 탈락 | 죄종 수치 증가 (TBD, [교만]이면 대폭 증가) | 변동 없음 |

### F-1. 무투회 — 「피의 왕관」

> 세력: 용병단
> 핵심 스탯: **힘 + 민첩**
> 방식: **전투 3라운드** (MapHuntPopup 재활용)
> 빈도: ~12턴 간격

**초청장:**
*"용병단이 무투회를 엽니다. 우승자에게 '피의 왕관'과 상금을 수여합니다. '죽이지만 않으면 됩니다. 아마도.'"*

**1라운드 — 예선:** 약한 상대와 1:1 전투 (MapHuntPopup)

승리 후 인터벌 선택지:
| 선택지 | 결과 |
|--------|------|
| ▶ 정면 승부한다 | 다음 전투 정상 진행 |
| ▶ 도발한다 "겁먹었냐" | 적 ATK↑ SPD↓. [교만] 수치 증가 (TBD) |
| ▶ 군중에게 어필한다 | 매력 체크. 성공: 적 전투력 감소 보너스 |
| ▶ 기권한다 | 예선 보상만 획득 |

**2라운드 — 본선:** 중간 상대. 인터벌에 회복약 사용 선택지 추가.

**3라운드 — 결승:** 강한 상대. 전투 후 결과 정산.

**보상:**

| 결과 | 보상 |
|------|------|
| 우승 | 장비 + 💰150 + 용병단 관계+20 |
| 준우승 | 💰80 + 관계+10 |
| 예선 탈락 | 💰30 |

### F-2. 교역 경매 — 「황금의 밤」

> 세력: 교역 도시
> 핵심 스탯: **지능 + 매력**
> 방식: **선택지 3라운드** (스탯 체크)
> 빈도: ~15턴 간격

**초청장:**
*"교역 도시에서 대경매를 엽니다. 희귀 물품이 나온다는 소문입니다. '좋은 눈과 좋은 혀가 필요합니다.'"*

**매 라운드:** 물건이 나오고 입찰 방법을 선택.

| 선택지 | 결과 |
|--------|------|
| ▶ 정가에 입찰한다 | 50% 낙찰 |
| ▶ 높게 부른다 (골드 1.5배) | 90% 낙찰 |
| ▶ 허세를 부린다 (매력 체크) | 성공: 할인 낙찰 / 실패: 퇴장 |
| ▶ 패스 | 다음 물건으로 |

**죄종 자동 반응 (2라운드 시작 시 삽입):**

| 주 성향 | 자동 사건 | 플레이어 대응 |
|---------|---------|-------------|
| [탐욕] | 옆 상인 지갑에 손이 간다 | 말리기/눈감기 |
| [교만] | "이 정도 물건에 이 값?" 코웃음 | 사과시키기/방치 |
| [폭식] | 경매장 다과를 혼자 먹어치움 | 말리기/같이 먹기 |

### F-3. 사냥 대회 — 「짐승왕의 뿔」

> 세력: 우호 마을
> 핵심 스탯: **힘 + 감각**
> 방식: **탐색 선택지 2라운드 + 보스 전투 1라운드**
> 빈도: ~12턴 간격

**초청장:**
*"인근 숲에 거대 짐승이 나타났습니다. 여러 세력이 사냥단을 보내고 있습니다. '먼저 잡는 쪽이 뿔과 명예를 가집니다.'"*

**1라운드 — 추적:**

| 선택지 | 결과 |
|--------|------|
| ▶ 발자국을 따른다 (감각 체크) | 성공: 선제권 / 실패: 라이벌이 앞서감 |
| ▶ 높은 곳에서 살핀다 (민첩 체크) | 성공: 짐승+라이벌 위치 파악 / 실패: HP 소모 |
| ▶ 함정을 설치한다 (지능 체크) | 성공: 전투에서 짐승 HP -30% / 실패: 함정 낭비 |

### F-8. 합동 훈련 — 「전우의 칼」

> 세력: 용병단 / 우호 세력
> 핵심 스탯: **힘 + 통솔**
> 방식: **전투 1라운드 + 선택지 2라운드**
> 빈도: ~15턴 간격

**초청장:**
*"용병단이 합동 훈련을 제안합니다. '서로의 약점을 알아야 함께 싸울 수 있소. 영웅 하나 보내시오.'"*

**1라운드 — 모의전:**
용병단 소속 전사와 1:1 모의전 (MapHuntPopup). **사망 없음** (HP 0 = 기절, 즉시 회복).

승패에 따라 2라운드 분기:

**2라운드 — 교류 (승리 시):**
장면: *"용병들이 감탄합니다. '대단한 솜씨요.' 수장이 술을 건넵니다."*

| 선택지 | 결과 |
|--------|------|
| ▶ 전술을 가르쳐준다 (통솔 체크) | 관계+15, 영웅 전투 스탯 임시 버프 (3턴) |
| ▶ 술을 받는다 | 관계+10, [폭식] 수치 증가 (TBD) |
| ▶ "다음엔 진검으로 하자" | [분노] 수치 감소 (정화, TBD), 용병단 관계+5 (용감한 놈) |

**2라운드 — 교류 (패배 시):**
장면: *"용병 수장이 어깨를 두드립니다. '약점을 아는 것도 훈련이오.'"*

| 선택지 | 결과 |
|--------|------|
| ▶ 가르침을 청한다 | 관계+10, 영웅 약점 스탯 +1 (임시, 5턴) |
| ▶ "다음엔 이긴다" | [교만] 수치 증가 (TBD), [분노] 수치 감소 (정화, TBD) |
| ▶ 조용히 돌아간다 | 관계+5, 죄종 수치 소폭 증가 (TBD) |

**3라운드 — 제안:**
장면: *"수장이 마지막으로 묻습니다. '다음 습격 때 같이 싸울 의향이 있소?'"*

| 선택지 | 결과 |
|--------|------|
| ▶ 동의한다 | 관계+20. **다음 방어전에 용병단 지원군 합류** (임시 NPC 동맹) |
| ▶ 거절한다 ("우리 힘으로 충분하다") | [교만] 수치 증가 (TBD), 관계-5 |
| ▶ "대가는?" | 💰-50, 관계+15, 지원군 합류 |

### 초청 이벤트 출현 규칙

#### 전제 조건

| 조건 | 내용 |
|------|------|
| 건설 | **역마차** 완성 필수 |
| 최소 일차 | 10일 이상 경과 |
| 세력 관계 | 해당 세력과 관계 41 이상 (중립~) |
| 동시 진행 | 초청 이벤트는 **턴당 1개만** 발생 |

#### 출현 확률/쿨다운

| 이벤트 | 쿨다운 | 턴당 출현률 | 최소 일차 | 챕터 강조 |
|--------|--------|-----------|---------|----------|
| F-1 무투회 | 12턴 | 15% | 10 | 챕터1 ×2 |
| F-2 교역 경매 | 15턴 | 12% | 15 | 챕터3 ×2 |
| F-3 사냥 대회 | 12턴 | 15% | 10 | 챕터2 ×2 |
| F-4 변론 대회 | 20턴 | 8% | 20 | 챕터6 ×2 |
| F-5 대연회 | 15턴 | 10% | 15 | 챕터5 ×2 |
| F-6 수확제 | 챕터당 1회 | 고정 | 10 | 챕터5 |
| F-7 속죄의 밤 | 20턴 | 8% | 20 | 챕터4 ×2 |
| F-8 합동 훈련 | 15턴 | 12% | 12 | 챕터7 ×2 |

> **쿨다운**: 해당 이벤트 마지막 발생 후 N턴 경과해야 재발생 가능.
> **턴당 출현률**: 쿨다운 충족 + 전제 충족 상태에서 매 턴 판정.
> **챕터 강조**: 해당 챕터에서 출현률 2배 (×2). 챕터4 나태는 전체 초청 출현률 ×0.5.
> **수확제**는 챕터당 1회 고정 발생 (쿨다운 대신 챕터 플래그).

## 8. 죄종 시스템과의 시너지

세력 이벤트가 **죄종 드라마를 증폭**시켜야 한다:

| 세력 이벤트 | 죄종 연쇄 |
|------------|----------|
| 교역 경매에서 사치품 도착 | [탐욕] 수치 증가 (탐욕 축적) → [시기] 수치 증가 (시기 축적) |
| 우호 마을이 도움 요청 | [교만] '그깟 마을' 수치 증가 / [나태] '귀찮다' 수치 증가 |
| 무투회 우승 | 우승 영웅 수치 감소 → [시기] 다른 영웅 수치 증가 → 연쇄반응 |
| 대연회 초대 | [폭식] 과식 → [탐욕] 은식기 → [색욕] 분쟁 → 다중 연쇄 |
