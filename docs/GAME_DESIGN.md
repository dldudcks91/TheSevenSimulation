## 1. 게임 개요

- **장르**: 거점 디펜스 + 로스터 서사 시뮬레이션
- **테마**: 7대 죄악 + 중세 다크 판타지
- **배경**: Aurea — 신이 걸었던 땅, 7대 죄악이 지배하는 세계 (TheSevenTactics 계승)
- **핵심 재미**: 결함 있는 영웅들의 **감정 충돌과 폭주**가 **매일 밤 덮쳐오는 디펜스 전투**의 방어선을 흔드는 운영 드라마
- **코어 축**: ① 거점 디펜스(매일 밤) ② 결함 있는 영웅들의 서사(감정·관계·폭주) — 원정은 **선택적 서브 루프**
- **컨셉**: "타락한 마왕이 결함 있는 인간들을 거점에 모아, 7대 죄악의 물결을 막아내며 복수의 발판을 쌓는다"
- **스토리 전달**: 컷신이 아닌 **시뮬레이션 체감** — 챕터별 환경 변조 + 이벤트 + 보스 서사 ([chapter_scenario.md](game_design/chapter_scenario.md))
- **영웅 정체성**: 인간 (용병, 떠돌이, 버림받은 자) — 각자 **메인 죄종 1개**를 짊어진 결함 있는 인간들
- **플랫폼**: Phase 1 웹 프로토타입 (Phaser.js) → Phase 2 Godot(스팀)

## 2. 핵심 게임 루프

### 설계 의도: 디펜스 + 영웅 서사

**코어 = 거점 디펜스 + 결함 있는 영웅들의 서사**
매일 밤 거점을 향해 죄악의 군세가 밀려오고, 플레이어는 결함 있는 영웅들로 그것을 막아낸다.
영웅들의 감정·관계·죄종은 **전투 전열을 흔드는 변수**로 작동하며, 폭주한 영웅은 방어선을 무너뜨린다.

- **낮 (준비)**: 방어 배치 · 시설 보강 · 감정 관리 · 선택적 원정 파견
- **밤 (실전)**: 디펜스 전투(오토배틀 관전) + 자율 행동에서 감정/관계 드라마 발생
- **아침 (피드백)**: 전투 사상자 · 감정 변화 · 관계 사건 판결
- **원정 (서브)**: 자원/정보 확보를 위해 방어선 약화를 감수하는 선택적 파견

### 하루(1턴) 구조

```
[아침] 밤의 결과 보고 + 소식 & 이벤트
  → ★ 밤 자율 행동 결과 보고 ("어젯밤 일어난 일" 텍스트 이벤트)
  → 문제 발생 시 플레이어 판결 (수습/처벌/무시)
  → 세계 소식, 영웅 간 사건, 죄종 관련 이벤트 발생

[낮] 디펜스 준비 (바알의 통제)
  → ★★ 방어 배치: 오늘 밤 디펜스에 투입할 영웅을 전열/후열/예비로 배치 (코어 판단)
  → 시설 운영: 건설 / 연구 / 생산 — 방어 보강 + 자원 축적
  → 감정 관리: 갈등 조합 분리, 국시 선포로 분위기 조율
  → 채집/벌목: 식량/나무 획득 (맵 위 행동 팝업 — 프로그레스 + 결과 + 죄종 대사)
  → 사냥: 영웅 1명 → 야생 적과 1:1 전투 (맵 위 팝업)
  → 원정 파견 (선택): 영웅 파견 — 방어선 약화를 감수하고 자원/정보 확보
  ★ 핵심 딜레마: 오늘 밤을 버틸 전열 vs 내일을 위한 자원/원정 — 누구를 방어선에 세울지

[저녁] 원정 결과 & 정보
  → 원정 결과 보고 (승리/패배, 전리품, 부상)
  → "더 진행시킬까, 귀환시킬까?" 판단
  → 감시탑 정보: 오늘 밤 습격 규모 예고 (감시탑 레벨에 따라 정보량 변동)

[밤] 디펜스 전투 + 자율 행동 (실전)
  → ★★ 적 습격 — 매일 밤 발생, 규모 가변 (소규모 정찰 ~ 대규모 공성, 챕터 후반 누적)
  → 디펜스 전투: 맵 위 오버레이로 방어전 (오토배틀 관전 + 죄종 반응 + 폭주 위험)
    → 전투 중 감정 폭주 시 방어선 붕괴, 이탈 시 전열 이탈
  → ★ 영웅 자율 행동 (비전투 영웅) — 죄종/체력에 따라 밤 이용 가능 시설을 자율 이용
    → 숙소 배치(사전 설정)에 따른 동실 영웅 간 상호작용
    → 죄종 드라마 발생 (텍스트 이벤트로 다음 아침 보고)
  → 폭주/이탈 체크 → 연쇄 반응?
  → 다음 날로
```

### 플레이어 행동 요약

| 시간대 | 능동적 판단 |
|--------|-----------|
| 아침 | 밤 전투 사상자 확인 + 감정/관계 사건 판결 |
| 낮 | ① **방어 배치** (코어) ② 시설 운영 + 감정 관리 ③ 작업 지시 (건설/연구/생산/사냥/채집) ④ **원정 파견** (선택) |
| 저녁 | 원정 계속/철수 판단, 감시탑 정보로 오늘 밤 습격 규모 확인 |
| 밤 | 디펜스 전투 관전 + 폭주/죄종 반응 이벤트 |

### 마을 안/밖 구분

| 위치 | 행동 | 밤 방어 | 챕터 진행 |
|------|------|--------|----------|
| **마을 안** | 건설, 연구, 생산, 채집, 휴식 | O | X |
| **마을 밖** | 원정, 벌목, 사냥 | X | O (원정만) |

### 메타 루프

```
[낮: 방어 준비]   방어 배치 + 시설 보강 + 감정 관리
    ↓
[밤: 디펜스 전투] 매일 밤 습격 — 오토배틀 관전 + 폭주/이탈이 방어선을 흔든다 ★ 메인 축
    ↓
[밤: 영웅 드라마] 비전투 영웅 자율 행동 → 숙소/시설 상호작용 → 감정/관계 사건
    ↓
[아침: 수습]     전투 사상자 + 감정/관계 판결 → 죄종 수치 변동, 연쇄 반응
    ↓
[선택: 원정 서브] 방어선 약화를 감수하고 파티 파견 → 챕터/스테이지 진행 → 전리품
    ↓
반복 → 챕터 막바지 대규모 침공 격퇴 → 7챕터 클리어 → 엔딩
```

## 3. 바알(플레이어)의 역할

> 핵심 설계 원칙: **바알은 판결자이자 영주** — 국시(國是)로 거점을 통치하며, 이벤트 판결로 영웅들에 영향

### 구조

바알(플레이어)은 누적 수치 없이, **국시(Edict)** 를 통해 직접 통치 방향을 결정한다:
- 7죄종 중 1개를 **국시로 선포** (기간형 5턴 지속)
- 선포한 국시의 효과가 거점 전반에 적용 (전투력/자원/생산성/영웅 수치 변동)
- 만료 후 공백 기간(1턴)을 거쳐 다음 국시 선포 가능
- 매 턴 이벤트에 대해 선택지를 고르면 **영웅들의 사기**에 영향 (사기 modifier 추가/제거)
- 트레이드오프 — 누군가를 만족시키면 다른 누군가가 불만

### 국시(Edict) 시스템 (2026-04-17 도입)

- **7종 국시**: 강병령(분노)/축재령(탐욕)/풍요령(폭식)/안식령(나태)/경쟁령(시기)/열정령(색욕)/위광령(교만)
- **기간형**: 선포 시 5턴 지속 후 자동 만료
- **자유 선포**: 해금 조건 없음. 언제든 7종 중 선택 가능
- **쿨다운**: 만료 후 1턴 / 조기 해제 시 2턴 / 대립 국시 전환 시 +1턴
- **효과**: 전투력/자원 획득/죄종 누적 가속/건설 속도 등 거점 전반에 배율/증감

> 상세: [sin_system.md](game_design/sin_system.md) §5 국시(Edict) 시스템

### 플레이어 선택이 게임에 미치는 영향

1. **방어 배치** — 오늘 밤 전열/후열/예비에 누구를 세울지 (디펜스 코어 레버)
2. **감정·관계 관리** — 충돌 조합을 피하고 폭주를 지연시키는 숙소/시설 배치
3. **영웅 사기 변동** — 선택에 따라 관련 영웅의 사기 modifier가 추가/제거됨
4. **연쇄 반응** — 한 영웅의 폭주 결과가 다른 영웅의 사기 modifier로 번져 새로운 사건을 만듦
5. **국시 선포** — 7종 중 1개 선택, 5턴간 거점 정책 적용
6. **거점 상태** — 시설 배치, 자원 배분 등 경영 판단의 직접적 결과
7. **원정 편성** (서브) — 방어선 약화를 감수할지의 트레이드오프

## 4. 영웅 시스템

### 기본 구조 (2026-04-27 메인 죄종 고정 + 사기 시스템 롤백)

```
[영웅] — 반고정 생성 (이름+메인 죄종+시작특성 고정, 스탯 랜덤)
├── 이름 (21명 고정 풀에서 추출, 중복 불가)
├── 메인 죄종 1개 (이름별 1:1 고정, 폭주 양상 결정자)
├── 시작특성 1개 (이름별 1:1 고정, 장점 전용) — 능력+사기+폭주 양상에 다축 영향
├── 사기(Morale) 1~100 (기본값 50, 단일 게이지)
│   └── modifier 합산식 (memory + situational + permanent) — RimWorld Mood 차용
├── Stamina (체력) + HP (전투 내구력)
├── 후천특성 최대 2개 (Lv5, Lv10 레벨업 시 행동 XP or 폭주 이력 기반)
├── 레벨 (1~10) + 경험치 (카테고리별 누적: 전투/방어/노동/건설/연구/사냥)
├── 7스탯 (힘/민첩/지능/건강/감각/통솔/매력) — 죄종 무관, 랜덤 배분
├── bonds (다른 영웅에 대한 호감도, asymmetric)
├── 외형: LPC 스프라이트 런타임 합성 (랜덤 파츠 조합)
└── 장비 3슬롯 — Phase 2
```

> **특성 3슬롯**: 시작특성 1 + 후천특성 2 = 최대 3개. 시작특성이 정체성 담당.
> **수식어 시스템 폐기** (2026-04-27) — 이름 단독 표시.

### 사기(Morale) 시스템 — RimWorld Mood 방식

영웅당 **단일 게이지 1~100** (기본 50). 임계값에 따라 폭주(낮을 때) / 구원(높을 때) 분기.

```
사기 100 ─┬─ [구원 영역] — 매 턴 구원 확률 체크 (RimWorld Inspiration)
        50 ─┼─ 정상
        0 ─┴─ [폭주 영역] — 매 턴 폭주 확률 체크 (RimWorld Mental Break)
```

- 사기 = `기본값 50 + Σ(활성 modifier.offset)`
- modifier 종류: **memory**(사건 후 시간 만료) / **situational**(조건 활성) / **permanent**(영구, 시작특성)
- 폭주 직후 Catharsis(+30 사기, 1턴) 자동 부여 — 무한 루프 방지

### 폭주 — 메인 죄종 × 시작특성 변주

사기가 폭주 임계 도달 시 매 턴 확률 발동. **메인 죄종**이 폭주 양상의 큰 틀을 결정.

| 메인 죄종 | 폭주 양상 |
|---------|-------------|
| 분노 | Berserk — 랜덤 영웅 사기 -10, 적 추격 강박 |
| 교만 | 명령 거부 (지시 무시), 단독 행동 |
| 탐욕 | 자원 횡령 (food/gold 차감), 창고 약탈 |
| 시기 | 사보타주, 타 영웅 사기 -8 |
| 폭식 | 추가 식량 소비, 폭식 |
| 나태 | 행동 거부, 주변 영웅 사기 -3 전파 |
| 색욕 | 집착 대상 분리 거부, 배치 거부 |

→ 시작특성에 따라 변주됨 (총 7×21 = 147 양상). 자세한 변주는 [hero_design.md](game_design/hero_design.md) 참조.

### 구원 — 메인 죄종별 보상

사기가 구원 임계 도달 시 매 턴 확률 발동. 발동 시 **메인 죄종별 구원 특성** 후천 슬롯에 부여.

| 메인 죄종 | 구원 보상 |
|---|---|
| 분노 → 평정 / 교만 → 현자 / 탐욕 → 청빈 / 시기 → 대범 / 색욕 → 순결 / 폭식 → 단식자 / 나태 → 명상가 |

### 연쇄 반응 (사기 modifier 전파)

한 영웅의 폭주 결과가 다른 영웅에게 사기 modifier로 전파되어 추가 폭주를 유발할 수 있음.

```
예시) 탐욕 영웅 폭주 → 자원 횡령 (식량 -)
  → 거점 식량 부족 → 영웅 전원에 "굶주림" -3 sit. modifier
    → bonds 높은 영웅의 동료 사망 등 다른 사건이 겹치면 사기 임계 진입
      → 폭주 추가 발동 (Catharsis로 무한 루프 방지)
```

- 연쇄 깊이: 한 턴 최대 `[balance.csv:max_chain_reactions]` 회

### 플레이어 행동 → 사기 modifier

플레이어의 경영 판단이 영웅 사기를 오르내리게 한다.

| 플레이어 행동 | 사기 modifier (예) |
|---|---|
| 원정 파견 | 영웅에게 "원정 출분" -3 (situational, 원정 동안) |
| idle 유지 | 영웅에게 "휴식" +2/턴 (situational) |
| 연회 개최 | 참석 영웅에게 "연회 참석" +6 (memory, 3턴) |
| 좋은 시설(숙소/식당) | situational +5 등 |
| 유리한 판결 | 영웅에게 "유리한 판결" +8 (memory, 5턴) |
| 불리한 판결 | 영웅에게 "불리한 판결" -8 (memory, 5턴) |

> 상세 설계: [hero_design.md](game_design/hero_design.md) §2-2 (사기 시스템) §4 (modifier 카탈로그)

## 5. 거점(림보) 시스템

### 거점 = 게임의 무대

바알이 운영하는 거점 "림보". 건설/연구/생산으로 거점을 성장시키고, 원정으로 챕터를 진행한다.

### 자원: 3종

| 자원 | 획득 방법 | 주요 사용처 |
|------|----------|------------|
| **식량** | 채집 (거점 주변, 밤 방어 O) | 영웅 유지비 (턴당 소비), 연회 |
| **나무** | 벌목 (거점 외곽, 밤 방어 X) | 건설 (전 티어) |
| **골드** | 원정 전리품, 방어전 보상, 시장, 교역, 사냥 | 연구, 장비/소모품 구매, 영웅 고용 |

### 건설 (HOMM3 방식, 턴당 1개)

3단계 건설 트리. 영웅 1명 투입, 스탯(힘+민첩)에 따라 속도 차이. 비용은 **나무**.

| Tier | 시설 |
|------|------|
| 1 | 주점(영웅 고용), 대장간, 감시탑, 농장(식량 자동), 벌목장(나무 자동) |
| 2 | 병원(치료), 역마차(교역), 훈련장(영웅 HP 회복 속도 +50%), 감시탑 Lv.2 |
| 3 | 연금술소, 시장(수입), 감시탑 Lv.3 |

### 연구 (턴당 1개 진행, 다턴 소요)

영웅 1명 투입. 2계열: 경제(수입/건설), 영웅(죄종 이해/성장).

### 플레이어 행동 (낮)

| 행동 | 설명 | 제한 | 밤 방어 |
|------|------|------|--------|
| **건설** | 시설 신축/업그레이드 (나무 소모) | 턴당 1개, 영웅 1명 | O |
| **연구** | 기술 연구 (골드 소모) | 턴당 1개 진행, 영웅 1명 | O |
| **생산** | 장비/소모품 제작 | 시설당 1개, 영웅 1명 | O |
| **채집** | 식량 획득 (거점 주변) | 영웅 1명 | O |
| **벌목** | 나무 획득 (거점 외곽) | 영웅 1명 | X |
| **사냥** | 야생 적과 1:1 전투 (맵 위 팝업) | 영웅 1명 | X |
| **원정 파견** | 챕터 스테이지로 출발 | 영웅 1~3명 | X |
| **교역** | 상인 방문 시 구매/판매 | 상인 방문 시만 | - |
| **영웅 고용** | 영웅 1명을 보내 후보 데려오기 | 턴당 1명, 영웅 1명 투입 + 골드 (매력→할인), 최대 7명 | O |
| **국시** | 7죄종 중 1개 국시 선포 (기간형 5턴) | 쿨다운 없으면 언제든, 거점 전반에 효과 적용 ([sin_system.md](game_design/sin_system.md) §5) | - |
| **연회** | 폭식/색욕 수치 상승, 폭주 위험 | 5턴에 1회, 주점 필요 | - |
| **치료** | 부상 치료 / HP 회복 | 병원 or 소모품 | - |

### 영웅 투입과 마을 안/밖

```
영웅 7명
  ├── 마을 안 (건설/연구/생산/채집/치료/휴식) → 밤 방어 O
  └── 마을 밖 (원정/벌목/사냥) → 밤 방어 X
```

## 6. 이벤트/선택지 시스템

### 이벤트 = 게임의 심장

매 턴 1~2개 이벤트 발생. 플레이어의 판결이 영웅 죄종 수치 + 거점 상태를 바꿈.

### 이벤트 유형

| 유형 | 트리거 | 예시 |
|------|--------|------|
| **영웅 간 갈등** | 메인 죄종 관계가 "대립"인 영웅이 같은 시설 배치 | [분노] vs [나태] 충돌 |
| **영웅 요청** | 사기 낮은 영웅 발생 | "원정 보내달라" ([분노] 메인 영웅 사기 회복 요청) |
| **폭주 대응** | 영웅 사기가 폭주 임계 도달 + 확률 발동 | [탐욕] 메인 영웅이 창고 자원 횡령 시도 |
| **외부 사건** | 턴 수/챕터 진행 기반 | 이웃 세력 발견, 상인 방문 |
| **밤 습격 후유증** | 밤 방어전 결과에 따라 | 부상자 발생, 시설 파손 |

### 선택지 구조

모든 선택지는 **강력하지만 대가가 있는** 설계:

```
[이벤트] [분노 메인 영웅 A]가 [나태 메인 영웅 B]를 때렸습니다. 훈련을 빠졌다는 이유로.

▶ A를 처벌한다
  → A 사기 -8 (memory 5턴), B 사기 +3 (처벌받은 측 인정)

▶ B에게 참으라 한다
  → B 사기 -5, A 사기 -3 (억눌린 분노 누적, 양쪽 다 불만)

▶ 둘 다 처벌한다
  → 둘 다 사기 -5, 단기 질서 회복 (전체 영웅 +2 "엄격한 통치")

▶ 무시한다
  → A·B 사기 -3, 다른 영웅 -1씩 ("무법지대" 인식)
```

> 상세 설계: [event_design.md](game_design/event_design.md)

## 8. 전투 시스템

### 기본 구조: X축 오토배틀 + 죄종 반응

- **영웅이 알아서 싸움** — 자동 이동, 자동 공격 (X축, east/west)
- **플레이어(바알)는 죄종 반응으로 개입** — 전투 중 영웅 죄종에 따른 이벤트 발생 시 선택
- **죄종 기반 일기토** — 전투 중 영웅 성격에 따라 1:1 일기토 자동 발동
- 7스탯 기반 (원정: 힘+민첩, 방어: 힘+통솔)
- **영웅 HP 직격**: 영웅이 피격 담당 (HP = 50 + 건강×5)
- **스프라이트**: LPC 런타임 합성 (영웅 생성 시 랜덤 외형)

### 죄종 반응 시스템 (카드 시스템 대체)

- 전투 중 영웅 죄종 조건 충족 시 **죄종 반응 이벤트** 자동 발동
- 플레이어가 반응에 대해 선택 (격려/제지/허가/금지)
- 일기토 신청도 플레이어 판단으로 전환 (자동 발동 → 허가/금지)
- 전투 전 **작전 지시** (공격적/방어적/균형) 선택으로 사전 전략

### 전투 유형

| | 원정 전투 | 방어 전투 (밤 습격) | 사냥 |
|---|---|---|---|
| 편성 | 영웅 1~3명 | 잔류 영웅 전원 | 영웅 1명 |
| 진행 | 실시간 오토배틀 (상세 기획 예정) | **맵 위 오버레이** (MapDefenseMode) | **맵 위 팝업** (MapHuntPopup) |
| 개입 | 죄종 반응 이벤트 | 죄종 반응 이벤트 | 불가 |
| 증원 | 불가 | **가능** (대기 영웅) | 불가 |
| 빈도 | 파견 시 | **3~5일 간격** | 낮 행동 |
| 핵심 | 파티 편성 + 작전 지시 | 증원 타이밍 + 죄종 반응 판단 | 영웅 개인 전력 |

핵심 딜레마: 원정에 보낸 **영웅**은 밤 방어에 못 씀. 전투로 HP가 깎인 영웅은 거점 행동/차기 원정이 제약됨.

### 전투 공식 (balance.csv 기반)

```
영웅 HP = 50 + 건강(vitality) × 5  (기본 10 → HP 100)
HP 자연 회복 = 매 턴 10 (병원 시설로 배율 가중)
HP 0 → knocked_out → 귀환 후 'injured' → 회복 턴 소요

ATK(원정) = 힘×0.7 + 민첩×0.4
ATK(방어) = 힘×0.8 + 통솔×0.4
데미지 = ATK × (0.8~1.2 랜덤) × 버프배율 × (1 - 피해경감)
```

### 사망 시스템 (TheSevenTactics 계승)

| 상태 | 조건 | 효과 |
|------|------|------|
| 기절 | HP 0 | 귀환 후 회복 필요 |
| 중상 | 기절 + 추가 피격 | 장기 치료 (수 턴 출전 불가) |
| 사망 | 중상 + 추가 피격 | **영구 상실** |

> 상세 설계: [battle_design.md](game_design/battle_design.md)

## 10. 개발 범위 (Phase 1)

### Phase 1 목표: 핵심 재미 검증 (웹 프로토타입, 2~3개월)

**포함**:
- 이벤트/선택지 → 영웅 사기 modifier 변동 반영
- 영웅 7명 (반고정 생성: 이름+메인 죄종+시작특성 고정, 스탯 랜덤, LPC 스프라이트 합성)
- **메인 죄종 1개 고정** (영웅별 1:1) + **사기(Morale) 1~100** 단일 게이지 + modifier 합산식 + 폭주/구원
- 시작특성 21종 (장점 전용, 영웅별 고정)

## 죄종 간 관계 매트릭스 (2026-04-24 재설계)

> 상세: [game_design/sin_system.md §3](game_design/sin_system.md) 참조

### 4가지 관계 유형 + bonds 배율

| 관계 | 배율 | 의미 |
|---|:-:|---|
| **동류** (같은 메인 죄종) | ×2 | 같은 죄를 짊어진 자끼리 — 가장 강한 결속 |
| **강화** | ×1.5 | 성향 궁합 — 서로를 북돋움 |
| **중립** | ×1 | 특별한 상호작용 없음 |
| **대립** | ×-2 | 성향 충돌 — 부호 반전 |

### 핵심 특징

- **교만 완전 고립** — 타 6개 죄종 모두 대립 ("모든 죄의 근원", 루시퍼 추락 모티프)
- **강화 3쌍**: 분노↔시기 / 탐욕↔색욕 / 폭식↔나태
- **대립 9쌍**: 교만↔타6 (6쌍) + 분노↔나태 / 탐욕↔폭식 / 시기↔색욕 (3쌍)
- **생성 관계 삭제** (기존 TheSevenTactics 매트릭스 대비)
- **방향성 없음** — 페어로 대칭 판정

### 2가지 작용

1. **bonds 변동 배율** — 사건 발생 시 두 영웅의 **메인 죄종**(고정)을 매트릭스에 대입해 변동량 배율 결정
2. **사기 modifier 가속/감속** — 같은 파티/시설 배치 시, 두 영웅의 메인 죄종이 대립이면 "껄끄러운 동거" -2 situational modifier, 강화면 "친구와 함께" +3
