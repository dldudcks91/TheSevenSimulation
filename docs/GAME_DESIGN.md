## 1. 게임 개요

- **장르**: 거점 경영 시뮬레이션 (로스터 관리 + 자동전투 원정)
- **테마**: 7대 죄악 + 중세 다크 판타지
- **배경**: Aurea — 신이 걸었던 땅, 7대 죄악이 지배하는 세계 (TheSevenTactics 계승)
- **핵심 재미**: 죄종 시스템이 만들어내는 **창발적 드라마** — 영웅 한 명의 폭주가 거점 전체를 뒤흔드는 연쇄 반응
- **컨셉**: "타락한 마왕이 결함 있는 인간들을 이끌고, 7대 죄악의 화신에게 복수한다"
- **스토리 전달**: 컷신이 아닌 **시뮬레이션 체감** — 챕터별 환경 변조 + 이벤트 + 보스 서사 ([chapter_scenario.md](game_design/chapter_scenario.md))
- **영웅 정체성**: 인간 (용병, 떠돌이, 버림받은 자) — 각자 성격적 결함(=죄종 태그)을 가진 결함 있는 인간들
- **플랫폼**: Phase 1 웹 프로토타입 (Phaser.js) → Phase 2 Godot(스팀)

### TheSevenTactics에서의 전환

| 항목 | TheSevenTactics | TheSevenSimulation |
|------|-----------------|-------------------|
| 장르 | 던전 크롤러 RPG | **거점 경영 시뮬레이션** |
| 핵심 | 던전 탐색 + 전투 | **거점 운영 + 죄종 드라마** |
| 전투 비중 | 높음 (직접 경험) | **낮음 (원정 파견 → 결과 확인)** |
| 죄종 시스템 | 컨셉 논의 중 | **핵심 시스템으로 완성** |
| 죄종 | 전투 패시브 + 만족/불만 | **성격 + 만족/불만 + 폭주/이탈 통합** |
| 타락 | 던전 내 7축 축적 | **거점 내 게이지(0~100) 1개** |
| 바알 | 지휘관 버프만 | **판결자 — 죄종 없이 경영 판단으로 영웅 관리** |
| 던전 | 방 탐색형 | **원정(자동 파견)** |

## 2. 핵심 게임 루프

### 하루(1턴) 구조

```
[아침] 소식 & 이벤트
  → 세계 소식, 영웅 간 사건, 죄종 관련 이벤트 발생
  → 플레이어 판결 → 영웅 사기 반응

[낮] 거점 운영 & 원정
  → 작업 지시: 건설 / 연구 / 생산 / 사냥 등 → 투입할 영웅 선택
  → 사냥: 영웅 1명 → 야생 적과 1:1 자동전투 (Loop Hero 스타일)
  → 원정 파견: 영웅 선택 + 병사 배정 → 챕터 스테이지로 출발
  → 마을 안 영웅 = 밤 방어 참여 O
  → 마을 밖 영웅(원정/사냥) = 밤 방어 참여 X
  ★ 핵심 딜레마: 영웅 + 병사를 어디에 얼마나 배분할 것인가

[저녁] 원정 결과 & 정보
  → 원정 결과 보고 (승리/패배, 전리품, 부상)
  → "더 진행시킬까, 귀환시킬까?" 판단
  → 감시탑 정보: 오늘 밤 습격 규모 예고 (감시탑 레벨에 따라 정보량 변동)

[밤] 적 습격 & 결산
  → ★ 적 습격 — 매일 발생, 규모 가변 (소규모 정찰 ~ 대규모 공성)
  → 마을 안 영웅 + 잔여 병사로 방어전 (완전 자동전투, 실시간 관전)
  → 영웅 간 죄종 상호작용 → 이벤트 발생
  → 플레이어 행동에 따른 사기 변화
  → 폭주/이탈 체크 → 연쇄 반응?
  → 다음 날로
```

### 플레이어 행동 요약

| 시간대 | 능동적 판단 |
|--------|-----------|
| 아침 | 이벤트 선택지 고르기 |
| 낮 | ① 작업 지시 (건설/연구/생산/사냥) + 투입 영웅 선택 ② 원정 파견: 영웅 선택 + 병사 배정 |
| 저녁 | 원정 계속/철수 판단 |
| 밤 | 관전 + **증원 판단** (대기 영웅 투입) |

### 마을 안/밖 구분

| 위치 | 행동 | 밤 방어 | 챕터 진행 |
|------|------|--------|----------|
| **마을 안** | 건설, 연구, 생산, 휴식 | O | X |
| **마을 밖** | 원정, 사냥 | X | O (원정만) |

### 메타 루프

```
[거점 성장] 건설 / 연구 / 생산 / 사냥 → 거점 강화
    ↓
[원정] 영웅 + 병사 파견 → 챕터/스테이지 진행 → 전리품
    ↓
[방어] 매일 밤 습격 → 잔류 영웅 + 잔여 병사로 자동 방어 + 증원 판단
    ↓
[드라마] 사기 변동, 폭주/이탈, 연쇄 반응
    ↓
반복 → 7챕터 클리어 → 엔딩
```

## 3. 바알(플레이어)의 역할

> 핵심 설계 원칙: **바알은 판결자** — 죄종 수치 없이, 순수하게 경영 판단으로 영웅들의 드라마를 이끈다

### 구조

바알(플레이어)은 죄종 수치를 가지지 않는다. 바알의 역할은 **판결**:
- 매 턴 발생하는 이벤트에 대해 선택지를 고른다
- 선택의 결과는 **영웅들의 사기**에 직접 반영된다
- 어떤 선택이든 트레이드오프가 있다 — 누군가를 만족시키면 다른 누군가가 불만

### 플레이어 선택이 게임에 미치는 영향

1. **영웅 사기 변동** — 선택에 따라 관련 영웅의 사기가 오르거나 내림
2. **연쇄 반응** — 사기 변동이 폭주/이탈로 이어지고, 또 다른 사건을 낳음
3. **거점 상태** — 시설 배치, 자원 배분 등 경영 판단의 직접적 결과
4. **원정 편성** — 누구를 보내고 누구를 남길지가 핵심 딜레마

## 4. 영웅 시스템

### 기본 구조

```
[영웅] — 랜덤 생성
├── 이름 (랜덤 생성)
├── 죄종 1개 (고정) — 성격 + 만족/불만 조건 + 폭주/이탈 행동 결정
├── 사기 (0~100) — 유일한 관리 수치
├── 7스탯 (힘/민첩/지능/체력/감각/통솔/매력, 각 1~20 랜덤, 죄종 무관)
├── 고유 스킬 1개
├── 특성풀 (고유 1 + 죄종 접사 2)
└── 장비 3슬롯 (자유 장착 — 아무 아이템이나 장착 가능)
```

### 사기 시스템 (핵심)

영웅당 수치 **딱 1개**: 사기 (0~100). 죄종은 고정 성격 태그이며, 사기가 극단에 갔을 때 **어떤 행동을 하는지**를 결정.

| 구간 | 상태 | 효과 |
|------|------|------|
| 0 | **이탈** | 거점을 떠남 (죄종에 따라 방식 다름) |
| 1~30 | 불만 | 능력 저하, 명령 거부 위험 |
| 31~70 | 안정 | 정상 운용 |
| 71~99 | 고양 | 능력 강화, 충성적 |
| 100 | **폭주** | 통제 불능 (죄종에 따라 사고 다름) |

### 폭주 (사기 100) — 너무 고양되어 폭주

| 죄종 | 폭주 이벤트 |
|------|-----------|
| 분노 | 가장 가까운 건물을 **부숨** |
| 교만 | 명령을 **전부 거부**, 독자 행동 |
| 탐욕 | 창고에서 자원을 **훔쳐서 도주** 시도 |
| 시기 | 가장 강한 영웅의 장비를 **파괴** |
| 폭식 | 식량 저장소를 **비움** |
| 나태 | **아무것도 안 함** + 주변 영웅 사기 저하 |
| 색욕 | 영웅 2명을 **분쟁에 휘말리게** 함 |

### 이탈 (사기 0) — 의욕을 잃고 떠남

| 죄종 | 이탈 방식 |
|------|---------|
| 분노 | 폭력적 탈영 |
| 교만 | "너 밑에 있을 수준이 아니다" 선언 후 이탈 |
| 탐욕 | 창고 털고 도주 |
| 시기 | 뒤에서 사보타주 후 이탈 |
| 폭식 | 보급품 갖고 이탈 |
| 나태 | 조용히 사라짐 |
| 색욕 | 다른 영웅 꼬여서 같이 떠남 |

### 폭주 → 타락 or 구원 (DD 고난/미덕 방식)

| 결과 | 확률 | 효과 |
|------|------|------|
| **타락** | 70% | 죄종 영구 강화. 더 강하지만 더 위험 |
| **구원** | 30% | 죄종 극복. 고유 능력 해금, 안정적 |

### 연쇄 반응 (이 게임의 핵심 재미)

```
탐욕 영웅이 자원을 훔침 (폭주)
  → 자원 부족 발생
    → 폭식 영웅 사기 급락 (불만)
      → 나태 영웅 사기 저하 (전염)
        → 분노 영웅 사기 급상승 (분노)
          → 분노 영웅도 폭주, 건물 파괴
            → ...
```

### 죄종별 만족/불만 조건

죄종이 영웅의 만족/불만 조건을 결정한다. 별도의 갈망 시스템 없이, 죄종 자체가 욕망이다.

| 죄종 | 만족 조건 (사기 상승) | 불만 트리거 (사기 하락) |
|------|-------------------|---------------------|
| 교만 | 보스 원정 성공 | 3연속 원정 실패 |
| 탐욕 | 자원 투자받음 | 자원 배분 안 됨 |
| 분노 | 원정 파견 | 3일 이상 미파견 |
| 시기 | 본인 레벨업 | 타 영웅이 더 많이 성장 |
| 색욕 | 선호 동료와 같은 시설 배치 | 선호 동료와 장기 분리 |
| 나태 | 시설 배치 없이 휴식 | 연속 배치 |
| 폭식 | 장비 풀장착 | 장비 슬롯 비어있음 |

### 플레이어 행동 → 사기 변동

플레이어의 경영 판단이 직접 사기를 올리거나 내린다:

| 플레이어 행동 | 사기 변동 |
|---|---|
| 죄종 만족 조건 충족 (원정 파견, 장비 지급 등) | 사기 상승 |
| 죄종 불만 트리거 발동 (미파견, 장비 미지급 등) | 사기 하락 |
| 방어전 승리 | 전체 사기 소폭 상승 |
| 방어전 패배 / 동료 사망 | 전체 사기 하락 |
| 특정 죄종에 유리한 판결 | 해당 죄종 영웅 사기 상승 |
| 특정 죄종에 불리한 판결 | 해당 죄종 영웅 사기 하락 |

> 상세 설계: [hero_design.md](game_design/hero_design.md)

## 5. 거점(림보) 시스템

### 거점 = 게임의 무대

바알이 운영하는 거점 "림보". 건설/연구/생산으로 거점을 성장시키고, 원정으로 챕터를 진행한다.

### 자원

- **골드** (단일 화폐): 수입(원정 전리품, 방어전 보상, 시장, 교역, 사냥) / 지출(건설, 연구, 영웅 영입, 병사 모집, 장비, 소모품, 포고령 유지비)
- **병사 풀**: 거점에 상주하는 민병. 출격 시 배정, 전사 시 영구 소실, 골드로 모집. 시작 시 100명

### 건설 (HOMM3 방식, 턴당 1개)

3단계 건설 트리. 영웅 1명 투입, 죄종에 따라 속도 차이.

| Tier | 시설 |
|------|------|
| 1 | 주점(영웅 고용), 훈련소, 대장간, 성벽, 감시탑 |
| 2 | 무기고, 성당, 역마차(교역), 성벽/감시탑 Lv.2 |
| 3 | 함정 공방, 연금술소, 시장(수입), 성벽/감시탑 Lv.3 |

### 연구 (턴당 1개 진행, 다턴 소요)

영웅 1명 투입. 4계열: 군사(ATK/DEF), 경제(수입), 방어(성벽/함정), 영웅(사기 관리).

### 플레이어 행동 (낮)

| 행동 | 설명 | 제한 |
|------|------|------|
| **건설** | 시설 신축/업그레이드 | 턴당 1개, 영웅 1명 |
| **연구** | 기술 연구 | 턴당 1개 진행, 영웅 1명 |
| **생산** | 장비/소모품/함정 제작 | 시설당 1개, 영웅 1명 |
| **사냥** | 야생 적과 1:1 전투 (Loop Hero 스타일) | 영웅 1명, 병사 불필요 |
| **원정 파견** | 챕터 스테이지로 출발 | 영웅 1~3명 + 병사 N명, 밤 방어 불가 |
| **교역** | 상인 방문 시 구매/판매 | 상인 방문 시만 |
| **영웅 고용** | 주점 후보 중 선택 | 골드 지불, 최대 7명 |
| **포고령** | 배급/훈련/경계 정책 변경 | 즉시 적용, 영웅 사기 영향 |
| **방어 준비** | 함정 배치, 성벽 수리 | 함정 공방 필요 |
| **연회** | 전체 사기 +15, 폭주 위험 | 5턴에 1회, 주점 필요 |
| **치료** | 사기 안정화 / 부상 치료 | 성당 or 소모품 |
| **장비 관리** | 장착/해제/수리 | 제한 없음 |

### 영웅 투입과 마을 안/밖

```
영웅 7명
  ├── 마을 안 (건설/연구/생산/치료/휴식) → 밤 방어 O
  └── 마을 밖 (원정/사냥) → 밤 방어 X

병사 풀 N명
  ├── 원정 배정 → 원정 파티에 합류
  └── 잔여 → 밤 방어에 자동 투입
```

> 상세 설계: [base_design.md](game_design/base_design.md)

## 6. 이벤트/선택지 시스템

### 이벤트 = 게임의 심장

매 턴 1~2개 이벤트 발생. 플레이어의 판결이 영웅 사기 + 거점 상태를 바꿈.

### 이벤트 유형

| 유형 | 트리거 | 예시 |
|------|--------|------|
| **영웅 간 갈등** | 죄종 관계가 "대립"인 영웅이 같은 시설 배치 | [분노] vs [나태] 충돌 |
| **영웅 요청** | 죄종 불만 누적 | "원정 보내달라" ([분노] 욕구) |
| **폭주 대응** | 영웅 사기 100 도달 | [탐욕]이 창고 털기 시도 |
| **외부 사건** | 턴 수/챕터 진행 기반 | 이웃 세력 발견, 상인 방문 |
| **밤 습격 후유증** | 밤 방어전 결과에 따라 | 부상자 발생, 시설 파손 |

### 선택지 구조

모든 선택지는 **강력하지만 대가가 있는** 설계:

```
[이벤트] [분노]가 [나태]를 때렸습니다. 훈련을 빠졌다는 이유로.

▶ [분노]를 처벌한다
  → [분노] 사기 -20, [나태] 사기 +10

▶ [나태]에게 참으라 한다
  → [나태] 사기 -15, [분노] 사기 +5

▶ 둘 다 처벌한다
  → 둘 다 사기 -15, 전체 영웅 단기 안정

▶ 무시한다
  → 전체 사기 -5 ("무법지대" 인식)
```

> 상세 설계: [event_design.md](game_design/event_design.md)

## 7. 원정 시스템

### 구조

던전 탐색이 아닌 **파견 방식**. 파티를 보내고 결과를 받는다.

```
파티 편성 (3명) → 목표 선택 (챕터/스테이지) → 파견
  → [낮 동안 자동 진행]
  → 저녁에 결과 보고
    → 전리품, 부상, 경험치
    → "더 진행시킬까, 귀환시킬까?" 판단
```

### 7챕터 원정 (TheSevenTactics 계승)

| 챕터 | 죄종 | 지역 | 보스 | 스테이지 |
|------|------|------|------|---------|
| 1 | 분노 | 불타는 전장 | 사탄 | 3 + 보스 |
| 2 | 시기 | 뒤틀린 숲 | 레비아탄 | 3 + 보스 |
| 3 | 탐욕 | 황금의 사막 | 맘몬 | 3 + 보스 |
| 4 | 나태 | 망각의 동토 | 벨페고르 | 3 + 보스 |
| 5 | 폭식 | 심연의 동굴 | 바알제붑 | 3 + 보스 |
| 6 | 색욕 | 타락한 궁전 | 아스모데우스 | 3 + 보스 |
| 7 | 교만 | 신의 폐허 | 루시퍼 | 3 + 보스 |

## 8. 전투 시스템

### 기본 구조

- **영웅 + 병사 자동전투**: 영웅 1~3명 + 병사 N명 vs 적 다수
- **하이브리드 자동전투**: 기본 자동 + **증원 개입** (대기 영웅 목록에서 선택 투입)
- 7스탯 기반 자동 계산 (원정: 힘+민첩, 방어: 힘+통솔)
- 명중/회피 없음, 마나 없음
- **병사 풀**: 거점에 병사 풀 존재, 출격 시 배정. 적은 병사만 공격 가능, 병사 전멸 시 영웅 이탈
- **해상도**: 1280x720
- **스프라이트**: LPC 스프라이트 (4캐릭터 × 4액션)

### 전투씬 A/B 분리

전투 유형에 따라 두 가지 씬으로 분리:

| | BattleSceneA (돌진형) | BattleSceneB (필드 이동형) |
|---|---|---|
| 구조 | 양측이 직선으로 돌진하여 충돌 | 필드 위 이동 + 전투 |
| 적합 유형 | 방어 전투 (밤 습격), 사냥 | 원정 전투 |
| 특징 | 빠른 전개, 웨이브 기반 | 넓은 필드, 탐색 요소 |

### 증원 시스템

전투 중 플레이어가 **대기 영웅 목록**에서 영웅을 선택하여 전장에 투입할 수 있다.

- **대기 영웅**: 마을 안에 잔류 중이며 전투에 미참여 상태인 영웅
- **투입 타이밍**: 전투 진행 중 언제든 (실시간)
- **투입 제한**: 마을 밖 영웅(원정/사냥 중)은 증원 불가
- **핵심 판단**: 지금 투입할 것인가, 다음 전투를 위해 아낄 것인가

### 시뮬레이션에서의 전투 위치

전투는 **하이브리드 자동전투** — 기본 자동 진행 + 증원 개입. 플레이어 판단은 전투 전(편성, 장비, **병사 배분**), 전투 중(**증원 투입**), 전투 후(계속/철수)에 걸쳐 분포.

| | 원정 전투 | 방어 전투 (밤 습격) | 사냥 |
|---|---|---|---|
| 편성 | 영웅 1~3명 + 병사 N명 | 잔류 영웅 + 잔여 병사 | 영웅 1명 (병사 없음) |
| 전투씬 | BattleSceneB (필드 이동형) | BattleSceneA (돌진형) | BattleSceneA (돌진형) |
| 진행 | 파견 후 결과만 받음 | **실시간으로 지켜봄** | 실시간 1:1 전투 |
| 증원 | 불가 (원정 중) | **가능** (대기 영웅 투입) | 불가 (1:1 전투) |
| 리플레이 | 결과 화면에서 다시 볼 수 있음 | - | - |
| 조작 | 없음 (자동) | **증원 판단** | 없음 (자동) |

핵심 딜레마: 원정에 보낸 **영웅 + 병사**는 밤 방어에 못 씀. 밤 방어 중 증원을 보내면 다음 습격에 대비할 여력이 줄어듦.

### 사망 시스템 (TheSevenTactics 계승)

| 상태 | 조건 | 효과 |
|------|------|------|
| 기절 | HP 0 | 귀환 후 회복 필요 |
| 중상 | 기절 + 추가 피격 | 장기 치료 (수 턴 출전 불가) |
| 사망 | 중상 + 추가 피격 | **영구 상실** |

> 상세 설계: [battle_design.md](game_design/battle_design.md)

## 죄종 간 관계 매트릭스 (TheSevenTactics 계승)

```
         교만  시기  분노  나태  탐욕  폭식  색욕
교만      ──   생성  강화   대립  강화   ·    강화
시기     전제   ──   생성   ·    강화   ·     ·
분노     강화  강화   ──   대립   강화   ·     ·
나태     대립   ·   대립   ──    대립  강화   대립
탐욕     강화  강화  강화   대립   ──   대립   강화
폭식      ·    ·    ·    강화   대립   ──   생성
색욕     강화   ·    ·    대립   강화  생성    ──
```

- **생성**: 앞의 죄가 뒤의 죄를 직접 낳음 (가장 강한 시너지)
- **강화**: 합치면 더 위험 (시너지)
- **대립**: 표면적 모순 (갈등, 단 극복 시 강한 보너스)
- **중립(·)**: 직접적 관계 없음
