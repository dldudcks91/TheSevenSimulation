## 1. 게임 개요

- **장르**: 거점 경영 시뮬레이션 (로스터 관리 + 자동전투 원정)
- **테마**: 7대 죄악 + 중세 다크 판타지
- **배경**: Aurea — 신이 걸었던 땅, 7대 죄악이 지배하는 세계 (TheSevenTactics 계승)
- **핵심 재미**: 죄종 시스템이 만들어내는 **창발적 드라마** — 영웅 한 명의 폭주가 거점 전체를 뒤흔드는 연쇄 반응
- **컨셉**: "타락한 마왕이 결함 있는 인간들을 이끌고, 7대 죄악의 화신에게 복수한다"
- **스토리 전달**: 컷신이 아닌 **시뮬레이션 체감** — 챕터별 환경 변조 + 이벤트 + 보스 서사 ([chapter_scenario.md](game_design/chapter_scenario.md))
- **영웅 정체성**: 인간 (용병, 떠돌이, 버림받은 자) — 각자 성격적 결함(=죄종 태그)을 가진 결함 있는 인간들
- **플랫폼**: Phase 1 웹 프로토타입 (Phaser.js) → Phase 2 Godot(스팀)

### TheSevenTactics에서의 전환

| 항목 | TheSevenTactics | TheSevenSimulation |
|------|-----------------|-------------------|
| 장르 | 던전 크롤러 RPG | **거점 경영 시뮬레이션** |
| 핵심 | 던전 탐색 + 전투 | **거점 운영 + 죄종 드라마** |
| 전투 비중 | 높음 (직접 경험) | **낮음 (원정 파견 → 결과 확인)** |
| 죄종 시스템 | 컨셉 논의 중 | **핵심 시스템으로 완성** |
| 죄종 | 전투 패시브 + 만족/불만 | **성격 + 만족/불만 + 폭주/이탈 통합** |
| 타락 | 던전 내 7축 축적 | **거점 내 게이지(0~100) 1개** |
| 바알 | 지휘관 버프만 | **판결자 — 죄종 없이 경영 판단으로 영웅 관리** |
| 던전 | 방 탐색형 | **원정(자동 파견)** |

## 2. 핵심 게임 루프

### 하루(1턴) 구조

```
[아침] 소식 & 이벤트
  → 세계 소식, 영웅 간 사건, 죄종 관련 이벤트 발생
  → 플레이어 판결 → 영웅 사기 반응

[낮] 거점 운영 & 원정
  → 작업 지시: 건설 / 연구 / 생산 / 사냥 / 개척 등 → 투입할 영웅 선택
  → 채집/벌목: 식량/나무 획득 (맵 위 행동 팝업 — 프로그레스 + 결과 + 죄종 대사)
  → 사냥: 영웅 1명 → 야생 적과 1:1 전투 (맵 위 팝업)
  → 원정 파견: 영웅 선택 → 챕터 스테이지로 출발
  → ★ 방어 배치: 밤 습격에 참전할 영웅을 명시적으로 배치
  → 방어에 배치하지 않은 영웅은 밤 방어에 참여 불가
  ★ 핵심 딜레마: 영웅을 경제(건설/채집/사냥) vs 진행(원정) vs 생존(방어)에 어떻게 배분?

[저녁] 원정 결과 & 정보
  → 원정 결과 보고 (승리/패배, 전리품, 부상)
  → "더 진행시킬까, 귀환시킬까?" 판단
  → 감시탑 정보: 오늘 밤 습격 규모 예고 (감시탑 레벨에 따라 정보량 변동)

[밤] 적 습격 & 결산
  → ★ 적 습격 — 3~5일 간격, 규모 가변 (소규모 정찰 ~ 대규모 공성)
  → 습격 있는 밤: 맵 위 오버레이로 방어전 (오토배틀 + SP/카드, 실시간 관전)
  → 습격 없는 밤: 평화롭게 결산으로 넘어감
  → 영웅 간 죄종 상호작용 → 이벤트 발생
  → 플레이어 행동에 따른 사기 변화
  → 폭주/이탈 체크 → 연쇄 반응?
  → 다음 날로
```

### 플레이어 행동 요약

| 시간대 | 능동적 판단 |
|--------|-----------|
| 아침 | 이벤트 선택지 고르기 |
| 낮 | ① 작업 지시 (건설/연구/생산/사냥/개척) + 투입 영웅 선택 ② 원정 파견 ③ **방어 배치** |
| 저녁 | 원정 계속/철수 판단 |
| 밤 | 습격 시 관전 + 카드 사용 (방어 배치 영웅만 참전) |

### 마을 안/밖 구분

| 위치 | 행동 | 밤 방어 | 챕터 진행 |
|------|------|--------|----------|
| **마을 안** | 건설, 연구, 생산, 채집, 휴식 | O | X |
| **마을 밖** | 원정, 벌목, 사냥 | X | O (원정만) |

### 메타 루프

```
[거점 성장] 건설 / 연구 / 생산 / 사냥 → 거점 강화
    ↓
[원정] 영웅 + 병사 파견 → 챕터/스테이지 진행 → 전리품
    ↓
[방어] 3~5일 간격 습격 → 잔류 영웅 + 잔여 병사로 맵 위 방어전 + 카드/증원
    ↓
[드라마] 사기 변동, 폭주/이탈, 연쇄 반응
    ↓
반복 → 7챕터 클리어 → 엔딩
```

## 3. 바알(플레이어)의 역할

> 핵심 설계 원칙: **바알은 판결자** — 죄종 수치 없이, 순수하게 경영 판단으로 영웅들의 드라마를 이끈다

### 구조

바알(플레이어)은 죄종 수치를 가지지 않는다. 바알의 역할은 **판결**:
- 매 턴 발생하는 이벤트에 대해 선택지를 고른다
- 선택의 결과는 **영웅들의 사기**에 직접 반영된다
- 어떤 선택이든 트레이드오프가 있다 — 누군가를 만족시키면 다른 누군가가 불만

### 플레이어 선택이 게임에 미치는 영향

1. **영웅 사기 변동** — 선택에 따라 관련 영웅의 사기가 오르거나 내림
2. **연쇄 반응** — 사기 변동이 폭주/이탈로 이어지고, 또 다른 사건을 낳음
3. **거점 상태** — 시설 배치, 자원 배분 등 경영 판단의 직접적 결과
4. **원정 편성** — 누구를 보내고 누구를 남길지가 핵심 딜레마

## 4. 영웅 시스템

### 기본 구조

```
[영웅] — 랜덤 생성
├── 이름 = "수식어" + 이름 (죄종 수치 상위 2개 조합 → 21종 수식어 풀)
├── 죄종 7수치 (분노/시기/탐욕/나태/폭식/색욕/교만, 각 1~20) — 성격 차원
│   └── 최고값 = 주 성향(primarySin) → 폭주/이탈/만족/불만 결정
├── 선천 특성 1개 (traits.csv 50개 풀에서 랜덤) — 행동 변주, 양면성
├── 후천 특성 최대 2개 (Lv5, Lv10 레벨업 시 행동 XP or 폭주 이력 기반)
├── 레벨 (1~10) + 경험치 (카테고리별 누적: 전투/방어/노동/건설/연구/사냥)
├── 사기 (0~100) — 유일한 관리 수치
├── 7스탯 (힘/민첩/지능/건강/감각/통솔/매력)
│   └── 총합 60~80, 최소 2, 최대 18 (죄종 무관, 랜덤 배분)
├── 외형: LPC 스프라이트 런타임 합성 (랜덤 파츠 조합)
├── 고유 스킬 1개
└── 장비 3슬롯 — Phase 2
```

### 사기 시스템 (핵심)

영웅당 수치 **딱 1개**: 사기 (0~100). 죄종 7수치의 최고값(주 성향)이 사기 극단에서 **어떤 행동을 하는지**를 결정. 선천 특성이 사기 변동을 변조.

| 구간 | 상태 | 효과 |
|------|------|------|
| 0 | **이탈** | 거점을 떠남 (주 성향에 따라 방식 다름) |
| 1~30 | 불만 | 능력 저하, 명령 거부 위험 |
| 31~70 | 안정 | 정상 운용 |
| 71~99 | 고양 | 능력 강화, 충성적 |
| 100 | **폭주** | 통제 불능 (주 성향에 따라 사고 다름) |

### 폭주 (사기 100) — 너무 고양되어 폭주

주 성향(죄종 수치 최고값)이 폭주 행동을 결정:

| 주 성향 | 폭주 이벤트 |
|---------|-----------|
| 분노 | 가장 가까운 건물을 **부숨** |
| 교만 | 명령을 **전부 거부**, 독자 행동 |
| 탐욕 | 창고에서 자원을 **훔쳐서 도주** 시도 |
| 시기 | 가장 강한 영웅의 장비를 **파괴** |
| 폭식 | 식량 저장소를 **비움** |
| 나태 | **아무것도 안 함** + 주변 영웅 사기 저하 |
| 색욕 | 영웅 2명을 **분쟁에 휘말리게** 함 |

### 이탈 (사기 0) — 의욕을 잃고 떠남

| 주 성향 | 이탈 방식 |
|---------|---------|
| 분노 | 폭력적 탈영 |
| 교만 | "너 밑에 있을 수준이 아니다" 선언 후 이탈 |
| 탐욕 | 창고 털고 도주 |
| 시기 | 뒤에서 사보타주 후 이탈 |
| 폭식 | 보급품 갖고 이탈 |
| 나태 | 조용히 사라짐 |
| 색욕 | 다른 영웅 꼬여서 같이 떠남 |

### 폭주 → 타락 or 구원 (DD 고난/미덕 방식)

| 결과 | 확률 | 효과 |
|------|------|------|
| **타락** | 70% | 죄종 영구 강화. 더 강하지만 더 위험 |
| **구원** | 30% | 죄종 극복. 고유 능력 해금, 안정적 |

### 연쇄 반응 (이 게임의 핵심 재미)

```
탐욕 영웅이 자원을 훔침 (폭주)
  → 자원 부족 발생
    → 폭식 영웅 사기 급락 (불만)
      → 나태 영웅 사기 저하 (전염)
        → 분노 영웅 사기 급상승 (분노)
          → 분노 영웅도 폭주, 건물 파괴
            → ...
```

### 주 성향별 만족/불만 조건

주 성향(죄종 수치 최고값)이 만족/불만 조건의 기본 틀을 결정한다. 부 성향도 약한 강도로 영향.

| 주 성향 | 만족 조건 (사기 상승) | 불만 트리거 (사기 하락) |
|---------|-------------------|---------------------|
| 교만 | 보스 원정 성공 | 3연속 원정 실패 |
| 탐욕 | 자원 투자받음 | 자원 배분 안 됨 |
| 분노 | 원정 파견 | 3일 이상 미파견 |
| 시기 | 본인 레벨업 | 타 영웅이 더 많이 성장 |
| 색욕 | 선호 동료와 같은 시설 배치 | 선호 동료와 장기 분리 |
| 나태 | 시설 배치 없이 휴식 | 연속 배치 |
| 폭식 | 장비 풀장착 | 장비 슬롯 비어있음 |

### 플레이어 행동 → 사기 변동

플레이어의 경영 판단이 직접 사기를 올리거나 내린다. **선천 특성**이 사기 변동폭을 변조.

| 플레이어 행동 | 사기 변동 |
|---|---|
| 주 성향 만족 조건 충족 (원정 파견, 장비 지급 등) | 사기 상승 (죄종 수치에 비례) |
| 주 성향 불만 트리거 발동 (미파견, 장비 미지급 등) | 사기 하락 (죄종 수치에 비례) |
| 방어전 승리 | 전체 사기 소폭 상승 |
| 방어전 패배 / 동료 사망 | 전체 사기 하락 |
| 특정 성향에 유리한 판결 | 해당 성향 영웅 사기 상승 |
| 특성에 의한 변조 | 낙천적(-30% 하락), 변덕쟁이(±50%) 등 |

## 5. 거점(림보) 시스템

### 거점 = 게임의 무대

바알이 운영하는 거점 "림보". 건설/연구/생산으로 거점을 성장시키고, 원정으로 챕터를 진행한다.

### 자원: 3종 + 병사

| 자원 | 획득 방법 | 주요 사용처 |
|------|----------|------------|
| **식량** | 채집 (거점 주변, 밤 방어 O) | 영웅 유지비 (턴당 소비), 배급 포고령, 연회 |
| **나무** | 벌목 (거점 외곽, 밤 방어 X) | 건설 (전 티어) |
| **골드** | 원정 전리품, 방어전 보상, 시장, 교역, 사냥 | 연구, 장비/소모품 구매, 영웅 고용, 병사 모집 |

- **병사 풀**: 거점에 상주하는 민병. 출격 시 배정, 전사 시 영구 소실, 골드로 모집. 시작 시 100명

### 건설 (HOMM3 방식, 턴당 1개)

3단계 건설 트리. 영웅 1명 투입, 스탯(힘+민첩)에 따라 속도 차이. 비용은 **나무**.

| Tier | 시설 |
|------|------|
| 1 | 주점(영웅 고용), 대장간, 감시탑, 농장(식량 자동), 벌목장(나무 자동) |
| 2 | 병원(치료/사기안정), 역마차(교역), 훈련장(병사 생산), 감시탑 Lv.2 |
| 3 | 연금술소, 시장(수입), 감시탑 Lv.3 |

### 연구 (턴당 1개 진행, 다턴 소요)

영웅 1명 투입. 2계열: 경제(수입/건설), 영웅(사기 관리/죄종 이해).

### 플레이어 행동 (낮)

| 행동 | 설명 | 제한 | 밤 방어 |
|------|------|------|--------|
| **건설** | 시설 신축/업그레이드 (나무 소모) | 턴당 1개, 영웅 1명 | O |
| **연구** | 기술 연구 (골드 소모) | 턴당 1개 진행, 영웅 1명 | O |
| **생산** | 장비/소모품 제작 | 시설당 1개, 영웅 1명 | O |
| **채집** | 식량 획득 (거점 주변) | 영웅 1명 | O |
| **벌목** | 나무 획득 (거점 외곽) | 영웅 1명 | X |
| **사냥** | 야생 적과 1:1 전투 (맵 위 팝업) | 영웅 1명 | X |
| **원정 파견** | 챕터 스테이지로 출발 | 영웅 1~3명 + 병사 N명 | X |
| **교역** | 상인 방문 시 구매/판매 | 상인 방문 시만 | - |
| **영웅 고용** | 영웅 1명을 보내 후보 데려오기 | 턴당 1명, 영웅 1명 투입 + 골드 (매력→할인), 최대 7명 | O |
| **포고령** | 배급/훈련/경계 정책 변경 | 즉시 적용, 영웅 사기 영향 | - |
| **연회** | 전체 사기 +15, 폭주 위험 | 5턴에 1회, 주점 필요 | - |
| **치료** | 사기 안정화 / 부상 치료 | 병원 or 소모품 | - |

### 영웅 투입과 마을 안/밖

```
영웅 7명
  ├── 마을 안 (건설/연구/생산/채집/치료/휴식) → 밤 방어 O
  └── 마을 밖 (원정/벌목/사냥) → 밤 방어 X

병사 풀 N명
  ├── 원정 배정 → 원정 파티에 합류
  └── 잔여 → 밤 방어에 자동 투입
```

> 상세 설계: [base_design.md](game_design/base_design.md)

## 6. 이벤트/선택지 시스템

### 이벤트 = 게임의 심장

매 턴 1~2개 이벤트 발생. 플레이어의 판결이 영웅 사기 + 거점 상태를 바꿈.

### 이벤트 유형

| 유형 | 트리거 | 예시 |
|------|--------|------|
| **영웅 간 갈등** | 죄종 관계가 "대립"인 영웅이 같은 시설 배치 | [분노] vs [나태] 충돌 |
| **영웅 요청** | 죄종 불만 누적 | "원정 보내달라" ([분노] 욕구) |
| **폭주 대응** | 영웅 사기 100 도달 | [탐욕]이 창고 털기 시도 |
| **외부 사건** | 턴 수/챕터 진행 기반 | 이웃 세력 발견, 상인 방문 |
| **밤 습격 후유증** | 밤 방어전 결과에 따라 | 부상자 발생, 시설 파손 |

### 선택지 구조

모든 선택지는 **강력하지만 대가가 있는** 설계:

```
[이벤트] [분노]가 [나태]를 때렸습니다. 훈련을 빠졌다는 이유로.

▶ [분노]를 처벌한다
  → [분노] 사기 -20, [나태] 사기 +10

▶ [나태]에게 참으라 한다
  → [나태] 사기 -15, [분노] 사기 +5

▶ 둘 다 처벌한다
  → 둘 다 사기 -15, 전체 영웅 단기 안정

▶ 무시한다
  → 전체 사기 -5 ("무법지대" 인식)
```

> 상세 설계: [event_design.md](game_design/event_design.md)

## 7. 원정 시스템

### 구조

던전 탐색이 아닌 **파견 방식**. 파티를 보내고 결과를 받는다.

```
파티 편성 (3명) → 목표 선택 (챕터/스테이지) → 파견
  → [낮 동안 자동 진행]
  → 저녁에 결과 보고
    → 전리품, 부상, 경험치
    → "더 진행시킬까, 귀환시킬까?" 판단
```

### 7챕터 원정 (TheSevenTactics 계승)

| 챕터 | 죄종 | 지역 | 보스 | 스테이지 |
|------|------|------|------|---------|
| 1 | 분노 | 불타는 전장 | 사탄 | 3 + 보스 |
| 2 | 시기 | 뒤틀린 숲 | 레비아탄 | 3 + 보스 |
| 3 | 탐욕 | 황금의 사막 | 맘몬 | 3 + 보스 |
| 4 | 나태 | 망각의 동토 | 벨페고르 | 3 + 보스 |
| 5 | 폭식 | 심연의 동굴 | 바알제붑 | 3 + 보스 |
| 6 | 색욕 | 타락한 궁전 | 아스모데우스 | 3 + 보스 |
| 7 | 교만 | 신의 폐허 | 루시퍼 | 3 + 보스 |

## 8. 전투 시스템

### 기본 구조: X축 오토배틀 + 카드

- **영웅이 알아서 싸움** — 자동 이동, 자동 공격 (X축, east/west)
- **플레이어(바알)는 카드로 개입** — SP 게이지 모아서 전체 버프/디버프/회복 카드 사용
- **죄종 기반 일기토** — 전투 중 영웅 성격에 따라 1:1 일기토 자동 발동
- 7스탯 기반 (원정: 힘+민첩, 방어: 힘+통솔)
- **병사 풀**: 카운터 표시, 적은 병사 우선 공격
- **스프라이트**: LPC 런타임 합성 (영웅 생성 시 랜덤 외형)

### 카드 시스템

- 거점에서 카드 획득/제작 → 원정/방어전 전에 **덱 구성** (4~5장)
- 전투 중 SP 충족 시 카드 터치하여 사용
- 카드는 **소모품** — 원정 연속 스테이지 시 잔여 카드 관리가 전략
- **죄종 시너지**: 파티 내 해당 죄종 영웅이 있으면 카드 효과 강화

### 전투 유형

| | 원정 전투 | 방어 전투 (밤 습격) | 사냥 |
|---|---|---|---|
| 편성 | 영웅 1~3명 + 병사 N명 | 잔류 영웅 + 잔여 병사 | 영웅 1명 (병사 없음) |
| 진행 | 실시간 오토배틀 (상세 기획 예정) | **맵 위 오버레이** (MapDefenseMode) | **맵 위 팝업** (MapHuntPopup) |
| 카드 | 사용 가능 (SP/카드) | 사용 가능 (SP/카드) | 불가 |
| 증원 | 불가 | **가능** (대기 영웅) | 불가 |
| 빈도 | 파견 시 | **3~5일 간격** | 낮 행동 |
| 핵심 | 덱 구성 + 파티 편성 + 병사 배분 | 증원 타이밍 + 카드 타이밍 | 영웅 개인 전력 |

핵심 딜레마: 원정에 보낸 **영웅 + 병사**는 밤 방어에 못 씀 + 카드를 언제 쓸 것인가

### 전투 공식 (balance.csv 기반)

```
HP = 200 + 힘×10 (병사가 먼저 피격, 건강은 스태미나 전용)
ATK(원정) = 힘×0.7 + 민첩×0.4
ATK(방어) = 힘×0.8 + 통솔×0.4
데미지 = ATK × (0.8~1.2 랜덤) × 버프배율 × (1 - 피해경감)
SP: 매 틱 +2, 적 처치 시 +10, 최대 100
```

### 사망 시스템 (TheSevenTactics 계승)

| 상태 | 조건 | 효과 |
|------|------|------|
| 기절 | HP 0 | 귀환 후 회복 필요 |
| 중상 | 기절 + 추가 피격 | 장기 치료 (수 턴 출전 불가) |
| 사망 | 중상 + 추가 피격 | **영구 상실** |

> 상세 설계: [battle_design.md](game_design/battle_design.md)

## 9. 엔딩 시스템

> 상세 미확정 — 스토리 설계 시 구체화 예정.

### 방향 (확정)
- **숨겨진 7죄종 수치**: 플레이어의 선택이 거점(림보)에 7개 죄종 수치를 누적
- **비공개**: 수치는 플레이어에게 직접 보여주지 않음
- **분위기 힌트**: 챕터 정산 시 "림보에 ○○의 기운이 짙다" 등 텍스트로 방향성만 암시
- **지배적 죄종 → 엔딩 분기**: 게임 종료 시 가장 높은 죄종이 엔딩을 결정 (7개 엔딩)
- 바알 개인의 수치가 아니라 **거점 운영 방침의 결과**로서 누적

### 미정 (스토리 설계 시 결정)
- 각 죄종 엔딩의 구체적 내용
- 선택지별 죄종 태그 배분
- 힌트 텍스트 목록
- 복수 엔딩(Ending A) / 끝없는 전쟁(Ending B)과의 통합 방식

## 죄종 간 관계 매트릭스 (TheSevenTactics 계승)

```
         교만  시기  분노  나태  탐욕  폭식  색욕
교만      ──   생성  강화   대립  강화   ·    강화
시기     전제   ──   생성   ·    강화   ·     ·
분노     강화  강화   ──   대립   강화   ·     ·
나태     대립   ·   대립   ──    대립  강화   대립
탐욕     강화  강화  강화   대립   ──   대립   강화
폭식      ·    ·    ·    강화   대립   ──   생성
색욕     강화   ·    ·    대립   강화  생성    ──
```

- **생성**: 앞의 죄가 뒤의 죄를 직접 낳음 (가장 강한 시너지)
- **강화**: 합치면 더 위험 (시너지)
- **대립**: 표면적 모순 (갈등, 단 극복 시 강한 보너스)
- **중립(·)**: 직접적 관계 없음
