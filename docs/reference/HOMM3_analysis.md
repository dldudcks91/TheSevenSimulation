# Heroes of Might and Magic 3 — 시스템 분석

> 레퍼런스 목적: TheSevenTactics 기획 참고용
> 최종 업데이트: 2026-03-23

## 1. 영웅 시스템

### 영웅의 역할
HOMM3에서 영웅은 **직접 전투하지 않는 지휘관**이다. 맵에서 군대를 이끌고 다니며, 전투에서는 뒤에서 스킬/마법으로 지원하는 구조.

### 영웅 클래스 (2대 계열, 16종)
| 계열 | 클래스 | 특징 |
|------|--------|------|
| **Might** (전사형) | Knight, Barbarian, Battle Mage 등 | 공격/방어 스탯 성장 우수, 전투 스킬 특화 |
| **Magic** (마법형) | Cleric, Wizard, Necromancer 등 | 지식/마력 스탯 성장 우수, 마법 특화 |

### 영웅 4대 스탯
- **Attack** — 아군 유닛의 데미지 보너스 (영웅 공격력이 유닛에 전이)
- **Defense** — 아군 유닛의 피해 감소
- **Spell Power** — 마법 효과/데미지 증폭
- **Knowledge** — 마나 풀 결정 (Knowledge × 10 = 최대 마나)

핵심: **영웅 스탯이 유닛 전체에 적용**된다. 영웅 Attack +1이면 모든 아군 유닛 데미지가 올라감.

### 레벨업 구조
- 경험치 획득 → 레벨업 시 **스탯 1 증가 + 스킬 선택**
- 스킬은 2개 중 1개 선택 (랜덤 제시)
- 클래스별 스킬 출현 확률이 다름 (가중치 테이블)

### 스킬 시스템
- 총 **28개 보조 스킬** 존재
- 각 스킬 3단계: Basic → Advanced → Expert
- 영웅당 최대 **8개 스킬 슬롯**
- 한 번 찍으면 **교체 불가** (잘못 찍으면 영구적 손해)

**주요 스킬 예시:**

| 스킬 | 효과 | 평가 |
|------|------|------|
| Logistics | 맵 이동력 증가 | S급 (필수) |
| Earth Magic | 땅 마법 강화 (Town Portal 등) | S급 |
| Wisdom | 3레벨 이상 마법 학습 가능 | 마법형 필수 |
| Necromancy | 전투 후 적 사체를 스켈레톤으로 부활 | 넥로 전용 사기 스킬 |
| Eagle Eye | 적이 쓴 마법을 배움 | F급 (쓰레기) |
| Learning | 경험치 보너스 | F급 |

→ 스킬 간 **밸런스가 극심하게 안 맞음**. 이게 오히려 "고인물 지식"을 만들어내는 재미 요소가 됨.

### 영웅 고유 특성 (Specialty)
영웅마다 **고유 특성 1개**가 고정:
- **유닛 특화**: "Griffin 전문가" → 그리핀 스탯 영구 보너스
- **스킬 특화**: "Logistics 전문가" → 이동력 추가 보너스
- **마법 특화**: "Haste 전문가" → Haste 마법 효과 증폭
- **스탯 특화**: 특정 스탯 성장 보너스

→ 같은 클래스여도 **영웅 개체별 가치 차이**가 크다.

### 유명 사기 영웅들
- **Solmyr** — Chain Lightning 특화. 초반부터 적 쓸어버림
- **Luna** — Firewall 특화. 데미지 비정상적으로 높음
- **Isra** — Necromancy 특화. 스켈레톤 무한 증식
- **Loggins** — Logistics 특화. 맵 이동이 미침

## 2. 유닛/군대 시스템

### 구조
- 진영(Castle)별 **7티어 유닛** (티어 1~7)
- 각 유닛은 **업그레이드 1회** 가능 (예: Griffin → Royal Griffin)
- 영웅 1명당 **최대 7슬롯** 유닛 스택
- 스택 = 같은 유닛 N마리 묶음 (수량 무제한)

### 전투
- **hex 그리드** 턴제 전투
- 유닛 속도순 행동 (빠른 유닛 먼저)
- 영웅은 **턴당 마법 1회** 시전 가능
- 유닛 수 × 유닛 데미지 = 총 데미지 (물량이 곧 전력)

## 3. 경제/맵 시스템

- **성(Town)** — 유닛 생산 + 건물 업그레이드 + 마법 길드
- **자원 7종**: 금, 나무, 광석, 수은, 유황, 수정, 보석
- **주간 유닛 생산**: 매주 월요일 유닛 리필
- 맵에서 광산/자원지 점령 → 수입 확보 → 더 많은 유닛 → 스노우볼

## 4. TheSevenTactics 적용 검토 포인트

| HOMM3 요소 | 적용 가능성 |
|------------|------------|
| **영웅 = 지휘관 (직접 안 싸움)** | 바알이 지휘관 역할로 파티 버프/마법 지원하는 구조와 유사 |
| **영웅 스탯 → 유닛 전이** | 바알의 지휘 능력이 파티원 전체에 영향을 주는 시스템 |
| **영웅 고유 특성** | 영웅별 Specialty → 전직/죄종 특성과 매핑 가능 |
| **스킬 8슬롯 제한 + 교체 불가** | 선택의 무게감 부여 (웹 RPG에서는 리스펙 허용이 나을 수도) |
| **스킬 밸런스 의도적 비대칭** | 모든 선택지가 동등할 필요 없음 — "정답 찾기"도 재미 |
| **주간 유닛 생산 사이클** | 웹게임의 일일/주간 콘텐츠 리셋과 잘 맞음 |

## 5. 다키스트 던전과의 비교

### 공통점
- 영웅/유닛을 모아서 파티/군대 편성
- 턴제 전투
- 유닛마다 고유 스킬/능력
- 로스터 관리 (쓸모없으면 버리고 새로 뽑고)

### 차이점
- HOMM3 — 맵 탐험 + 자원 관리 + 대규모 군대 전략
- 다키스트 던전 — 4인 파티 던전 크롤링 + 포지션 시스템 + 스트레스/광기 메카닉
- 다키스트 던전의 뿌리는 Wizardry 계열 던전 크롤러 + 로그라이크에 더 가까움
