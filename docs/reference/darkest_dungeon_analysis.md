# Darkest Dungeon 1 & 2: 게임 디자인 구조 심층 분석

> 최종 업데이트: 2026-03-23
> 참고 게임: Darkest Dungeon (Red Hook Studios, 2016) / Darkest Dungeon II (2023)
> 목적: TheSevenTactics 개발 참고용 레퍼런스

---

## 1. 유저가 해야 하는 행동

### 1-1. 코어 게임플레이 루프

**DD1: 마을 관리 → 원정 준비 → 던전 탐험 → 귀환 및 관리**

4단계 순환 구조. 핵심은 **자원의 영구적 소모(영웅 퍼마데스)와 영구적 축적(마을 업그레이드)이 공존**한다는 것. 영웅이 죽어도 마을 인프라는 남아 "실패해도 진전이 있다(Failure is Progress)"를 구조적으로 보장.

**DD2: 로그라이크 런 기반 로드 트립**

영웅 선택 → 경로 선택 → 노드 상호작용(전투/상점/성지) → 여관(Inn) 정산 → 산(The Mountain) 최종 보스. 메타 진행은 **희망(Hope) 포인트**로 영구적 잠금해제.

### 1-2. 던전/경로 선택

**DD1 - 5개 지역**, 각각 유효 상태이상과 보급품 요구가 다름:
- **폐허(Ruins)**: 언데드, 역병(Blight) 유효, 성수 핵심
- **숲(Weald)**: 야수, 출혈(Bleed) 유효
- **굴(Warrens)**: 돼지인간, 해독제 필수
- **해안동굴(Cove)**: 심해어인, 삽 다수 필요

3단계 난이도(Apprentice/Veteran/Champion)에 영웅 레벨 입장 제한 — **과잉 레벨링으로 쉽게 클리어하는 것을 원천 차단**.

**DD2** - 맵 분기점에서 안전 도로(초록)와 위험 도로(파란) 선택. 노드 아이콘으로 전투/상점/성지 등을 파악하고 파티 상태에 맞춰 전략적 결정.

### 1-3. 파티 편성

DD1은 16개 클래스(DLC 포함), 포지션 시너지가 핵심:
- **전위**: Crusader, Leper, Hellion, Man-at-Arms
- **중위**: Highwayman, Grave Robber, Hound Master
- **후위**: Vestal, Plague Doctor, Occultist, Jester

고려 요소: 던전별 유효 상태이상, 스턴 배분, 스트레스 힐러 포함 여부, 포지션 시너지

DD2는 여기에 **영웅 경로(Hero Path)** 추가 — 경로에 따라 스킬 효과와 스탯이 변화 (예: Grave Robber의 Nightsworn 경로는 HP↓ 대신 은신 공격 대미지↑)

### 1-4. 전투 시스템 — 포지션(Rank 1-4)

DD의 가장 핵심적인 메카닉. 아군·적군 모두 Rank 1~4에 일렬 배치.

- 각 스킬에 **사용 가능 포지션**과 **타격 가능 포지션**이 지정
- 밀림/당김으로 포지션이 변하면 사용 가능 스킬이 바뀜 → 핵심 긴장감

주요 스킬 예시:
- **Hellion의 Iron Swan**: Rank 1에서 적 Rank 4 직격 (최전방→최후방 공격이라는 희귀 능력)
- **Plague Doctor의 Blinding Gas**: Rank 3-4에서 적 3-4 스턴 (후방 스트레스 딜러 무력화)
- **Hellion YAWP! + PD Blinding Gas**: 전방 2칸 + 후방 2칸 = **적 4명 전원 1턴 스턴**

DD2는 **토큰(Token) 시스템**과 **콤보(Combo)** 추가 — 특정 스킬이 Combo 토큰을 부여하고, 다른 스킬이 소모하며 추가 효과 발생.

### 1-5. 스트레스 관리

**DD1**: 0~200 게이지. 100 도달 시 결의 판정.
- **75% 확률 → 고뇌(Affliction)**: Paranoid(치유 거부), Fearful(턴 스킵), Abusive(동료 공격) 등 7종. 파티원에게 스트레스 연쇄 전파 → **데스 스파이럴** 가능
- **25% 확률 → 미덕(Virtue)**: Powerful, Courageous 등. 극적 역전의 순간

**DD2**: 0~10으로 단순화. 10 도달 시 멜트다운 → 파티 전체 스트레스 2 + 관계 악화.

### 1-6. 리스크/리워드 트레이드오프

거의 모든 의사결정에 내장:
- **횃불 밝기**: 밝으면 안전, 어두우면 크리티컬↑ + 전리품↑ ("어둠 파밍")
- **큐리오 상호작용**: 올바른 아이템 없이 → 함정/저주 위험
- **Death's Door**: HP 0 → 즉사 아닌 "죽음의 문턱", 추가 피격마다 확률적 즉사. "한 번 더 버텨볼까"의 긴장
- **후퇴 판단**: 후퇴 시도 자체가 1턴 소모 + 실패 확률 + 전리품 손실 → **매몰 비용 오류를 의도적으로 유발**

### 1-7. 마을(DD1) / 여관(DD2) 관리

**DD1 마을 건물**: 역마차(영웅 모집), 대장간(장비 강화), 길드(스킬 업그레이드), 수도원/술집(스트레스 해소), 요양원(기벽 관리). 모든 업그레이드 영구적.

**DD2 여관**: 아이템 사용, 역마차 장비, 관계 정산 — DD1 마을의 압축판.

### 1-8. 보급품과 인벤토리

DD1 던전 입장 전 보급품 구매:
- 짧은 던전: 식량 12, 횃불 8, 삽 2
- 중간 던전: 식량 20, 횃불 12, 삽 3
- 긴 던전: 식량 28, 횃불 16, 삽 4
- 던전 유형별 특수 보급품(성수, 해독제, 약초, 붕대) 추가

보급품 과다 구매 → 전리품 공간 부족 / 과소 구매 → 던전 내 위험. 장신구(Trinkets)는 영웅당 2개, 대부분 트레이드오프 구조(공격력 +15% / 속도 -1).

### 1-9. 메타 진행

**DD1**: 마을 건물 업그레이드 순서가 장기 전략 좌우. 가보(Heirloom)는 던전 유형별로 다른 종류 드롭 → 업그레이드 목표에 따라 던전 선택.

**DD2**: 희망(Hope) 포인트 + 프로필 레벨. 런 실패/성공 시 Hope로 영구 잠금해제. 성지 완료 시 영웅 추가 스킬 영구 해금.

## 2. 영웅 관계

### 2-1. DD1: 기벽(Quirks) 시스템

영웅당 최대 **긍정 7개 + 부정 7개** 기벽. 던전 탐험·큐리오 상호작용 등으로 획득.

**긍정적 기벽 예시:**
- Quickdraw: 첫 라운드 SPD +4
- Natural Swing: 근접 대미지 +3
- Irrepressible: 미덕 확률 +5%

**부정적 기벽 예시:**
- Kleptomaniac: 보물 큐리오 강제 상호작용 (플레이어 통제 불가)
- Curious: 모든 큐리오 강제 상호작용
- Tippler: 술집에서만 스트레스 해소 가능

핵심 설계: **영웅이 완벽한 도구가 아니라 결함 있는 인간**이라는 메시지를 기계적으로 전달. 요양원에서 나쁜 기벽 제거, 좋은 기벽 잠금(영구화)에 높은 비용 → 장기 육성 투자 유도.

### 2-2. DD1: 고뇌/미덕과 파티 역학

스트레스 100 도달 시 결의 판정. 고뇌 유형별 파티 영향:

- **Irrational(불합리)**: 무작위 행동, 아이템 낭비
- **Paranoid(편집증)**: 치유 거부, 버프 거부 → 힐러가 Paranoid면 파티 생존 위협
- **Masochistic(자학적)**: 자해, 동료에게 공격 요구
- **Fearful(공포)**: 후퇴 요구, 턴 스킵
- **Hopeless(절망)**: 파티 전체 스트레스 대미지
- **Selfish(이기적)**: 음식/아이템 독점
- **Abusive(폭력적)**: 동료 언어 공격, 스트레스 전파

**한 영웅의 고뇌가 다른 영웅의 스트레스를 연쇄 증가** → 연쇄 고뇌 = "스트레스 데스 스파이럴". DD1에서 가장 긴장감 있는 순간.

고뇌 이력이 누적되어 다음 판정 시 같은 유형 고뇌 확률↑ — "사람은 스트레스에 일관된 방식으로 반응한다"는 심리학적 관찰의 기계적 구현.

**미덕 유형**: Powerful(대미지↑), Courageous(스트레스↓), Vigorous(매턴 HP회복), Focused(명중/크리↑), Stalwart(방어/저항↑). 25%라는 낮은 확률이 **발동 시 극적 반전 경험** 극대화.

### 2-3. DD1: 영웅의 일회성(Disposability)

가장 도발적인 설계. 기계적으로 영웅은 **소모품**(무료 모집, 무한 보충). 그러나 동시에:
- 기벽 육성 투자, 장비 업그레이드, 이름·얼굴·성격
- 고레벨 영웅 대체 비용 급증

**기계적 대체 가능성과 감정적 애착의 긴장** = DD1 핵심 감정 경험.
시작 화면: *"영웅을 잃을 것입니다. 이것은 실패가 아닙니다."*

### 2-4. DD2: 친밀도/관계(Affinity) 시스템

DD1의 암묵적 역학을 **명시적 시스템**으로:
- 모든 영웅 쌍이 친밀도 0~20 보유 (런 시작 시 9)
- 치유/버프 → 친밀도↑ / 스트레스 전파·멜트다운 → 친밀도↓
- 임계값 도달 시 **관계(Relationship)** 형성

**친밀도를 올리는 행동**: 동료 치유/버프, 전투에서 동료 보호, 여관 아이템 사용
**친밀도를 내리는 행동**: 스트레스 전파, 멜트다운, 자기중심적 행동

### 2-5. DD2: 관계의 전투 내 효과

**긍정적 관계**:
- **Amorous(연인)**: 치유/버프 시 추가 효과
- **Respectful(존경)**: 한쪽 공격 시 다른 쪽 추가 공격(Follow-up)
- **Inseparable(불가분)**: Death's Door 시 즉시 보호/치유 시도

**부정적 관계**:
- **Resentful(원한)**: 서로 스트레스 부여
- **Suspicious(의심)**: 치유/버프 거부
- **Hateful(증오)**: 상대방 효과 감소

→ 부정적 관계가 **플레이어의 최적 전략을 방해**. DD1 고뇌 시스템의 "통제 불능" 경험을 관계 역학으로 재해석.

### 2-6. DD2: 기도의 성지(Shrine of Reflection)

영웅별 5개 챕터의 과거 미니 던전. 완료 시 새 스킬 영구 해금 + 배경 스토리 제공.
DD1에서 영웅은 "잃어서 아까운 자원"이었다면, DD2에서는 "과거를 알게 되어 이해하게 되는 인물".

### 2-7. 파티 시너지

**DD1 시너지 예시:**
- **PD + Occultist**: PD의 Battlefield Medicine로 Occultist 불안정 치유의 출혈 제거 (리스크 완화)
- **Highwayman + Man-at-Arms**: Riposte + Guard = 반격 탱킹
- **Hellion YAWP! + PD Blinding Gas**: 전방 2칸 스턴 + 후방 2칸 스턴 = 적 전원 1턴 제압

**DD2 시너지 예시:**
- **콤보 체인**: Man-at-Arms의 Bellow(적 전체 Combo 토큰) + Highwayman의 Pistol Shot(Combo 소모 추가 대미지)
- **관계 시너지**: 긍정적 관계 + 근접 포지션 = Follow-up 공격 확률↑

## 3. 다른 게임과의 차이점

### 3-1. vs 전통 RPG (Final Fantasy, Dragon Quest)

| 요소 | 전통 RPG | Darkest Dungeon |
|------|----------|-----------------|
| 영웅의 죽음 | 기절→부활 아이템 | 퍼마데스, 영구 소멸 |
| 레벨링 | 순수 파워 증가 | 레벨업 = 고난이도 던전 의무화 |
| 자원 | 풍족, 보스전만 긴장 | 모든 전투가 소모전 |
| 실패 | 세이브 복귀 | 결과가 영구 누적 |
| 영웅 정체성 | 서사의 주인공 | 소모되는 도구 |

전통 RPG에서 플레이어는 영웅의 **분신**, DD에서는 **관리자/지주**.

### 3-2. vs 로그라이크 (Slay the Spire, FTL)

- **Slay the Spire**: 완벽한 정보(적 의도 표시), 덱빌딩 핵심. DD: **불완전 정보**(적 행동 예측 불가 + 스트레스/기벽 변수), 파티 편성 + 포지션 핵심
- **FTL**: 함선(하드웨어) 중심. DD: 인간(심리) 중심. 스트레스 시스템이 FTL에 없는 **인간적 불확실성 레이어** 추가

### 3-3. vs 전술 RPG (XCOM, Fire Emblem)

**XCOM** (가장 많이 비교):
- 공통: 퍼마데스, 로스터 관리
- **핵심 차이**: XCOM은 핵심 멤버 사망 시 데스 스파이럴 → 게임 진행 불가능. DD는 마을 업그레이드가 영웅 생존과 독립 → **돌이킬 수 없는 시점(Point of No Return)이 없음**
- XCOM: 2D 그리드. DD: **1D 라인(Rank 1-4)**이라는 단순화로 깊이 유지 + 인터페이스 간결

**Fire Emblem**: 서사적 애착 유도(지원 대화, 결혼). DD: **기계적 행동**(기벽, 고뇌 패턴)으로 애착 형성.

### 3-4. vs 던전 크롤러 (Wizardry, Etrian Odyssey)

- 전통 던전 크롤러: "깊이 들어갈수록 **강해진다**" (파워 커브 상승)
- DD: "깊이 들어갈수록 **닳아간다**" (자원 소모 누적). **방향이 정반대**
- 전통: 적은 "장애물". DD: 적은 **스트레스 원천** — 체력뿐 아니라 정신을 공격

### 3-5. DD만의 고유 설계

**스트레스를 HP와 동등한 독립 자원으로:**
DD 이전 RPG에서 "정신력"은 MP 변형이거나 상태이상 저항치. DD는 스트레스를 HP와 동등한 무게로 만들어 모든 의사결정에 추가 차원 부여.

**심리적 공포를 메카닉으로:**
대부분 공포 게임은 시청각 연출(점프 스케어). DD는 **게이지가 95→100으로 넘어가는 순간**이 공포. 기계적 상태에서 발생하는 공포가 서사적 무게를 가짐.

**"실패는 진행이다(Failure is Progress)":**
단순한 위로가 아닌 **구조적 사실** — 마을 업그레이드 영구, 영웅 무한 보충, 패배 시에도 가보/골드 유지.

**Wayne June 내레이션 = 게임 메카닉:**
- 크리티컬 시 *"A decisive pounce!"*, Death's Door 시 *"Death waits for the slightest misstep..."*
- UI 숫자보다 먼저 상황 심각성 전달
- *"Remind yourself that overconfidence is a slow and insidious killer"* — 핵심 디자인 메시지를 직접 전달
- H.P. Lovecraft 오디오북 경력의 Wayne June 바리톤 = 게임 정체성의 불가분한 부분

**아트 스타일:**
Mike Mignola(Hellboy) 영향의 굵은 윤곽선, 높은 명암 대비, 목판화 스타일. 영웅을 "개인"이 아닌 "유형"으로 읽게 하여 소모품적 성격을 시각적으로 지원.

### 3-6. DD1 vs DD2 구조적 차이

| 요소 | DD1 | DD2 |
|------|-----|-----|
| 구조 | 마을 경영 시뮬 + 던전 크롤링 | 로그라이크 로드 트립 |
| 영웅 | 영구 로스터, 퍼마데스 | 런 내 한정, 다음 런에서 부활 |
| 스트레스 | 0~200, 고뇌/미덕 | 0~10, 멜트다운 |
| 관계 | 암묵적 (고뇌 전파만) | 명시적 친밀도/관계 시스템 |
| 영웅 서사 | 없음 | 기도의 성지 배경 스토리 |
| 마을/거점 | 마을(Hamlet) — 다수 건물, 영구 업그레이드 | 여관(Inn) — 런 내 임시 거점 |
| 난이도 | 3단계 던전 난이도 + 영웅 레벨 제한 | 런 내 지역별 난이도 상승 |
| 최종 목표 | 다키스트 던전 4차 원정 완료 | 산(The Mountain) 보스 처치 |
| 감정 설계 | 수십 시간 마라톤, 장기 투자→상실 | 1~2시간 스프린트, 매 런 다른 조합 |

DD2는 **접근성**을 얻었지만, DD1의 **장기적 감정 투자**와 **마을 성장 만족감**은 약화. 반면 **영웅 간 관계의 명시적 표현**은 DD2에서만 가능.
