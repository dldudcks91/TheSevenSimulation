## 1. RimWorld

### 게임 소개
SF 거점 경영 시뮬레이션. 낯선 행성에 불시착한 식민자 3~15명을 관리하며 거점을 건설. 각 식민자에게 성격 특성, 기분, 스킬이 있고, 단순한 규칙들이 연쇄 반응하여 예측 불가한 드라마를 만든다.

### 유사점 / 차이점
| | RimWorld | TheSevenSimulation |
|---|---------|-------------------|
| **유사** | 거점 경영 + 주민 관리 + 이벤트 연쇄 반응 | 거점 경영 + 영웅 관리 + 죄종 연쇄 반응 |
| **유사** | 식민자 기분(Mood) 시스템 | 영웅 사기(0~100) 시스템 |
| **유사** | 기분 극단 → 정신붕괴 | 사기 극단 → 폭주/이탈 |
| **차이** | 식민자 수십 명, 자유 행동 AI | 영웅 7명, 플레이어가 배치 |
| **차이** | 실시간 + 일시정지 | 턴제 (하루 4단계) |
| **차이** | 장비에 죄종/성격 연동 없음 | 죄종이 장비 효과/이벤트에 영향 |

### 슬롯 구조
레이어 기반 착용 — Skin/Middle/Outer/Belt/Headgear/Eyes 6레이어. 같은 레이어+같은 부위만 충돌. 무기 1개.

### 효과 범위
- **전투**: Sharp/Blunt/Heat 3종 방어
- **비전투**: 기분(해진 옷 -5, 나체 -6), 온도 절연, 이동속도, 사회적 영향력(왕관 +20%), 연구 속도(연구복 +5%), 독성 저항

### 획득
제작(장인 스킬 = 품질), 교역, 습격자 전리품(Tainted 딱지), 퀘스트

### 트레이드오프
- 중장갑 = 방어↑ 이동↓
- 사망자 장비 = 무료 but 기분 패널티
- 쉴드벨트 = 원거리 차단 but 원거리무기 사용 불가
- 특성과 장비 상호작용 (누디스트 = 옷 입으면 기분↓)

### 성장
품질 7단계 (Awful~Legendary) x 소재 (천~Hyperweave). 업그레이드 없음 — 더 좋은 것을 새로 만들어야 함. 장인의 성장 = 장비의 간접 성장.

### 핵심 인사이트
**장비 상태가 드라마를 만든다.** 옷이 해지면 → 기분 하락 → 정신붕괴 → 연쇄 반응. 역할별 의상(연구원/외교관/전투원)이 로스터 관리를 심화.

## 2. Darkest Dungeon

### 게임 소개
고딕 호러 던전 크롤러 + 로스터 관리. 영웅 15~25명 로스터에서 4명 파티를 편성해 던전에 파견. 영웅에게 스트레스/질병/기벽이 쌓이고, 관리 실패 시 영구 상실. "이 게임은 고통에 관한 것이다."

### 유사점 / 차이점
| | Darkest Dungeon | TheSevenSimulation |
|---|----------------|-------------------|
| **유사** | 로스터 관리 + 파티 파견 | 영웅 관리 + 원정 파견 |
| **유사** | 스트레스 → 고난/미덕 분기 | 사기 극단 → 타락/구원 분기 |
| **유사** | 영구 상실 (사망) | 영구 상실 (사망/이탈) |
| **유사** | 트린켓의 강력한 트레이드오프 | 죄종 장비의 양면성 |
| **차이** | 던전 직접 탐험 (전투 주도) | 원정은 자동 파견, 거점 경영이 핵심 |
| **차이** | 스트레스 = 별도 수치 | 사기 1개로 통합 |
| **차이** | 거점 기능 최소 (마을 업그레이드) | 거점 = 게임의 무대 (건설/연구/생산) |

### 슬롯 구조
트린켓 2슬롯 (교체형) + 무기/방어구 (업그레이드형, 대장간 5단계)

### 효과 범위
- **전투**: DMG/ACC/CRIT/SPD/PROT/DODGE/HP
- **스트레스**: 받는 스트레스 증감
- **탐험**: 정찰 확률, 함정 해제, 횃불 밝기
- **클래스 특화**: 특정 스킬만 선택적 강화

### 획득
원정 보상, 상점, 던전 루팅, 보스 트로피, DLC 세트

### 트레이드오프 — DD의 핵심
**"혜택에는 반드시 대가가 따른다"**
- 직접: DMG↑ / HP↓
- 조건부: 특정 상황에서만 페널티
- 역할: 본역할 강화 + 부역할 약화
- **약점이 겹치면 사실상 무페널티**: DODGE 불필요한 영웅에게 DODGE 페널티 장신구 = "공짜"

### 성장
등급 체계 (Common~Ancestral). **업그레이드 불가** — 더 어려운 던전에서 획득. 장신구 = 진행도의 증거.

### 핵심 인사이트
**"완벽한 빌드는 없다."** 상황마다 최적이 달라져 다양한 영웅+장신구 조합이 필요. 희소 장신구의 "누구에게 줄 것인가" = 로스터 레벨 의사결정. 트로피 장신구가 영웅에게 이력과 개성을 부여 → 상실감 극대화.

## 3. Crusader Kings 3

### 게임 소개
중세 왕조 대전략 시뮬레이션. 왕/영주 캐릭터를 플레이하며 영토, 봉신, 가문을 관리. 캐릭터의 성격 특성이 외교/전쟁/음모에 영향. 왕조가 대를 이어가며 아티팩트가 상속/약탈된다.

### 유사점 / 차이점
| | CK3 | TheSevenSimulation |
|---|-----|-------------------|
| **유사** | 지도자(왕) 특성이 게임플레이 결정 | 바알의 판결이 거점 운명 결정 |
| **유사** | 봉신 관계 관리 (충성/반란) | 영웅 사기 관리 (충성/이탈) |
| **유사** | 아티팩트가 서사/갈등의 촉매 | 장비가 이벤트/연쇄반응의 씨앗 |
| **차이** | 수백 년 왕조 경영 | 7챕터 단기 캠페인 |
| **차이** | 아티팩트 상속/전쟁 시스템 | 영웅 사망 시 장비 회수/소실 |
| **차이** | 실시간 대전략 | 턴제 거점 경영 |

### 슬롯 구조
인벤토리 8슬롯 (왕관/무기/갑옷/왕권상징/장신구x4) + 궁정 전시물 15슬롯

### 효과 범위
- **비전투 중심**: 외교/군사/관리/음모/학식, 위신/신앙/명성, 공포, 봉신 평판, 영지 한도
- **전투**: 기사 효율, 병종 보너스
- **경제**: 건설비 감소, 개발도 증가

### 획득
의뢰 제작, 영감받은 장인(NPC), 전쟁/포위, 청구권 전쟁, 상속, 이벤트

### 트레이드오프
- **내구도**: 매년 1 감소, 0이면 영구 파괴
- **소유의 대가**: 강력한 아티팩트 → 청구권 전쟁/봉신 요구 유발
- **상속 열화**: 비가족 상속 시 추가 내구도 손실
- **리포지**: 인벤토리→궁정 전환 = 효과 변환, 비가역

### 성장
4등급 (Common~Illustrious). 골동품상 통해 한 단계씩 승급 가능. 재질+장식 조합으로 동일 아이템 없음.

### 핵심 인사이트
**"장비가 아니라 이야기의 씨앗."** 아티팩트는 파워업이 아니라 관계/갈등의 촉매. 소유→전쟁→상속→열화→복원이라는 서사 아크. 비전투 효과가 전투보다 풍부.

## 4. Battle Brothers

### 게임 소개
중세 로우 판타지 용병단 경영 + 턴제 전술. 12명 용병 로스터를 관리하며 계약을 수행. 고정 클래스 없이 장비와 퍼크로 역할을 정의. 높은 난이도와 영구 사망.

### 유사점 / 차이점
| | Battle Brothers | TheSevenSimulation |
|---|----------------|-------------------|
| **유사** | 용병단(로스터) 관리 | 영웅 로스터 관리 |
| **유사** | 장비가 역할을 정의 (클래스 없음) | 장비+스탯으로 역할 결정 |
| **유사** | 영구 사망, 장비 손실 위험 | 영구 사망/이탈 |
| **유사** | Named 아이템 파밍 = 핵심 동기 | 보스 드롭/유니크 장비 = 원정 동기 |
| **차이** | 플레이어가 전투 직접 지휘 | 전투는 완전 자동 |
| **차이** | 거점 없음 (방랑 용병단) | 거점 경영이 핵심 |
| **차이** | 성격/사기 시스템 단순 | 죄종+사기가 게임의 심장 |

### 슬롯 구조
머리/몸통/갑옷부착물/주무기/보조(방패)/장신구/가방(2~4칸)

### 효과 범위
- **전투**: 방어(내구도=방어력), 피로 패널티, 시야, 이니셔티브
- **무기 고유 기술**: 메이스=기절, 도끼=방패파괴, 단검=관통
- **비전투**: 수리 자원 소모, 부착물 효과

### 획득
전리품(주), 상점(Named 1%), 제작(DLC, 트로피 재료), 전설적 장소, 주점 소문

### 트레이드오프
- **방어↔피로/기동**: 중갑 = 방어↑ 피로↓ 이니셔티브↓
- **투구: 방어↔시야**: 폐쇄형 = 방어↑ 시야↓ (궁수에게 치명적)
- **한손+방패↔양손무기**: 생존↔화력
- **Named 아이템 = 트레이드오프 완화**: "중갑인데 경갑 수준 피로" → 파밍 동기

### 성장
Tier 1~3 + Named(랜덤 강화) + Legendary(고유 효과). 내구도 = 방어력 (실시간 감소). 무기 내구도 0 = 영구 파괴.

### 핵심 인사이트
**"클래스가 없다 — 장비가 역할을 정의한다."** 같은 용병에게 그레이트소드 주면 딜러, 방패 주면 탱커. 스탯×장비 시너지 판단이 핵심. Named 아이템 획득이 곧 스토리.

## 5. XCOM 2

### 게임 소개
SF 분대 전술 + 기지 경영. 외계인 점령 지구에서 저항군을 이끈다. 4~6명 분대를 미션에 파견하고, 기지에서 연구/제작/시설 건설. 영구 사망이 있어 병사에 대한 감정적 투자가 크다.

### 유사점 / 차이점
| | XCOM 2 | TheSevenSimulation |
|---|--------|-------------------|
| **유사** | 기지 경영 + 분대 파견 | 거점 경영 + 원정 파견 |
| **유사** | 연구/제작으로 전력 강화 | 연구/건설/생산으로 거점 성장 |
| **유사** | 영구 사망 + 장비 손실 위험 | 영구 사망/이탈 |
| **유사** | 희소 장비 배분 딜레마 | 7명 중 누구에게 줄 것인가 |
| **차이** | 미션 = 플레이어가 직접 전술 지휘 | 원정 = 완전 자동전투 |
| **차이** | 병사에 성격/감정 시스템 없음 | 죄종+사기가 핵심 |
| **차이** | 무기 티어 = 전군 적용 | 장비는 개별 영웅에 귀속 |

### 슬롯 구조
주무기(클래스 고정)/보조무기(클래스 고정)/방어구/유틸리티(1~2)/PCS/무기모드(1~2)

### 효과 범위
- **무기 모드**: 명중/치명타/즉사확률/무료행동 등
- **PCS**: 의지력/명중/이동/회피/체력 직접 보정
- **유틸리티**: 탄약(속성변경), 조끼(생존), 수류탄(범위), 특수장비
- **방어구**: HP + 유틸리티 슬롯 수 결정

### 획득
연구(무기티어 = 전군 적용), 엔지니어링(방어구 개별제작), 실험실(랜덤 결과), 전장 루팅, 블랙마켓

### 트레이드오프
- **범용 방어구(슬롯 2개) vs 특수 슈트(슬롯 1개 + 특수능력)**
- **실험실 랜덤**: 원하는 아이템을 위해 자원 반복 투자
- **자원 경쟁**: 시설 건설/연구/장비가 동일 풀에서 소모

### 성장
무기 3티어 (재래식→자기→플라즈마, 전군 적용). 방어구 3티어 (개별 제작). 무기 모드 3등급 (교체 시 기존 소멸).

### 핵심 인사이트
**"티어 업은 전체, 희귀템은 개별 배분."** 전군 파워가 올라가면서도 "Superior 스코프를 누구에게 줄 것인가" 딜레마 유지. **퍼마데스 + 장비 = 감정적 무게** — 최정예에 투자 집중 vs 사망 시 손실 리스크.

## 6. Fire Emblem: Three Houses

### 게임 소개
전술 RPG + 학원 경영. 사관학교 교관으로서 학생들을 육성하고 전쟁에 참전. 캐릭터 간 인연(Support) 시스템이 전투/스토리에 영향. 기병대(Battalion)라는 독특한 외부 전력 레이어.

### 유사점 / 차이점
| | FE: Three Houses | TheSevenSimulation |
|---|-----------------|-------------------|
| **유사** | 캐릭터 육성 + 편성 관리 | 영웅 관리 + 파티 편성 |
| **유사** | 캐릭터 간 관계(Support)가 전투에 영향 | 죄종 간 관계가 사기/이벤트에 영향 |
| **유사** | 무기 내구도 = 자원 관리 긴장감 | 원정 보급/장비 관리 |
| **차이** | 플레이어가 전투 직접 지휘 (턴제 전술) | 전투 완전 자동 |
| **차이** | 고정 캐릭터 30+ 명 | 랜덤 생성 영웅 7명 |
| **차이** | 거점 기능 제한 (수도원 탐색) | 거점 경영이 핵심 |

### 슬롯 구조
인벤토리 6칸 (무기+장비 혼합) + 기병대 1개 + 어빌리티 9개

### 효과 범위
- **무기**: 위력/명중/치명타/무게/내구도
- **기병대**: 스탯 직접 가산 + 고유 계략(범위공격/상태이상)
- **문장(Crest) 연동**: 대응 문장 영웅만 진가 발휘

### 획득
상점, 대장간(강화/수리), 기병대 길드, 외전 보상, 적 드롭

### 트레이드오프
- **내구도 소모**: 전투기술 사용 시 3~5 추가 소모 — "강력한 일격의 대가"
- **무게↔위력**: 강한 무기 = 무거움 = 공속↓
- **기병대 인내력**: 레벨업 불가, 소진 시 효과 전부 소실, 유료 보충
- **영웅 유물**: 최강이지만 수리 재료(Umbral Steel) 극히 한정

### 성장
무기 강화(+버전), 기병대 레벨(1~5), 문장 연동 장비.

### 핵심 인사이트
**기병대 = 개인 장비 너머의 외부 레이어.** "영웅 A의 부대를 B에게 재배치"하는 경영 판단. **내구도가 자원 관리 긴장감**을 만들되, 파손 = "사용불가"가 아니라 "성능저하" → 절망 없이 동기부여.

## 7. Dwarf Fortress

### 게임 소개
판타지 거점 경영 시뮬레이션의 원조. 난쟁이 7명으로 시작해 요새를 건설. 각 난쟁이에게 성격/취향/기분/관계가 있고, 극도로 복잡한 시뮬레이션이 예측 불가한 드라마를 만든다. "Losing is fun."

### 유사점 / 차이점
| | Dwarf Fortress | TheSevenSimulation |
|---|---------------|-------------------|
| **유사** | 거점 건설 + 주민 관리 | 거점 건설 + 영웅 관리 |
| **유사** | 주민 기분 → 분노 연쇄(Tantrum Spiral) | 사기 → 폭주 연쇄 반응 |
| **유사** | 아이템 파괴 → 감정 폭발 → 추가 파괴 | 폭주 → 시설/장비 파괴 → 연쇄 |
| **유사** | 취향 시스템 (선호 재료 = 행복) | 죄종 = 만족/불만 조건 |
| **차이** | 주민 수십~수백 명, 자동 AI | 영웅 7명, 플레이어 배치 |
| **차이** | 실시간 시뮬레이션 | 턴제 |
| **차이** | 엔드리스 모드 | 7챕터 캠페인 |

### 슬롯 구조
4레이어 (Under/Over/Armor/Cover) x 신체부위(머리/상체/하체/팔/손/다리/발). Layer Size/Permit 시스템으로 중첩 규칙.

### 효과 범위
- **전투**: 부위별 방어, 무게→이동속도
- **기분**: 나체/해진 옷 = 강력한 불행, 고품질 의복 = 사치 욕구 충족, 걸작 파괴 = 치명적 정신 타격
- **취향**: 선호 재료 아이템 착용 시 행복, 작업 시 품질 보정
- **사회적 지위**: 방/가구 가치 = 지위, 귀족에게 미지급 시 불행

### 획득
공방 제작(3티어), 교역, 전리품, **이상한 기분(Strange Mood)** — 광기 상태에서 아티팩트 제작 (실패 시 폭주/사망)

### 트레이드오프
- **재료 품질↔가용성**: 아다만타인 = 최강 but 채굴 위험
- **아티팩트 = 부(Wealth) 증가 → 강한 침략자 유발**
- **Fell 기분**: 동료 살해하여 뼈로 걸작 제작 — 강력하지만 사기에 치명적
- **의복 마모**: 2년마다 등급 하락 → 지속적 생산 압박

### 성장
품질 6+1단계 (기본~Masterwork + Artifact). 아티팩트 = 장인급의 60배 가치. 재료×품질 2축.

### 핵심 인사이트
**"단순한 규칙이 복잡한 드라마를 만든다."** 걸작 파괴 → 장인 분노 → 다른 물건 파괴 → 연쇄 분노(Tantrum Spiral) → 요새 붕괴. 취향 시스템 하나로 장비 착용/작업 품질/방 만족도 전부 연결. **옷이 닳는다 = 지속적 경제 루프.**

## 8. Kenshi

### 게임 소개
포스트 아포칼립스 오픈월드 RPG + 거점 경영. 아무것도 없는 상태에서 시작해 거점을 짓고, 분대를 키우고, 세력과 싸운다. 클래스 없이 장비와 스킬이 역할을 결정. 극도로 높은 자유도.

### 유사점 / 차이점
| | Kenshi | TheSevenSimulation |
|---|--------|-------------------|
| **유사** | 거점 건설 + 분대 관리 | 거점 건설 + 영웅 관리 |
| **유사** | 장비가 역할 정의 (클래스 없음) | 장비+스탯으로 역할 결정 |
| **유사** | 제작자 스킬 = 장비 품질 | 대장간 + 영웅 투입 = 장비 품질 |
| **차이** | 실시간 오픈월드 | 턴제 거점 중심 |
| **차이** | 비전투 활동에 장비 효과 없음 | **비전투에도 장비 효과 = 차별화 포인트** |
| **차이** | 성격/감정 시스템 없음 | 죄종+사기가 핵심 |

### 슬롯 구조
7슬롯 (머리/셔츠/몸통갑옷/하의/신발/주무기/보조무기) + 배낭

### 효과 범위
- **전투**: 공격/방어/회피/전투속도/은신/사격 보정 (갑옷 유형별 차등)
- **이동**: 짐 비율에 따라 이동속도/회피 감소
- **훈련**: 무거운 장비 = 근력 훈련 효과
- **비전투 활동에는 효과 없음** (농사/제작/노동)

### 획득
제작(스미스 스킬→품질), 상점, 루팅, 고대 유적 탐험, Meitou(고유 NPC 처치)

### 트레이드오프
- **경갑**: 공격/회피/은신 보너스, 낮은 방어
- **중갑**: 최고 방어, 공격/민첩/은신/사격에 대규모 페널티
- **무기×갑옷 시너지**: 참격무기+중갑 = 최악 (Dex 이중 페널티), 둔기+중갑 = 최적
- **배낭**: 인벤토리↑ 전투 스탯↓

### 성장
갑옷 6등급 (Prototype~Masterwork), 무기 15등급 (Rusted Junk~Meitou). 제작 벤치 레벨 + 스미스 스킬로 결정.

### 핵심 인사이트
**"완벽한 장비는 없다 = 역할 정체성 형성."** 장비 선택이 곧 캐릭터 역할 결정. **비전투 활동에 장비 효과 없음 — TheSevenSimulation이 차별화할 최대 기회.** 이중 성장 축(캐릭터 스킬 + 장비 품질)으로 제작자의 가치가 전투원만큼 중요.

## 9. Mount & Blade: Bannerlord

중세 전략 RPG + 액션 전투. 영주로서 영지를 관리하고, 군대를 이끌고, 직접 전장에서 싸운다. 영웅(동료) 장비는 수동 관리, 병사 장비는 승급으로 자동 관리하는 이중 구조.

### 유사점 / 차이점
| | Bannerlord | TheSevenSimulation |
|---|-----------|-------------------|
| **유사** | 영웅 + 병사 관리 | 영웅 관리 |
| **유사** | 대장간 제작 시스템 | 대장간에서 장비 제작 |

### 슬롯 구조
영웅 12슬롯 (머리/몸통/어깨/장갑/부츠/말/마갑/무기x4 + 민간 로드아웃 별도). 병사 장비는 승급 시 자동 변경.

### 효과 범위
- **전투**: 부위별 방어(머리/몸통/팔/다리), 무기 3종 데미지(베기/찌르기/타격)
- **비전투**: 말 → 전략맵 이동속도, 민간 장비 → 도시 내 전투
- **외교/교역에는 직접 효과 없음** (퍽 시스템에서 처리)

### 획득
상점(클랜 레벨에 따라 고급 등장), 전투 노획(부정적 수식어), 토너먼트 우승(문화별), 대장간 제작

### 트레이드오프
- **무게↔방어**: 중갑 = 도보 속도↓ (기마 시 무관)
- **마갑**: 말 생존↑ 속도↓
- **대장간 경제 문제**: 고티어 무기 가치가 너무 높아 다른 수입원 무력화

### 성장
수식어 시스템 (Rusty~Lordly), 대장간 파츠 해금(티어 1~5), 문화별 고유 장비 라인.

### 핵심 인사이트
**영웅 수동관리 + 병사 자동관리의 이원 구조**로 마이크로 폭발 방지. **문화별 장비 = 지역 탐험 동기** → 7대 죄악 챕터별 고유 장비와 대응 가능. 제작 경제 밸런스 실패가 반면교사.

## 10. Oxygen Not Included

### 게임 소개
SF 거점 경영 시뮬레이션. 소행성 내부에서 복제인(Duplicant) 3~20명을 관리하며 기지를 건설. 산소/온도/전력/식량 등 자원 체인을 설계하는 것이 핵심. 장비보다 인프라가 중요.

### 유사점 / 차이점
| | ONI | TheSevenSimulation |
|---|-----|-------------------|
| **유사** | 거점 건설 + 주민 관리 | 거점 건설 + 영웅 관리 |
| **유사** | 주민 사기(Morale) 관리 | 영웅 사기 관리 |
| **유사** | 장비 = 새로운 영역/활동 해금 | 장비 = 더 어려운 원정/방어 가능 |
| **차이** | 실시간 시뮬레이션 | 턴제 |
| **차이** | 장비 = 인프라 문제 (배관/전력/독) | 장비 = 개인 아이템 |
| **차이** | 성격/드라마 시스템 약함 | 죄종+연쇄반응이 핵심 재미 |

### 슬롯 구조
의류 슬롯 (플레이어 할당) + 엑소슈트 슬롯 (인프라 자동 착탈). 두 슬롯 독립, 동시 착용 가능.

### 효과 범위
- **환경 보호**: 산소 공급, 온도 극한, 독성, 방사선
- **능력치**: 운동능력 패널티(-2~-8), 굴착/힘 보너스
- **사기**: 장식(Decor) 보너스를 통한 간접 경로 (스내지 슈트 +30)
- **핵심 역할**: 숫자 부스트가 아니라 **새로운 영역 접근 해금**

### 획득
연구 해금 → 제작소에서 제작. 리드 섬유(농업)/정제 금속/강철 등 재료 필요.

### 트레이드오프
- **기압 슈트**: 완전 보호 but 운동능력 -6 + 인프라 투자 필수 (산소배관/전력/독)
- **의류**: 스내지 슈트(사기↑ 보호X) vs 웜코트(보호↑ 사기↓)
- **산소 마스크 vs 기압 슈트**: 싸고 약한 것 vs 비싸고 강한 것

### 성장
티어 진행: 맨몸 → 의류 → 산소마스크 → 기압슈트 → 제트슈트 → 납슈트. 스킬 훈련으로 패널티 완화.

### 핵심 인사이트
**"장비 = 인프라 문제."** 슈트 자체보다 지원 체인(산소/전력/독) 설계가 게임플레이. **"일상용(사기)"과 "임무용(생존)"의 이중 구조**로 선택이 단순하면서도 조합의 깊이 존재.

## 종합 시사점: TheSevenSimulation에 적용할 핵심 교훈

### 교훈 1: 장비는 전투 밖에서도 의미가 있어야 한다
**RimWorld/DF/CK3**가 증명 — 거점 경영 시뮬에서 장비의 비전투 효과(기분, 작업효율, 사회적 영향)가 전투 효과만큼 중요하다. **Kenshi가 이 부분이 비어있어** TheSevenSimulation이 차별화할 기회.

→ **장비가 7스탯을 보정**하면, 자연스럽게 건설/연구/생산/전투/이벤트 모든 활동에 영향

### 교훈 2: 트레이드오프 없는 장비는 의미 없다
**DD/Battle Brothers/Kenshi** 공통 — 모든 장비에 장단점이 있어야 "선택"이 된다. 단순 상위호환은 의사결정을 죽인다.

→ **죄종 연동 트레이드오프**: "분노의 검 = 전투력↑ 사기 변동폭↑" 같은 구조

### 교훈 3: 장비가 드라마를 만들어야 한다
**DF/CK3/RimWorld** — 장비 상태(해짐, 파괴, 도난)가 연쇄 반응의 씨앗. 아이템이 서사적 의미를 가질 때 상실감과 성취감이 극대화.

→ **폭주/이벤트와 장비 연결**: 탐욕 영웅이 남의 장비를 훔치는 이벤트, 시기 영웅이 장비를 파괴하는 이벤트 (이미 이벤트 설계에 반영됨)

### 교훈 4: "누구에게 줄 것인가" 딜레마
**XCOM 2/DD** — 희소 아이템의 배분이 로스터 관리의 핵심 긴장감. 전체 업그레이드와 개별 배분의 이중 구조가 효과적.

→ **보스 드롭 유니크, 대장간 제작 고급품**을 7명 중 누구에게 줄지 = 경영 판단

### 교훈 5: 장비가 역할을 정의할 수 있다
**Battle Brothers/Kenshi** — 클래스 시스템 없이 장비로 역할을 나눈다. 같은 영웅이라도 장비에 따라 전투원/연구원/건설자가 된다.

→ TheSevenSimulation은 이미 7스탯+행동별 가중치로 역할 분화. **장비가 스탯을 보정하면 역할 강화/전환 가능**

### 교훈 6: 품질 시스템은 단순할수록 좋다
**DF(품질x재료 2축)/RimWorld(품질 7단계)** — 복잡한 접사보다 직관적인 품질 등급이 경영 시뮬에 적합.

→ Phase 1은 **등급(보통/좋은/뛰어난) + 기본 스탯 보정**으로 시작, Phase 2에서 죄종 접사 추가

### 교훈 7: 제작 경제의 밸런스
**Bannerlord의 실패** — 제작 산출물 가치가 너무 높으면 다른 콘텐츠가 죽는다.

→ 대장간 제작은 원정 전리품/상인 구매와 **경쟁 관계**로 설계, 어느 하나가 지배적이지 않도록
