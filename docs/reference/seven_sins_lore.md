# 7대 죄악 원전 레퍼런스

> 게임 설계를 위한 신학/철학적 배경 정리
> 작성일: 2026-03-23

---

## 기원

- **4세기** 에바그리우스 폰티쿠스 — "8가지 악한 생각(logismoi)" 정리
- **590년** 교황 그레고리우스 1세 — 7가지로 재편 (Moralia in Job)
- **13세기** 토마스 아퀴나스 — "수도적 죄악(capital sins)" 체계화 (Summa Theologica)
  - capital = 라틴어 caput(머리). 다른 모든 죄의 근원이자 우두머리

---

## 단테의 구조 원리 (연옥편)

사랑의 왜곡 관점에서 3그룹 분류:

| 분류 | 층 | 죄종 | 의미 |
|------|---|------|------|
| 왜곡된 사랑 (타인에 해악) | 1~3 | 교만, 시기, 분노 | 사랑의 방향이 틀어짐 |
| 부족한 사랑 (의지 결핍) | 4 | 나태 | 사랑 자체가 부족 |
| 과도한 사랑 (세속 집착) | 5~7 | 탐욕, 폭식, 색욕 | 사랑의 양이 과잉 |

교만이 가장 아래(가장 근본적), 색욕이 가장 위(가장 가벼운 죄).

## 죄종별 상세

### 1. 교만 (Pride / Superbia) — 모든 죄의 뿌리

- **라틴어**: Superbia — super(위에) = "자신을 남 위에 놓는 것"
- **신학**: 아퀴나스 "모든 죄의 시작은 교만" / 아우구스티누스 "자신을 하나님보다 더 사랑하는 것"
- **악마**: 루시퍼 — 가장 아름다운 천사가 신과 같아지려다 추락
- **미덕**: 겸손 (Humilitas)
- **상징**: 보라색, 공작/사자, 불/공기
- **성경**: 루시퍼의 추락(이사야 14장), 바벨탑(창세기 11장), 느부갓네살(다니엘 4장)
- **단테 형벌**: 거대한 바위를 등에 짊어지고 걸어야 함 → 강제로 고개를 숙여 겸손을 배움
- **본질**: 나머지 6가지 죄 모두의 출발점
- **파생(아퀴나스)**: 불순종, 자만, 위선, 논쟁, 고집, 불화, 기이함 추종

### 2. 시기 (Envy / Invidia) — 타인의 선에 대한 슬픔

- **라틴어**: Invidia — in-videre(악의로 바라보다) = "곁눈질하여 악의적으로 보다"
- **신학**: 아퀴나스 "타인의 선(善)에 대한 슬픔" — 타인의 행복 자체가 고통
- **악마**: 레비아탄 — 바다의 거대한 괴물
- **미덕**: 친절/자비 (Humanitas)
- **상징**: 초록색("green with envy"), 뱀/개, 얼음/물
- **성경**: 사울과 다윗(사무엘상 18장), 요셉의 형제들(창세기 37장)
- **단테 형벌**: 눈꺼풀이 철사로 꿰매져 아무것도 볼 수 없음
- **본질**: 교만을 전제로 함. 단순히 원하는 게 아니라 타인이 가진 것 자체가 고통
- **파생(아퀴나스)**: 증오, 험담, 비방, 남의 불행에 기쁨, 남의 행복에 슬픔

### 3. 분노 (Wrath / Ira) — 이성을 삼키는 불

- **라틴어**: Ira — 인도유럽조어 h1eys-(열정, 격정)에서 유래
- **신학**: 아퀴나스 — 정당한 의분은 선, 이성의 질서를 벗어난 분노가 죄
- **악마**: 사탄 — 신에 대한 반역과 증오의 화신
- **미덕**: 인내/온유 (Patientia)
- **상징**: 빨간색, 늑대/곰, 불(Fire)
- **성경**: 카인과 아벨(창세기 4장) — 분노→살인 최초 사례
- **단테 형벌**: 짙은 검은 연기 속을 쉬지 않고 달림
- **본질**: 자신의 목표 좌절 시 발생. 이성 상실이 핵심
- **파생(아퀴나스)**: 분개, 마음의 부풂, 고함, 신성모독, 모욕, 말다툼

### 4. 나태 (Sloth / Acedia) — 돌봄의 부재

- **라틴어**: Acedia — 그리스어 ἀκηδία = ἀ-(결여) + κηδία(돌봄) = "돌봄의 부재"
- **신학**: 단순한 게으름이 아닌 영적 선(善)에 대한 슬픔. "정오의 악마(noonday demon)"
- **악마**: 벨페고르 — 발명/발견으로 유혹하는 악마
- **미덕**: 근면 (Industria)
- **상징**: 연파란색, 나무늘보/당나귀, 공기/흙
- **성경**: 달란트 비유(마태복음 25장) — 한 달란트를 땅에 묻어둔 종
- **단테 형벌**: 쉬지 않고 달려야 함
- **본질**: 게으름이 아니라 무관심. 해야 할 것을 알면서 하지 않는 의지의 결핍
- **파생(아퀴나스)**: 악의, 원한, 소심, 절망, 계명 태만, 불법적 방황

### 5. 탐욕 (Greed / Avaritia) — 끝없는 소유욕

- **라틴어**: Avaritia — avēre(갈망하다)에서 유래
- **신학**: 필요 이상의 물질적 부를 추구하는 과도한 욕망
- **악마**: 맘몬 — 재물의 의인화 (마태복음 6:24 "하나님과 맘몬을 동시에 섬길 수 없다")
- **미덕**: 자선/관용 (Caritas)
- **상징**: 노란색/금색, 두꺼비/여우, 흙(Earth)
- **성경**: 유다의 배신(은 30), 부자 청년(마태복음 19장)
- **단테 형벌**: 땅에 엎드려 누워 있음
- **본질**: 소유 자체가 목적. 채워도 채워지지 않는 만성적 불만족
- **파생(아퀴나스)**: 배신, 사기, 거짓, 위증, 불안, 폭력, 자비 결여

### 6. 폭식 (Gluttony / Gula) — 탐닉과 과잉

- **라틴어**: Gula — gula(목구멍)에서 유래
- **신학**: 아퀴나스 5가지 형태 — 너무 급하게 / 비싸게 / 많이 / 간절히 / 까다롭게
- **악마**: 베엘제붑 — "파리의 왕", 사탄 다음가는 고위 악마
- **미덕**: 절제 (Temperantia)
- **상징**: 주황색, 돼지, 흙(Earth)
- **성경**: 에서의 장자권(창세기 25장) — 팥죽에 장자권을 팜
- **단테 형벌**: 극심한 배고픔/갈증 속에서 손 닿지 않는 과일나무
- **본질**: 음식에 국한되지 않음. 모든 종류의 과도한 소비
- **파생(아퀴나스)**: 부적절한 기쁨, 조롱, 불결, 수다, 이해력 둔화

### 7. 색욕 (Lust / Luxuria) — 집착과 방종

- **라틴어**: Luxuria — luxus(과잉, 사치)에서 유래. 원래 모든 종류의 과잉을 뜻함
- **신학**: 이성과 신법의 질서를 벗어난 쾌락의 추구
- **악마**: 아스모데우스 — 토빗기의 악마
- **미덕**: 순결 (Castitas)
- **상징**: 파란색/진홍색, 염소/뱀, 불(Fire)
- **성경**: 삼손과 들릴라(사사기 16장), 다윗과 밧세바(사무엘하 11장)
- **단테 형벌**: 거대한 불길의 벽을 통과 — 연옥 최상층(가장 가벼운 죄)
- **본질**: 단순한 성욕이 아닌 대상에 대한 과도한 집착과 탐닉
- **파생(아퀴나스)**: 정신 눈멂, 경솔, 변덕, 성급, 자기애, 신 증오, 현세 집착, 내세 절망

## 죄악 간 상호관계

### 에바그리우스의 연쇄 — 육체적→영적 순서

```
폭식 → 색욕 → 탐욕 → 슬픔 → 분노 → 나태 → 허영 → 교만
```

분기 구조:

```
        폭식
       ↙    ↘
    색욕     탐욕
            ↙    ↘
        슬픔     교만
         ↓
        분노
         ↓
        허영
```

영혼의 삼분법 대응:

| 영혼의 부분 | 해당 죄악 |
|------------|----------|
| 욕망적 (본능) | 폭식, 색욕, 탐욕 |
| 기개적 (감정) | 슬픔, 분노, 나태 |
| 이성적 (정신) | 허영, 교만 |

### 그레고리우스의 군사 체계

```
교만 (Superbia) = 죄악의 여왕
  └─ 7장수: 허영, 시기, 분노, 우울, 탐욕, 폭식, 색욕
     └─ 각각 자기 군대(딸=파생 죄악)를 이끔
```

### 관계 매트릭스

```
         교만  시기  분노  나태  탐욕  폭식  색욕
교만      ──   생성  강화   대립  강화   ·    강화
시기     전제   ──   생성   ·    강화   ·     ·
분노     강화  강화   ──   대립   강화   ·     ·
나태     대립   ·   대립   ──    대립  강화   대립
탐욕     강화  강화  강화   대립   ──   대립   강화
폭식      ·    ·    ·    강화   대립   ──   생성
색욕     강화   ·    ·    대립   강화  생성    ──
```

- 생성: 앞의 죄가 뒤의 죄를 직접 낳음
- 강화: 합치면 더 위험해짐
- 대립: 표면적 모순 (실제로는 공존 가능)
- 가장 많은 강화 관계: 교만(5개) — "죄악의 여왕"
- 가장 많은 대립 관계: 나태(5개) — 단테가 연옥 정중앙에 배치

### 강화 조합 사례 (문학)

| 조합 | 이름 | 사례 |
|------|------|------|
| 교만 + 분노 | 폭군 | 네로, 리처드 3세 |
| 교만 + 시기 | 사탄형 | 밀턴의 사탄, 이아고 |
| 시기 + 분노 | 복수자 | 메데아, 오셀로의 이아고 |
| 탐욕 + 분노 | 약탈자 | 정복 군주, 해적 |
| 색욕 + 교만 | 유혹자 | 돈 후안, 카사노바 |
| 폭식 + 나태 | 퇴폐 | 로마 제국 말기 |
| 교만 + 나태 | 냉소주의 | 니힐리스트 |
