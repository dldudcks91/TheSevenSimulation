## 공통 아트 프롬프트 프리픽스

```
16-bit pixel art, SNES-style RPG character portrait card, dark fantasy theme,
bordered card frame with gold trim, black background,
upper body portrait, front-facing, detailed shading,
[캐릭터별 프롬프트]
```
