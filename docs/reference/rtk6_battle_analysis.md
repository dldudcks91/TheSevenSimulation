# 삼국지6 전투 시스템 분석

> 작성일: 2026-03-30
> 목적: TheSevenSimulation 전투 시스템 레퍼런스

---

## 1. 전투 구조

### 기본 흐름
```
전투 개시 → 매 턴 각 부대에 명령 → 자동 실행 → 다음 턴 명령 → ... → 승패
```

- 턴제. 매 턴마다 아군 각 부대에 명령을 1개씩 내림
- 명령 후 양측이 동시에 행동 (실행은 자동)
- 장수의 성격/꿈에 따라 **명령 불복** 가능

### 전투 유닛
- 각 장수가 1부대를 이끔 (병력 수 = 부대 체력)
- 총대장 + 부장 구조. 총대장 부대 괴멸 = 패배
- 병종: 보병, 기병, 노병(궁병), 수군 등

---

## 2. 명령 체계

### 매 턴 각 부대에 내리는 명령

| 명령 | 효과 | 비고 |
|------|------|------|
| **돌격** | 지정 적 부대를 공격 | 기본 공격 명령 |
| **수비** | 방어 자세, 피해 경감 | 그 자리에서 버팀 |
| **이동** | 지정 위치로 이동 | 포진 변경용 |
| **계략** | 화공, 매복, 혼란 등 지능 기반 전술 | 장수 지능에 따라 성공률 |
| **일기토** | 적 장수에게 1:1 대결 신청 | 무력 기반, 거부 가능 |
| **후퇴** | 전장 이탈 | 추격 피해 가능성 |
| **대기** | 아무것도 안 함 | 상황 관망 |

### 계략 상세
- **화공**: 지능 기반, 화재 지역 생성. 바람 방향 영향
- **혼란**: 적 부대 혼란 → 다음 턴 행동 불능
- **매복**: 사전에 매복 배치 → 적 진입 시 기습
- **위보**: 가짜 군세로 위협

---

## 3. 꿈 시스템과 명령 불복

### 삼국지6의 핵심 — "꿈(이상)"
모든 장수에게 고유한 **꿈**(가치관)이 있음:

| 꿈 | 설명 | 성향 |
|---|------|------|
| 패왕 | 천하를 무력으로 통일 | 공격적, 독단적 |
| 의협 | 의리와 정의 | 충성적, 의리 있는 명령 복종 |
| 야망 | 자신의 출세 | 이기적, 불리하면 배신 |
| 명예 | 명성과 체면 | 체면 손상 시 불복 |
| 이상 | 이상 국가 건설 | 합리적, 무모한 명령 거부 |
| 안녕 | 평화로운 삶 | 소극적, 전쟁 기피 |

### 명령 불복 조건
- **충성도가 낮으면**: 명령 무시 확률↑
- **꿈과 명령이 충돌하면**: 불복
  - "안녕" 장수에게 돌격 명령 → 높은 확률로 거부
  - "패왕" 장수에게 수비 명령 → 독단으로 돌격할 수 있음
  - "명예" 장수에게 후퇴 명령 → "치욕적이다!" 거부
- **군주와 꿈이 같으면**: 복종률 높음
- **군주와 꿈이 다르면**: 불복 가능성 상존

### 불복 시 행동
- 명령 무시하고 **자기 성격대로 행동**
- "패왕" 장수가 수비 명령 무시 → 독단 돌격
- "안녕" 장수가 돌격 명령 무시 → 대기 or 후퇴
- 심한 경우 전투 중 **이탈** (도주)

---

## 4. 일기토 시스템

- 무력이 높은 장수가 적 장수에게 1:1 대결 신청
- 상대가 수락하면 일기토 전투 돌입
- 결과: 승리 → 적 부대 사기↓↓, 장수 부상/포로
- 패배 → 아군 사기↓↓, 장수 부상/포로
- 거부 → 거부한 측 사기 소량 하락
- **무력 차이**가 클수록 결과가 뻔함 → 약한 장수를 보내는 건 자살

## 5. 전투 UI

```
┌─── 전투 화면 ──────────────────┐
│  지형 맵 (격자 기반)             │
│                                 │
│  [아군 부대들]   [적 부대들]     │
│   장수명          장수명         │
│   병력 수         병력 수        │
│   사기 게이지     사기 게이지    │
│                                 │
├─── 명령 패널 ──────────────────┤
│  선택된 부대: 관우 (기병 5000)   │
│                                 │
│  [돌격] [수비] [계략] [일기토]  │
│  [이동] [대기] [후퇴]           │
│                                 │
│  대상: [적 부대 선택]            │
└─────────────────────────────────┘
```

- 매 턴 각 부대 클릭 → 명령 선택 → 대상 선택
- 모든 부대 명령 완료 → "실행" → 양측 동시 행동
- 실행 중에는 관전 (개입 불가)

## 6. 위임 전투

- 전투를 AI에게 위임 가능 → 자동으로 명령/실행 반복
- AI가 장수 성격에 따라 명령을 내림
- 직접 지휘보다 비효율적이지만 편리

---

## 7. 전투 결과 → 경영 영향

| 결과 | 효과 |
|------|------|
| 적 장수 포로 | 등용(영입) or 처형 or 석방 선택 |
| 아군 장수 포로 | 적이 처형하거나 자동 귀환 |
| 장수 사망 | 영구 사망 (삼국지6은 사망 있음) |
| 병력 손실 | 모병/징병으로 보충 필요 |
| 사기 하락 | 다음 전투에 영향 |

## 8. TheSevenSimulation 적용 포인트

### 직접 대응 관계

| 삼국지6 | TheSevenSimulation |
|---------|-------------------|
| 장수의 꿈 | 영웅의 죄종 |
| 충성도 | 사기 (0~100) |
| 명령 불복 | 죄종 기반 불복 |
| 총대장 괴멸 = 패배 | 영웅 전원 기절 = 패배 |
| 일기토 | 일기토 (이미 기획됨) |
| 위임 전투 | 스킵 버튼 |

### 핵심 차용 요소
1. **매 N라운드 각 영웅에게 개별 명령** — 공격/방어/계략/일기토/대기
2. **죄종에 따른 명령 불복** — 분노 영웅에게 "수비" → 무시하고 돌격
3. **사기에 따른 복종률** — 사기 높으면 자기 뜻대로, 중간이면 복종, 낮으면 소극적
4. **일기토 = 플레이어가 명령으로 신청** (기존 자동 발동에서 변경)
